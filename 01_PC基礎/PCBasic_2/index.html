    📋 セクション1: ファイル・フォルダって何？

    1-1. ファイルとは何か？

    ファイルとは、デジタルデータを保存する「入れ物」のことです。

    身近な例で理解しよう

    現実世界で例えると： - 写真アルバム → 画像ファイル（.jpg、.png） - 手紙・文書 → テキストファイル（.txt、.docx） - 音楽CD → 音声ファイル（.mp3、.wav） - DVD映画 → 動画ファイル（.mp4、.avi）

    📘​ ファイル要素の説明

    ファイルの基本要素

    1. ファイル名: 何のファイルかを表す名前
    2. 拡張子: ファイルの種類を表す記号（.txtや.jpgなど）
    3. ファイルサイズ: そのファイルがどのくらいの容量を使うか

    例: 「写真1.jpg」 - ファイル名: 写真1 - 拡張子: .jpg（画像ファイルの証） - ファイルサイズ: 2.5MB

    1-2. フォルダとは何か？

    フォルダとは、ファイルを整理するための「整理箱」のことです。

    身近な例で理解しよう

    現実世界で例えると： - 本棚の段 → フォルダ - 本 → ファイル - 本棚全体 → コンピュータのハードディスク

    📘​ フォルダーの概念の説明

    階層構造の理解

    フォルダは「入れ子構造」になっています：

    コンピュータ
├─ ドキュメント（フォルダ）
│   ├─ 学校関係（フォルダ）
│   │   ├─ レポート1.docx（ファイル）
│   │   └─ 課題.txt（ファイル）
│   └─ プライベート（フォルダ）
│       ├─ 日記.txt（ファイル）
│       └─ 旅行写真（フォルダ）
│           ├─ 沖縄1.jpg（ファイル）
│           └─ 沖縄2.jpg（ファイル）

    この構造を「パス」や「住所」と呼びます

    1-3. 拡張子の理解

    拡張子は、ファイルの「種類」を表す識別子です。

    主要な拡張子一覧

    拡張子 ファイルの種類 例
    .txt テキスト（文字） メモ.txt
    .docx Word文書 レポート.docx
    .jpg、.png 画像 写真.jpg
    .mp3、.wav 音声 音楽.mp3
    .mp4、.avi 動画 映画.mp4
    .pdf PDF文書 資料.pdf
    .exe プログラム アプリ.exe

    🚨 安全性について

    • 安全な拡張子: .txt、.jpg、.png、.pdf、.docx
    • 注意が必要: .exe、.bat、.com（プログラムファイル）

    基本ルール: 知らない拡張子のファイルは、詳しい人に聞いてから開く

    💡 理解度チェック

    以下について説明できますか？

    1. ファイルとフォルダの違いは何ですか？
    2. 「ドキュメント\学校\レポート.docx」のパスの意味は？
    3. 拡張子「.txt」と「.exe」の違いと危険性は？
    💡 解答例 1. **ファイル**: データの入れ物、**フォルダ**: ファイルを整理する箱 2. **ドキュメントフォルダ**の中の**学校フォルダ**の中にある**レポート.docx**という住所 3. **.txt**は安全なテキストファイル、**.exe**は実行される可能性があるので注意が必要

    2-5. パス（アドレス）の理解

    パスとは、ファイルやフォルダの「住所」のことです。

    パスの読み方

    例: C:\Users\田中\Documents\学校\レポート.docx

    • C:：ハードディスクの名前（C ドライブ）
    • Users：ユーザーフォルダ
    • 田中：ユーザー名
    • Documents：ドキュメントフォルダ
    • 学校：自分で作成したフォルダ
    • レポート.docx：ファイル名

    アドレスバーの活用

    1. 現在地確認: 今どこにいるかが分かる
    2. 直接移動: パスを入力して直接移動可能
    3. パスのコピー: 他の人にファイルの場所を教える時に便利

    よくある問題と解決方法

    問題1: ファイルが見つからない

    解決手順： 1. エクスプローラーの検索機能を使用 - エクスプローラー右上の検索ボックスにファイル名を入力 - 部分的な名前でも検索可能

    1. 最近使用したファイルを確認
    2. エクスプローラー左側の「クイックアクセス」を確認
    3. 最近使ったファイルが表示される

    📘​ ファイル検索機能の使い方

    問題2: 間違ってファイルを削除した

    解決手順： 1. ごみ箱を確認 - デスクトップのごみ箱をダブルクリック - 削除したファイルを探す

    1. 復元の実行
    2. ファイルを右クリック→「元に戻す」
    3. 元の場所に復元される
    問題3: ファイル名を変更できない

    原因と解決： - ファイルが使用中: アプリケーションを閉じる - 権限がない: 管理者に相談 - 読み取り専用: プロパティで読み取り専用を解除

    問題4: フォルダがいっぱいで整理できない

    整理のステップ： 1. 不要ファイルの識別: 更新日が古い、使わないファイル 2. 一時フォルダの作成: 「整理中」「確認待ち」フォルダ 3. 段階的な整理: 一度にすべてやらず、少しずつ
