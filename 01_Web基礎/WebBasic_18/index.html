    WebBasic 18

    text Week 1の集大成として、これまで学んだHTML構造、バリデーション機能、CSSスタイリングを段階的に積み重ねて統合し、Formspree.ioを使って実際にデータ送信できるフォームを完成させる。レッスン1-4の技術を一つずつ復習しながら組み合わせることで、確実な理解と達成感を提供する。 18. 実践：お問い合わせフォームを完成させて送信体験 # 実践：お問い合わせフォームを完成させて送信体験

    🎯 このレッスンの目標

    • Week 1で学んだ全ての技術を段階的に組み合わせる
    • レッスン1-4の内容を復習しながら発展させる
    • Formspree.ioを使って実際にデータ送信を体験する
    • 送信完了ページを作成する
    • 本当に使えるお問い合わせフォームを自分の手で完成させる
    • Week 1の学習成果を実感する

    📚 Week 1の学習内容の確認

    これまで4つのレッスンで段階的に学んできました：

    レッスン1: フォームの基本構造

    • &lt;form&gt;要素とaction、method属性
    • 基本的なinput要素（text、email、submit）
    • name属性の重要性

    レッスン2: 様々な入力要素

    • input要素の種類（password、number、date、checkbox、radio）
    • textareaとselect要素
    • label要素とアクセシビリティ

    レッスン3: HTMLバリデーション

    • required属性で必須入力設定
    • pattern属性で入力パターン制限
    • maxlength/minlengthで文字数制限
    • placeholder属性の効果的活用

    レッスン4: CSSスタイリング

    • フォーム要素の基本スタイリング
    • hover・focus状態の効果
    • ボタンデザインとアニメーション
    • 統一感のあるレイアウト

    今回は、これらの技術を一つずつ積み重ねて、実際に動作するプロフェッショナルなフォームを完成させます！

    📘 Week1の学習の流れ（図解）

    1. 基本構造の準備（Week 1復習）

    🛠️ ステップ1: HTMLファイルの作成

    VSCodeで新しいファイル contact-form-final.html を作成しましょう。

    まずは、これまで学んだ基本構造から始めます：

    &lt;!DOCTYPE html&gt;
&lt;html lang=&quot;ja&quot;&gt;
&lt;head&gt;
    &lt;meta charset=&quot;UTF-8&quot;&gt;
    &lt;meta name=&quot;viewport&quot; content=&quot;width=device-width, initial-scale=1.0&quot;&gt;
    &lt;title&gt;お問い合わせフォーム - 完成版&lt;/title&gt;
&lt;/head&gt;
&lt;body&gt;
    &lt;h1&gt;お問い合わせフォーム&lt;/h1&gt;

    &lt;form method=&quot;POST&quot;&gt;
        &lt;!-- ここにフォーム要素を追加していきます --&gt;
    &lt;/form&gt;
&lt;/body&gt;
&lt;/html&gt;

    ✅ 確認： ファイルを保存してLive Serverで開き、タイトルが表示されることを確認してください。

    🔄 振り返りポイント

    • &lt;!DOCTYPE html&gt; でHTML5文書を宣言
    • lang=&quot;ja&quot; で日本語文書を指定
    • viewport でレスポンシブ対応
    • &lt;form method=&quot;POST&quot;&gt; でフォームの基本設定

    2. 入力要素の追加（レッスン1・2の技術）

    📝 ステップ2: 基本的な入力項目を追加

    レッスン1で学んだ基本要素から追加していきましょう。

    &lt;form&gt;タグの中に、以下を一つずつ追加してください：

    &lt;!-- お名前 --&gt;
&lt;label for=&quot;name&quot;&gt;お名前&lt;/label&gt;
&lt;input type=&quot;text&quot; id=&quot;name&quot; name=&quot;name&quot; placeholder=&quot;例：山田太郎&quot;&gt;

&lt;br&gt;&lt;br&gt;

&lt;!-- メールアドレス --&gt;
&lt;label for=&quot;email&quot;&gt;メールアドレス&lt;/label&gt;
&lt;input type=&quot;email&quot; id=&quot;email&quot; name=&quot;email&quot; placeholder=&quot;例：yamada@example.com&quot;&gt;

&lt;br&gt;&lt;br&gt;

&lt;!-- 送信ボタン --&gt;
&lt;input type=&quot;submit&quot; value=&quot;送信する&quot;&gt;

    ✅ 確認： 保存してブラウザで確認してください。基本的なフォームが表示されているはずです。

    📝 ステップ3: より多くの入力要素を追加

    レッスン2で学んだ様々な入力要素を追加しましょう。

    送信ボタンの前に、以下を追加してください：

    &lt;!-- フリガナ --&gt;
&lt;label for=&quot;kana&quot;&gt;フリガナ&lt;/label&gt;
&lt;input type=&quot;text&quot; id=&quot;kana&quot; name=&quot;kana&quot; placeholder=&quot;例：ヤマダタロウ&quot;&gt;

&lt;br&gt;&lt;br&gt;

&lt;!-- 電話番号 --&gt;
&lt;label for=&quot;phone&quot;&gt;電話番号&lt;/label&gt;
&lt;input type=&quot;tel&quot; id=&quot;phone&quot; name=&quot;phone&quot; placeholder=&quot;例：090-1234-5678&quot;&gt;

&lt;br&gt;&lt;br&gt;

&lt;!-- 年齢 --&gt;
&lt;label for=&quot;age&quot;&gt;年齢&lt;/label&gt;
&lt;input type=&quot;number&quot; id=&quot;age&quot; name=&quot;age&quot; placeholder=&quot;例：25&quot;&gt;

&lt;br&gt;&lt;br&gt;

&lt;!-- お問い合わせ種別 --&gt;
&lt;label for=&quot;category&quot;&gt;お問い合わせ種別&lt;/label&gt;
&lt;select id=&quot;category&quot; name=&quot;category&quot;&gt;
    &lt;option value=&quot;&quot;&gt;選択してください&lt;/option&gt;
    &lt;option value=&quot;product&quot;&gt;商品について&lt;/option&gt;
    &lt;option value=&quot;service&quot;&gt;サービスについて&lt;/option&gt;
    &lt;option value=&quot;support&quot;&gt;サポートについて&lt;/option&gt;
    &lt;option value=&quot;other&quot;&gt;その他&lt;/option&gt;
&lt;/select&gt;

&lt;br&gt;&lt;br&gt;

&lt;!-- お問い合わせ内容 --&gt;
&lt;label for=&quot;message&quot;&gt;お問い合わせ内容&lt;/label&gt;
&lt;textarea id=&quot;message&quot; name=&quot;message&quot; placeholder=&quot;お問い合わせ内容を詳しくお書きください&quot;&gt;&lt;/textarea&gt;

&lt;br&gt;&lt;br&gt;

&lt;!-- 個人情報の取り扱い --&gt;
&lt;input type=&quot;checkbox&quot; id=&quot;privacy&quot; name=&quot;privacy&quot;&gt;
&lt;label for=&quot;privacy&quot;&gt;個人情報の取り扱いに同意します&lt;/label&gt;

&lt;br&gt;&lt;br&gt;

    ✅ 確認： 保存してブラウザで確認してください。多くの入力項目が追加されているはずです。

    🔄 振り返りポイント

    • labelとinputの関連付け（for属性とid属性）
    • 様々なinput typeの使い分け（text、email、tel、number）
    • select要素とoption要素の組み合わせ
    • textarea要素の複数行入力
    • checkbox要素の使用方法

    3. バリデーション機能の実装（レッスン3の技術）

    ✅ ステップ4: 必須項目の設定

    レッスン3で学んだバリデーション機能を追加しましょう。

    以下の項目にrequired属性を追加してください：

    &lt;!-- お名前を必須に --&gt;
&lt;input type=&quot;text&quot; id=&quot;name&quot; name=&quot;name&quot; placeholder=&quot;例：山田太郎&quot; required&gt;

&lt;!-- メールアドレスを必須に --&gt;
&lt;input type=&quot;email&quot; id=&quot;email&quot; name=&quot;email&quot; placeholder=&quot;例：yamada@example.com&quot; required&gt;

&lt;!-- 年齢を必須に --&gt;
&lt;input type=&quot;number&quot; id=&quot;age&quot; name=&quot;age&quot; placeholder=&quot;例：25&quot; required&gt;

&lt;!-- お問い合わせ種別を必須に --&gt;
&lt;select id=&quot;category&quot; name=&quot;category&quot; required&gt;

&lt;!-- お問い合わせ内容を必須に --&gt;
&lt;textarea id=&quot;message&quot; name=&quot;message&quot; placeholder=&quot;お問い合わせ内容を詳しくお書きください&quot; required&gt;&lt;/textarea&gt;

&lt;!-- 個人情報の取り扱いを必須に --&gt;
&lt;input type=&quot;checkbox&quot; id=&quot;privacy&quot; name=&quot;privacy&quot; required&gt;

    ✅ 確認： 空欄で送信ボタンを押して、エラーメッセージが表示されることを確認してください。

    ✅ ステップ5: より詳細なバリデーション

    さらに詳細なバリデーションを追加しましょう：

    &lt;!-- フリガナにカタカナパターンを追加 --&gt;
&lt;input type=&quot;text&quot; id=&quot;kana&quot; name=&quot;kana&quot; placeholder=&quot;例：ヤマダタロウ&quot; 
       pattern=&quot;[ァ-ヴー]+&quot; title=&quot;カタカナで入力してください&quot;&gt;

&lt;!-- 電話番号にパターンを追加 --&gt;
&lt;input type=&quot;tel&quot; id=&quot;phone&quot; name=&quot;phone&quot; placeholder=&quot;例：090-1234-5678&quot; 
       pattern=&quot;[0-9]{2,4}-[0-9]{2,4}-[0-9]{3,4}&quot; title=&quot;ハイフン区切りで入力してください&quot;&gt;

&lt;!-- 年齢に範囲を追加 --&gt;
&lt;input type=&quot;number&quot; id=&quot;age&quot; name=&quot;age&quot; placeholder=&quot;例：25&quot; 
       min=&quot;1&quot; max=&quot;120&quot; required&gt;

&lt;!-- 名前に文字数制限を追加 --&gt;
&lt;input type=&quot;text&quot; id=&quot;name&quot; name=&quot;name&quot; placeholder=&quot;例：山田太郎&quot; 
       maxlength=&quot;50&quot; required&gt;

&lt;!-- メールアドレスに文字数制限を追加 --&gt;
&lt;input type=&quot;email&quot; id=&quot;email&quot; name=&quot;email&quot; placeholder=&quot;例：yamada@example.com&quot; 
       maxlength=&quot;100&quot; required&gt;

&lt;!-- お問い合わせ内容に文字数制限を追加 --&gt;
&lt;textarea id=&quot;message&quot; name=&quot;message&quot; placeholder=&quot;お問い合わせ内容を詳しくお書きください（500文字以内）&quot; 
          maxlength=&quot;500&quot; required&gt;&lt;/textarea&gt;

    ✅ 確認： - フリガナに「やまだ」と入力して送信 → パターンエラー - 年齢に「0」と入力して送信 → 範囲エラー

    🔄 振り返りポイント

    • required属性で必須項目を設定
    • pattern属性で入力パターンを制限
    • min/max属性で数値範囲を制限
    • maxlength属性で文字数を制限
    • title属性でユーザーへのヒント提供

    4. CSSスタイリングの適用（レッスン4の技術）

    🎨 ステップ6: 基本的なスタイリング

    レッスン4で学んだCSS技術を適用していきましょう。

    &lt;head&gt;タグ内に&lt;style&gt;を追加してください：

    &lt;style&gt;
/* 全体の設定 */
body {
    font-family: Arial, sans-serif;
    background-color: #f8f9fa;
    margin: 0;
    padding: 20px;
}

/* フォーム全体 */
form {
    max-width: 600px;
    margin: 0 auto;
    background-color: white;
    padding: 30px;
    border-radius: 8px;
}

/* タイトル */
h1 {
    text-align: center;
    color: #333;
    margin-bottom: 30px;
}
&lt;/style&gt;

    ✅ 確認： 保存してブラウザで確認してください。背景色とフォームの枠が表示されているはずです。

    🎨 ステップ7: 入力要素のスタイリング

    入力要素を美しくスタイリングしましょう。&lt;style&gt;内に追加してください：

    /* ラベル */
label {
    display: block;
    margin-bottom: 5px;
    font-weight: bold;
    color: #333;
}

/* 入力要素 */
input, textarea, select {
    width: 100%;
    padding: 10px;
    border: 2px solid #ddd;
    border-radius: 4px;
    font-size: 16px;
    margin-bottom: 15px;
}

/* テキストエリア */
textarea {
    resize: vertical;
    height: 100px;
}

/* チェックボックス */
input[type=&quot;checkbox&quot;] {
    width: auto;
    margin-right: 8px;
}

    ✅ 確認： 入力フィールドが統一されたデザインになっていることを確認してください。

    🎨 ステップ8: ホバー・フォーカス効果

    インタラクティブな効果を追加しましょう：

    /* ホバー効果 */
input:hover, textarea:hover, select:hover {
    border-color: #999;
}

/* フォーカス効果 */
input:focus, textarea:focus, select:focus {
    outline: none;
    border-color: #007bff;
    box-shadow: 0 0 0 3px rgba(0, 123, 255, 0.25);
}

/* 送信ボタン */
input[type=&quot;submit&quot;] {
    background-color: #007bff;
    color: white;
    font-weight: bold;
    cursor: pointer;
    border: none;
    padding: 15px 30px;
}

input[type=&quot;submit&quot;]:hover {
    background-color: #0056b3;
}

    ✅ 確認： - 入力フィールドにマウスを合わせるとボーダー色が変わる - フォーカスすると青い枠が表示される - 送信ボタンがホバーで色が変わる

    🎨 ステップ9: レスポンシブ対応

    スマートフォン対応を追加しましょう：

    /* レスポンシブ対応 */
@media (max-width: 768px) {
    form {
        padding: 20px;
        margin: 10px;
    }

    h1 {
        font-size: 24px;
    }

    input, textarea, select {
        font-size: 16px; /* iOSでのズーム防止 */
    }
}

    ✅ 確認： ブラウザのデベロッパーツール（F12）でスマートフォンサイズにして確認してください。

    🔄 振り返りポイント

    • 基本的なレイアウト（中央配置、背景色）
    • 入力要素の統一デザイン（padding、border、border-radius）
    • インタラクティブ効果（hover、focus）
    • レスポンシブデザイン（メディアクエリ）

    5. Formspree.io連携（実際の送信機能）

    📧 ステップ10: Formspree.io設定

    📘 Formspree導入ガイド

    実際にメールを送信できるようにしましょう。

    10-1. Formspree.ioアカウント作成

    1. Formspree.io(https://formspree.io/) にアクセス
    2. 「Get Started」をクリック
    3. メールアドレスとパスワードを入力してアカウント作成
    4. メール認証を完了
    5. アンケートがいくつかある為回答する

    10-2. フォーム作成

    1. ダッシュボードで「New Form」をクリック
    2. フォーム名を入力（例：「お問い合わせフォーム」）
    3. 通知先メールアドレスを入力
    4. 「Create Form」をクリック

    10-3. エンドポイントURLの取得

    作成後に表示されるエンドポイントURLをコピーします：

    https://formspree.io/f/YOUR_FORM_ID

    🔧 ステップ11: フォームの設定変更

    HTMLの&lt;form&gt;タグを以下のように変更してください：

    &lt;!-- 修正前 --&gt;
&lt;form method=&quot;POST&quot;&gt;

&lt;!-- 修正後 --&gt;
&lt;form action=&quot;https://formspree.io/f/YOUR_FORM_ID&quot; method=&quot;POST&quot;&gt;

    💡 重要： YOUR_FORM_ID の部分を、実際に取得したエンドポイントURLに置き換えてください。

    📋 ステップ12: 隠しフィールドの追加

    送信ボタンの前に以下を追加してください：

    &lt;!-- 件名の指定 --&gt;
&lt;input type=&quot;hidden&quot; name=&quot;_subject&quot; value=&quot;【お問い合わせフォーム】新しいお問い合わせ&quot;&gt;

&lt;!-- リダイレクト先の指定 --&gt;
&lt;input type=&quot;hidden&quot; name=&quot;_next&quot; value=&quot;thank-you.html&quot;&gt;

    ✅ 確認： この段階で実際の送信テストを行いましょう！

    🧪 送信テスト

    テストデータを入力：
