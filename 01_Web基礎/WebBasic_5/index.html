    WebBasic 5

    text Web開発の入り口として、Webページの表示の仕組みとHTML/CSSの基本概念を理解させる。未経験者が「なぜHTMLとCSSが必要なのか」を体感できるよう、身近な例を使って説明する。 5.Webページの仕組みとHTML/CSSの役割 # Webページの仕組みとHTML/CSSの役割

    🎯 このレッスンで学ぶこと

    • Webページがどのように作られ、表示されるのか
    • HTMLとCSSの役割の違い
    • ブラウザがWebページを表示する仕組み

    1. Webページって何でできているの？

    普段見ているWebページ（Yahoo!、Google、YouTubeなど）は、実は文字で書かれたコードでできています。

    身近な例で考えてみよう

    新聞を思い浮かべてください： - 見出しがあって - 本文があって - 写真があって - 文字の大きさや色が違う

    Webページも同じです！ただし、新聞と違うのは「コンピューターが理解できる言葉」で書かれていることです。

    Webページを構成する2つの言語

    HTML（エイチティーエムエル） - Webページの「中身」を決める - 「ここは見出し」 「ここは本文」 「ここは画像」という構造を作る - 例：新聞の「記事の内容」

    CSS（シーエスエス） - Webページの「見た目」を決める - 「文字を大きく」 「色を赤に」 「レイアウトを整える」という装飾をする - 例：新聞の「レイアウトデザイン」

    2. 実際に確認してみよう

    例題：任意のWebサイトのコードを見てみる

    手順 1. Google Chrome を開く 2. 任意のWebサイト(例:https://www.google.com) にアクセス 3. ページ上で右クリック → 「ページのソースを表示」をクリック

    表示一例 

    &lt;!DOCTYPE html&gt;
&lt;html&gt;
&lt;head&gt;
    &lt;title&gt;Google&lt;/title&gt;
    &lt;style&gt;
        body { font-family: arial; }
        h1 { color: blue; }
    &lt;/style&gt;
&lt;/head&gt;
&lt;body&gt;
    &lt;h1&gt;Google&lt;/h1&gt;
    &lt;p&gt;検索エンジンです&lt;/p&gt;
&lt;/body&gt;
&lt;/html&gt;

    解説 - &lt;h1&gt;Google&lt;/h1&gt; → HTMLで「見出し」を指定 - h1 { color: blue; } → CSSで「見出しの色を青に」を指定

    3. HTMLとCSSの役割分担

    🏗️ HTML = 建物の骨組み

    &lt;h1&gt;会社名&lt;/h1&gt;
&lt;p&gt;会社の説明文です。&lt;/p&gt;
&lt;img src=&quot;company.jpg&quot;&gt;

    → 「ここに見出し、ここに文章、ここに画像を置く」という構造を決める

    🎨 CSS = 建物の内装・外装

    h1 {
    color: red;
    font-size: 30px;
}
p {
    color: gray;
    font-size: 16px;
}

    → 「見出しは赤色で30px、文章は灰色で16px」という見た目を決める

    4. ブラウザの表示の仕組み

    ステップ1：ファイルを読み込む

    ブラウザがHTMLファイルとCSSファイルをダウンロード

    ステップ2：構造を理解する

    HTMLの内容を読んで「どんな要素があるか」を把握

    ステップ3：装飾を適用する

    CSSの内容を読んで「どんな見た目にするか」を決定

    ステップ4：画面に表示する

    HTMLの構造＋CSSの装飾を組み合わせて、最終的なページを表示

    5. 実践課題

    課題：メモ帳でHTMLファイルを作ってみよう

    手順 1. メモ帳（Windowsの場合）またはテキストエディタを開く 2. 以下のコードをそのまま入力する

    &lt;!DOCTYPE html&gt;
&lt;html&gt;
&lt;head&gt;
    &lt;title&gt;初めてのWebページ&lt;/title&gt;
    &lt;style&gt;
        h1 {
            color: blue;
            font-size: 40px;
        }
        p {
            color: green;
            font-size: 18px;
        }
    &lt;/style&gt;
&lt;/head&gt;
&lt;body&gt;
    &lt;h1&gt;こんにちは、世界！&lt;/h1&gt;
    &lt;p&gt;これが私の初めてのWebページです。&lt;/p&gt;
&lt;/body&gt;
&lt;/html&gt;
    1. 「ファイル」→「名前を付けて保存」
    2. ファイル名を test.html にして保存
    3. 保存した test.html ファイルをダブルクリック

    期待される結果 - ブラウザが開いて - 「こんにちは、世界！」が青色の大きな文字で表示 - 「これが私の初めてのWebページです。」が緑色で表示

    解答・解説

    上記の手順通りに進めば、ブラウザで以下のWebページが表示されます。

    ポイント - &lt;h1&gt; タグ → 見出しの構造（HTML） - color: blue; → 色の装飾（CSS） - HTMLとCSSが組み合わさって、最終的な見た目が決まる

    6. まとめ

    学んだこと

    ✅ Webページは「HTML」と「CSS」というコードで作られている
    ✅ HTML = 構造（何を表示するか）
    ✅ CSS = 装飾（どう見せるか）
    ✅ ブラウザがHTMLとCSSを読み込んで、画面に表示する
