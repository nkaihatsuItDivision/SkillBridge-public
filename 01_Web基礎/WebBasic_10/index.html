    1. CSSとは何か？（復習と詳細）

    これまでのレッスンで、HTMLは構造、CSSは装飾という役割を学びました。今度は実際にCSSを書いてみましょう。

    HTML だけの場合 vs CSS ありの場合

    HTMLのみ： 

    &lt;h1&gt;私のWebサイト&lt;/h1&gt;
&lt;p&gt;これは普通の段落です。&lt;/p&gt;
    → ブラウザのデフォルトスタイル（黒い文字、標準フォント）

    CSS追加後： 

    &lt;h1 style=&quot;color: blue; font-size: 32px;&quot;&gt;私のWebサイト&lt;/h1&gt;
&lt;p style=&quot;color: green; font-size: 18px;&quot;&gt;これは装飾された段落です。&lt;/p&gt;
    → 青い見出し、緑の段落、好みのサイズ

    CSSができること

    • 文字の色・サイズ・フォントを変更
    • 背景色や背景画像を設定
    • 要素の配置やレイアウトを調整
    • 枠線や影などの装飾効果
    • アニメーションや動的効果

    2. CSSの基本的な書き方

    CSS の構文

    セレクタ {
    プロパティ: 値;
    プロパティ: 値;
}

    実際の例

    h1 {
    color: blue;
    font-size: 32px;
}

    各部分の説明： - セレクタ（h1）：どの要素に適用するか - プロパティ（color）：何を変更するか - 値（blue）：どのように変更するか - セミコロン（;）：各スタイルの区切り

    よく使用するプロパティの例

    /* 文字関連 */
color: red;           /* 文字色 */
font-size: 20px;      /* 文字サイズ */
font-family: Arial;   /* フォント */

/* 背景関連 */
background-color: yellow;  /* 背景色 */

/* 位置・サイズ関連 */
width: 300px;         /* 幅 */
height: 200px;        /* 高さ */
margin: 10px;         /* 外側の余白 */
padding: 15px;        /* 内側の余白 */

    3. HTMLにCSSを適用する3つの方法

    方法1：インラインスタイル（タグに直接記述）

    &lt;h1 style=&quot;color: red; font-size: 30px;&quot;&gt;赤い見出し&lt;/h1&gt;
&lt;p style=&quot;color: blue;&quot;&gt;青い段落&lt;/p&gt;

    メリット： - その要素だけに確実に適用される - すぐに確認できる

    デメリット： - 同じスタイルを複数の要素に適用しにくい - HTMLが読みにくくなる - 修正が大変

    方法2：内部スタイルシート（headに記述）

    &lt;!DOCTYPE html&gt;
&lt;html lang=&quot;ja&quot;&gt;
&lt;head&gt;
    &lt;meta charset=&quot;UTF-8&quot;&gt;
    &lt;title&gt;CSSテスト&lt;/title&gt;
    &lt;style&gt;
        h1 {
            color: red;
            font-size: 30px;
        }
        p {
            color: blue;
            font-size: 16px;
        }
    &lt;/style&gt;
&lt;/head&gt;
&lt;body&gt;
    &lt;h1&gt;赤い見出し&lt;/h1&gt;
    &lt;p&gt;青い段落&lt;/p&gt;
&lt;/body&gt;
&lt;/html&gt;

    メリット： - そのページ内の全ての要素に適用できる - HTMLから分離されて読みやすい

    デメリット： - 他のページでは使用できない

    方法3：外部スタイルシート（別ファイル）

    style.css（別ファイル）： 

    h1 {
    color: red;
    font-size: 30px;
}

p {
    color: blue;
    font-size: 16px;
}

    HTML： 

    &lt;!DOCTYPE html&gt;
&lt;html lang=&quot;ja&quot;&gt;
&lt;head&gt;
    &lt;meta charset=&quot;UTF-8&quot;&gt;
    &lt;title&gt;CSSテスト&lt;/title&gt;
    &lt;link rel=&quot;stylesheet&quot; href=&quot;style.css&quot;&gt;
&lt;/head&gt;
&lt;body&gt;
    &lt;h1&gt;赤い見出し&lt;/h1&gt;
    &lt;p&gt;青い段落&lt;/p&gt;
&lt;/body&gt;
&lt;/html&gt;

    メリット： - 複数のページで同じスタイルを使用できる - 管理しやすい - HTMLがすっきりする

    デメリット： - ファイルを分けて管理する必要がある

    使い分けの指針

    • テスト・練習：内部スタイルシート
    • 本格的なサイト：外部スタイルシート
    • 特別な装飾：インラインスタイル（最小限で）

    4. 基本的なスタイリング実践

    文字の装飾

    /* 文字色 */
h1 { color: red; }
h1 { color: #ff0000; }     /* 16進数カラーコード */
h1 { color: rgb(255, 0, 0); } /* RGB指定 */

/* 文字サイズ */
h1 { font-size: 32px; }    /* ピクセル */
h1 { font-size: 2em; }     /* 相対サイズ */
h1 { font-size: 150%; }    /* パーセント */

/* フォント */
h1 { font-family: Arial; }
h1 { font-family: &quot;Times New Roman&quot;; }
h1 { font-family: &quot;游ゴシック&quot;, Arial, sans-serif; }

/* 文字の太さ */
h1 { font-weight: bold; }    /* 太字 */
h1 { font-weight: normal; }  /* 通常 */

/* 文字の配置 */
h1 { text-align: center; }   /* 中央揃え */
h1 { text-align: left; }     /* 左揃え */
h1 { text-align: right; }    /* 右揃え */

    背景の装飾

    /* 背景色 */
body { background-color: #f0f0f0; }
div { background-color: lightblue; }

/* 背景画像 */
body { background-image: url(&quot;bg.jpg&quot;); }

    要素のサイズと余白

    /* 幅と高さ */
div {
    width: 300px;
    height: 200px;
}

/* 余白 */
div {
    margin: 20px;     /* 外側の余白 */
    padding: 15px;    /* 内側の余白 */
}

/* 枠線 */
div {
    border: 2px solid black;    /* 太さ スタイル 色 */
    border-radius: 10px;        /* 角を丸く */
}

    5. セレクタの種類と使い分け

    基本的なセレクタ

    /* 要素セレクタ */
h1 { color: red; }        /* すべてのh1 */
p { color: blue; }        /* すべてのp */

/* クラスセレクタ */
.highlight { background-color: yellow; }  /* class=&quot;highlight&quot; */
.large-text { font-size: 24px; }         /* class=&quot;large-text&quot; */

/* IDセレクタ */
#header { background-color: navy; }      /* id=&quot;header&quot; */
#footer { text-align: center; }         /* id=&quot;footer&quot; */

    HTMLでのクラスとIDの使用

    &lt;!-- クラス：繰り返し使用可能 --&gt;
&lt;p class=&quot;highlight&quot;&gt;この段落は黄色背景&lt;/p&gt;
&lt;p class=&quot;highlight&quot;&gt;この段落も黄色背景&lt;/p&gt;

&lt;!-- ID：1ページに1つだけ --&gt;
&lt;div id=&quot;header&quot;&gt;ヘッダー部分&lt;/div&gt;
&lt;div id=&quot;footer&quot;&gt;フッター部分&lt;/div&gt;

&lt;!-- 複数のクラスを適用 --&gt;
&lt;p class=&quot;highlight large-text&quot;&gt;大きくて黄色い文字&lt;/p&gt;

    組み合わせセレクタ

    /* 子孫セレクタ */
div p { color: green; }        /* div内のすべてのp */

/* 直接の子セレクタ */
div &gt; p { color: purple; }     /* divの直接の子のp */

/* 複数セレクタ */
h1, h2, h3 { color: darkblue; }  /* h1とh2とh3 */

    6. 実践課題

    課題：自己紹介ページにスタイルを適用しよう

    前回のレッスンで作成したHTMLページに、CSSで装飾を加えてください。

    装飾の要件： 1. ページ全体の背景色を薄いグレーに 2. メイン見出し（h1）を青色、中央揃え、大きなサイズに 3. セクション見出し（h2）を濃い青色、下線付きに 4. 段落（p）の文字サイズを適度に大きく 5. 重要な部分（strong）を赤色に 6. リスト項目に余白を追加して読みやすく

    サンプルコード： 

    &lt;!DOCTYPE html&gt;
&lt;html lang=&quot;ja&quot;&gt;
&lt;head&gt;
    &lt;meta charset=&quot;UTF-8&quot;&gt;
    &lt;title&gt;田中太郎のプロフィール&lt;/title&gt;
    &lt;style&gt;
        /* ページ全体のスタイル */
        body {
            background-color: #f5f5f5;
            font-family: &quot;游ゴシック&quot;, Arial, sans-serif;
            margin: 0;
            padding: 20px;
        }

        /* メイン見出し */
        h1 {
            color: #2c5282;
            text-align: center;
            font-size: 36px;
            margin-bottom: 30px;
        }

        /* セクション見出し */
        h2 {
            color: #1a365d;
            border-bottom: 2px solid #2c5282;
            padding-bottom: 5px;
            font-size: 24px;
        }

        /* 段落 */
        p {
            font-size: 16px;
            line-height: 1.6;
            margin-bottom: 15px;
        }

        /* 強調テキスト */
        strong {
            color: #e53e3e;
        }

        /* リスト */
        ul, ol {
            padding-left: 30px;
        }

        li {
            margin-bottom: 8px;
            font-size: 16px;
            line-height: 1.5;
        }

        /* 特別なクラス */
        .contact-info {
            background-color: #e6f3ff;
            padding: 15px;
            border-radius: 8px;
            margin: 20px 0;
        }

        .skills {
            background-color: #f0fff4;
            padding: 10px;
            border-left: 4px solid #38a169;
        }
    &lt;/style&gt;
&lt;/head&gt;
&lt;body&gt;
    &lt;h1&gt;田中太郎のプロフィール&lt;/h1&gt;

    &lt;h2&gt;基本情報&lt;/h2&gt;
    &lt;div class=&quot;contact-info&quot;&gt;
        &lt;p&gt;はじめまして！私の名前は&lt;strong&gt;田中太郎&lt;/strong&gt;です。&lt;/p&gt;
        &lt;p&gt;
            出身：大阪府大阪市&lt;br&gt;
            年齢：25歳&lt;br&gt;
            職業：会社員
        &lt;/p&gt;
    &lt;/div&gt;

    &lt;h2&gt;趣味・興味&lt;/h2&gt;
    &lt;p&gt;読書が大好きで、特にビジネス書をよく読みます。&lt;/p&gt;
    &lt;ul&gt;
        &lt;li&gt;読書（月に5冊程度）&lt;/li&gt;
        &lt;li&gt;映画鑑賞（特にSF映画）&lt;/li&gt;
        &lt;li&gt;料理（週末に新しいレシピに挑戦）&lt;/li&gt;
        &lt;li&gt;&lt;strong&gt;プログラミング学習&lt;/strong&gt;（現在勉強中）&lt;/li&gt;
    &lt;/ul&gt;

    &lt;h2&gt;学習目標&lt;/h2&gt;
    &lt;div class=&quot;skills&quot;&gt;
        &lt;p&gt;半年後には&lt;strong&gt;Webサイトを自作&lt;/strong&gt;できるようになりたいです。&lt;/p&gt;
        &lt;ol&gt;
            &lt;li&gt;HTML/CSSをマスターする&lt;/li&gt;
            &lt;li&gt;JavaScriptの基礎を学ぶ&lt;/li&gt;
            &lt;li&gt;ポートフォリオサイトを作成する&lt;/li&gt;
            &lt;li&gt;エンジニア転職を目指す&lt;/li&gt;
        &lt;/ol&gt;
    &lt;/div&gt;
&lt;/body&gt;
&lt;/html&gt;

    解答・解説

    期待される表示：

    使用したCSSの確認： - body：ページ全体の背景色とフォント設定 - h1, h2：見出しの色、サイズ、配置 - p, li：テキストのサイズと行間 - strong：強調部分の色変更 - クラス：特定の部分に特別なスタイル適用

    評価ポイント： - [ ] 背景色が適切に設定されているか - [ ] 見出しが目立って階層が分かりやすいか - [ ] 文字が読みやすいサイズと行間になっているか - [ ] クラスを使って部分的なスタイルが適用されているか

    7. よくある間違いと改善方法

    間違い1：セミコロンの記述漏れ

    /* 間違い */
h1 {
    color: red
    font-size: 30px  /* セミコロンがない */
}

/* 正しい */
h1 {
    color: red;
    font-size: 30px;
}

    間違い2：クラス・IDセレクタの書き方

    /* 間違い */
class-name { color: red; }    /* ドットがない */
id-name { color: blue; }      /* シャープがない */

/* 正しい */
.class-name { color: red; }   /* クラスはドット */
#id-name { color: blue; }     /* IDはシャープ */

    間違い3：カラーコードの指定ミス

    /* 間違い */
h1 { color: #ff00; }          /* 文字数不足 */
h1 { color: red blue; }       /* 複数色指定不可 */

/* 正しい */
h1 { color: #ff0000; }        /* 6桁のコード */
h1 { color: red; }            /* 色名 */
h1 { color: rgb(255, 0, 0); } /* RGB */

    間違い4：単位の記述漏れ

    /* 間違い */
h1 { font-size: 30; }         /* 単位がない */

/* 正しい */
h1 { font-size: 30px; }       /* 単位付き */

    8. まとめ

    学んだこと

    ✅ CSSの基本構文：セレクタ、プロパティ、値の関係
    ✅ 3つの適用方法：インライン、内部、外部スタイルシート
    ✅ 基本的なスタイリング：色、フォント、サイズ、余白
    ✅ セレクタの種類：要素、クラス、IDセレクタ
    ✅ 実践的なページ装飾：構造化されたスタイルの適用

    重要なポイント

    • セミコロンを忘れずに：各プロパティの最後に必ず付ける
    • クラスとIDを使い分ける：繰り返し使用はクラス、一意な要素はID
    • 外部スタイルシートを活用：複数ページで共通スタイルを使用
    • 読みやすいコードを書く：適切なインデントとコメント
