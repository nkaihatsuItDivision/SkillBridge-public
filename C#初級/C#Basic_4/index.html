    レッスン4

    text if文を使った条件分岐について学習する。プログラムが状況に応じて異なる処理を実行できるようになることを目指す。 レッスン4: 条件分岐（if文）でプログラムに判断力を # レッスン4: 条件分岐（if文）でプログラムに判断力を

    学習目標

    • if文の基本構文を理解する
    • 比較演算子を使った条件式を作成できる
    • if-else、if-else ifを使い分けられる
    • 論理演算子を使った複合条件を理解する

    条件分岐とは

    プログラムが状況に応じて異なる処理を実行することです。例えば： - 年齢によって入場料を変える - 点数によって合格・不合格を判定する - 天気によって服装を決める

    if文の基本構文

    基本のif文

    if (条件式)
{
    // 条件が真の時に実行される処理
}

    if-else文

    if (条件式)
{
    // 条件が真の時の処理
}
else
{
    // 条件が偽の時の処理
}

    if-else if文

    if (条件式1)
{
    // 条件式1が真の時の処理
}
else if (条件式2)
{
    // 条件式2が真の時の処理
}
else
{
    // すべての条件が偽の時の処理
}

    比較演算子

    2つの値を比較して、その結果をtrue（真）またはfalse（偽）で返す演算子です。 if文の条件式を作るときの基本的な要素となります。

    演算子 意味 例
    == 等しい age == 20
    != 等しくない name != &quot;admin&quot;
    > より大きい score &gt; 80
    < より小さい price &lt; 1000
    >= 以上 age &gt;= 18
    <= 以下 temperature &lt;= 25

    例題1：年齢による入場料判定

    まずは下記のコードを手作業で書き写して流れを理解しながら出力を確認しましょう

    using System;

namespace AdmissionFee
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine(&quot;=== 入場料計算 ===&quot;);
            Console.WriteLine(&quot;年齢を入力してください:&quot;);
            int age = int.Parse(Console.ReadLine());

            int fee;

            if (age &lt; 12)
            {
                fee = 500;
                Console.WriteLine(&quot;子供料金です。&quot;);
            }
            else if (age &gt;= 65)
            {
                fee = 800;
                Console.WriteLine(&quot;シニア料金です。&quot;);
            }
            else
            {
                fee = 1200;
                Console.WriteLine(&quot;大人料金です。&quot;);
            }

            Console.WriteLine(&quot;入場料: &quot; + fee + &quot;円&quot;);
        }
    }
}

    論理演算子

    複数の条件を組み合わせて、より複雑な判定を行うための演算子です。 比較演算子だけでは表現できない「AかつB」,「AまたはB」といった条件を作れます。

    && （かつ）

    if (age &gt;= 18 &amp;&amp; age &lt;= 65)
{
    Console.WriteLine(&quot;働く世代です&quot;);
}

    || （または）

    if (day == &quot;土曜日&quot; || day == &quot;日曜日&quot;)
{
    Console.WriteLine(&quot;週末です&quot;);
}

    ! （否定）

    if (!isStudent)
{
    Console.WriteLine(&quot;学生ではありません&quot;);
}

    例題2：成績判定システム

    まずは下記のコードを手作業で書き写して流れを理解しながら出力を確認しましょう

    using System;

namespace GradeSystem
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine(&quot;=== 成績判定システム ===&quot;);

            Console.WriteLine(&quot;生徒の名前を入力してください:&quot;);
            string studentName = Console.ReadLine();

            Console.WriteLine(&quot;点数を入力してください（0-100）:&quot;);
            int score = int.Parse(Console.ReadLine());

            string grade;
            string comment;

            if (score &gt;= 90)
            {
                grade = &quot;A&quot;;
                comment = &quot;優秀です！&quot;;
            }
            else if (score &gt;= 80)
            {
                grade = &quot;B&quot;;
                comment = &quot;良い成績です。&quot;;
            }
            else if (score &gt;= 70)
            {
                grade = &quot;C&quot;;
                comment = &quot;まずまずです。&quot;;
            }
            else if (score &gt;= 60)
            {
                grade = &quot;D&quot;;
                comment = &quot;もう少し頑張りましょう。&quot;;
            }
            else
            {
                grade = &quot;F&quot;;
                comment = &quot;再試験が必要です。&quot;;
            }

            Console.WriteLine(&quot;&quot;);
            Console.WriteLine(&quot;=== 結果 ===&quot;);
            Console.WriteLine(&quot;生徒名: &quot; + studentName);
            Console.WriteLine(&quot;点数: &quot; + score + &quot;点&quot;);
            Console.WriteLine(&quot;評価: &quot; + grade);
            Console.WriteLine(&quot;コメント: &quot; + comment);
        }
    }
}

    練習問題

    以下の仕様で「時間帯挨拶プログラム」を作成してください： - 現在の時刻（0-23の整数）を入力 - 6-11時：「おはようございます」 - 12-17時：「こんにちは」 - 18-21時：「こんばんは」 - その他：「遅い時間ですね」

    解答例

    using System;

namespace TimeGreeting
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine(&quot;=== 時間帯挨拶プログラム ===&quot;);
            Console.WriteLine(&quot;現在の時刻を入力してください（0-23）:&quot;);
            int hour = int.Parse(Console.ReadLine());

            string greeting;

            if (hour &gt;= 6 &amp;&amp; hour &lt;= 11)
            {
                greeting = &quot;おはようございます&quot;;
            }
            else if (hour &gt;= 12 &amp;&amp; hour &lt;= 17)
            {
                greeting = &quot;こんにちは&quot;;
            }
            else if (hour &gt;= 18 &amp;&amp; hour &lt;= 21)
            {
                greeting = &quot;こんばんは&quot;;
            }
            else
            {
                greeting = &quot;遅い時間ですね&quot;;
            }

            Console.WriteLine(&quot;&quot;);
            Console.WriteLine(hour + &quot;時ですね。&quot;);
            Console.WriteLine(greeting + &quot;！&quot;);
        }
    }
}

    出力結果

    例１

    === 時間帯挨拶プログラム ===
    現在の時刻を入力してください（0-23）:
    9

    9時ですね。
    おはようございます！

    例２

    === 時間帯挨拶プログラム ===
    現在の時刻を入力してください（0-23）:
    23

    23時ですね。
    遅い時間ですね！

    実践的な応用例：パスワード強度チェック

    using System;

namespace PasswordChecker
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine(&quot;=== パスワード強度チェック ===&quot;);
            Console.WriteLine(&quot;パスワードを入力してください:&quot;);
            string password = Console.ReadLine();

            int length = password.Length;
            bool hasNumber = false;
            bool hasUpper = false;
            bool hasLower = false;

            // 簡単なチェック（実際はもっと複雑な処理になります）
            if (password.Contains(&quot;1&quot;) || password.Contains(&quot;2&quot;) || 
                password.Contains(&quot;3&quot;) || password.Contains(&quot;4&quot;) || 
                password.Contains(&quot;5&quot;))
            {
                hasNumber = true;
            }

            Console.WriteLine(&quot;&quot;);
            Console.WriteLine(&quot;=== パスワード強度 ===&quot;);

            if (length &gt;= 8 &amp;&amp; hasNumber)
            {
                Console.WriteLine(&quot;強いパスワードです！&quot;);
            }
            else if (length &gt;= 6)
            {
                Console.WriteLine(&quot;普通のパスワードです。&quot;);
            }
            else
            {
                Console.WriteLine(&quot;弱いパスワードです。8文字以上にしてください。&quot;);
            }
        }
    }
}

    出力結果

    例１

    === パスワード強度チェック ===
    パスワードを入力してください:
    mypassword123

    === パスワード強度 ===
    強いパスワードです！

    例２

    === パスワード強度チェック ===
    パスワードを入力してください:
    abc

    === パスワード強度 ===
    弱いパスワードです。8文字以上にしてください。

    まとめ

    • if文でプログラムに判断力を持たせることができる
    • 比較演算子（==, !=, >, <, >=, <=）で条件を作成
    • 論理演算子（&&, ||, !）で複雑な条件を表現
    • else if を使って複数の条件を順次チェック
