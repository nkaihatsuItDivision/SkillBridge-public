    学習目標

    • C#がどのようなプログラミング言語かを理解する
    • 開発環境を準備し、最初のプログラムを実行できるようになる

    C#について

    C#（シーシャープ）は、Microsoft社が開発したプログラミング言語です。以下のような特徴があります：

    • 型安全性: エラーを事前に防ぐ仕組みが充実
    • オブジェクト指向: 大規模なプログラムを効率的に管理
    • 豊富なライブラリ: 様々な機能がすぐに使える
    • クロスプラットフォーム: Windows、Mac、Linuxで動作

    開発環境の準備

    Visual Studio Community のインストール

    1. Visual Studio Community をダウンロード
    2. インストール時に「.NET デスクトップ開発」を選択
    3. インストール完了後、Visual Studio を起動

    最初のプロジェクト作成

    1. 「新しいプロジェクトの作成」を選択
    2. 「コンソール アプリ」を選択
    3. プロジェクト名を「HelloWorld」に設定
    4. 「作成」をクリック

    例題：Hello Worldプログラム

    以下のコードがエディタに表示されているはずです：

    using System;

namespace HelloWorld
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine(&quot;Hello World!&quot;);
        }
    }
}

    実行方法

    1. F5キーを押すか、「デバッグ」→「デバッグの開始」
    2. コンソールウィンドウに「Hello World!」が表示される

    実行結果

    Hello World!

    練習問題

    自分の名前を表示するプログラムを作成してみましょう。 <br> <br> <br> <br>

    解答例

    using System;

namespace HelloWorld
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine(&quot;こんにちは、私の名前は田中です！&quot;);
            Console.WriteLine(&quot;C#の学習を始めました。&quot;);
        }
    }
}

    実行結果

    こんにちは、私の名前は田中です！
    C#の学習を始めました。

    まとめ

    • C#は初心者にも学びやすいプログラミング言語
    • Visual Studio Communityは無料で使える優秀な開発環境
    • Console.WriteLine()で文字を画面に表示できる
