    text for文とwhile文を使った繰り返し処理について学習する。効率的にコードを書き、大量のデータを処理する基礎を身につける。 レッスン5: ループ文で効率的な繰り返し処理 # レッスン5: ループ文で効率的な繰り返し処理

    学習目標

    • for文の基本構文を理解し、決まった回数の繰り返し処理ができる
    • while文を使って条件に基づく繰り返し処理ができる
    • break文とcontinue文でループを制御できる
    • ネストしたループを理解する

    繰り返し処理とは

    同じ処理を何度も実行することです。例えば： - 1から10まで数える - ユーザーが終了を選ぶまで処理を続ける - 配列の全要素を処理する

    なぜ繰り返し処理が必要なのか

    同じコードを何度も書くのは非効率的です。
    例えば、1から10まで表示したい場合に繰り返し処理を使用しない場合とする場合を比較します。 

    // 繰り返し処理を使わない場合（非効率）
Console.WriteLine(1);
Console.WriteLine(2);
Console.WriteLine(3);
// ... 10行も書く必要がある

// 繰り返し処理を使った場合（効率的）
for (int i = 1; i &lt;= 10; i++)
{
    Console.WriteLine(i);
}

    for文（決まった回数の繰り返し）

    for文の構成要素の詳細

    • 初期化：ループが始まる前に1回だけ実行される（カウンタ変数の準備）
    • 条件式：毎回のループ開始前にチェックされる（true なら続行、false なら終了）
    • 増減式：毎回のループ終了後に実行される（カウンタ変数の更新）

    基本構文

    for (初期化; 条件式; 増減式)
{
    // 繰り返したい処理
}

    例：1から5まで表示

    for (int i = 1; i &lt;= 5; i++)
{
    Console.WriteLine(i);
}

    while文（条件による繰り返し）

    基本構文

    while (条件式)
{
    // 条件が真の間、繰り返される処理
}

    例：ユーザーが0を入力するまで続ける

    int number = -1;
while (number != 0)
{
    Console.WriteLine(&quot;数値を入力してください（0で終了）:&quot;);
    number = int.Parse(Console.ReadLine());
    Console.WriteLine(&quot;入力された数値: &quot; + number);
}

    for文とwhile文の使い分け

    for文を使う場面

    • 繰り返し回数が事前に決まっている場合
    • カウンタ変数を使った処理の場合

    while文を使う場面

    • 繰り返し回数が分からない場合
    • 特定の条件が満たされるまで続ける場合
    • ユーザーの入力に応じて処理を続ける場合

    for文とwhile文の実行順序

    for文の実行順序

    1. 初期化（int i = 1）← 最初に1回だけ
    2. 条件式チェック（i <= 5）
    3. 処理実行（Console.WriteLine(i)）
    4. 増減式実行（i++）
    5. 2に戻る（条件式が false になるまで繰り返し）

    while文の実行順序

    1. 初期化（int i = 1）← 事前に手動で準備
    2. 条件式チェック（i <= 5）
    3. 処理実行（Console.WriteLine(i)）
    4. 増減式実行（i++）← 手動で書く必要がある
    5. 2に戻る（条件式が false になるまで繰り返し）

    for文は「初期化」「条件式」「増減式」が1箇所にまとまっているため、 カウンタを使った繰り返し処理では見やすく、書き忘れも防げます。

    無限ループの危険性

    無限ループとは

    条件式が永遠にtrueのままになってしまい、プログラムが止まらなくなってしまう状態です。
    一例を下記に示します。 

    // 危険な例：無限ループ
int count = 1;
while (count &lt;= 5)
{
    Console.WriteLine(count);
    // count++; を忘れると永遠に1のまま
}

// 正しい例
int count = 1;
while (count &lt;= 5)
{
    Console.WriteLine(count);
    count++; // 必ずカウンタを更新する
}

    ネストしたループ

    ネストしたループとは

    ループの中にさらにループを入れることです。外側のループが1回実行される間に、 内側のループは指定回数分すべて実行されます。 

    for (int i = 1; i &lt;= 3; i++)        // 外側：3回実行
{
    for (int j = 1; j &lt;= 2; j++)    // 内側：毎回2回実行
    {
        // この部分は合計6回実行される（3×2）
    }
}

    変数のスコープ（有効範囲）

    for文で宣言した変数は、そのfor文の中でのみ使用できます。 

    for (int i = 1; i &lt;= 5; i++)
{
    Console.WriteLine(i); // ここではiが使える
}
Console.WriteLine(i); // エラー！ここではiは使えない

    using System;

namespace NumberGuessing
{
    class Program
    {

        {

            Random random = new Random();

                }
                else if (guess &gt; secretNumber)
                {
                    Console.WriteLine(&quot;もっと小さい数です！&quot;);
                }
                else
                {
                    Console.WriteLine(&quot;正解です！&quot;);
                    Console.WriteLine(&quot;試行回数: &quot; + attempts + &quot;回&quot;);
                }
            }
        }
    }
}

    break文とcontinue文

    break文（ループを途中で終了）

    ループを途中で強制終了させる命令です。 「もうこれ以上繰り返す必要がない」という場面で使います。 

    for (int i = 1; i &lt;= 10; i++)
{
    if (i == 5)
    {
        break; // i=5でループを終了
    }
    Console.WriteLine(i); // 1,2,3,4が表示される
}

    continue文（現在の繰り返しをスキップ）

    現在の繰り返し処理をスキップして、次の繰り返しに進む命令です。 「この回だけ処理をスキップしたいけど、ループ自体は続けたい」という場面で使います。 

    for (int i = 1; i &lt;= 5; i++)
{
    if (i == 3)
    {
        continue; // i=3の時はスキップ
    }
    Console.WriteLine(i); // 1,2,4,5が表示される
}

    break文とcontinue文の違い

    break文：ループ全体を終了

    for (int i = 1; i &lt;= 5; i++)
{
    if (i == 3) break;     // ここでループ全体が終了
    Console.WriteLine(i);   // 1, 2 のみ表示
}
Console.WriteLine(&quot;ループ終了&quot;); // ここに移動

    continue文：現在の繰り返しをスキップして次へ

    for (int i = 1; i &lt;= 5; i++)
{
    if (i == 3) continue;   // 3の時だけスキップ
    Console.WriteLine(i);   // 1, 2, 4, 5 が表示
}

    身近な例で例えると

    • break文：映画を途中で「もう見たくない」と思って映画館から出る
    • continue文：映画の嫌なシーンで「この場面だけ目を閉じる」けど映画は最後まで見る

    例題3：簡単な電卓プログラム

    まずは下記のコードを手作業で書き写して流れを理解しながら出力を確認しましょう

    using System;

namespace SimpleCalculator
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine(&quot;=== 簡単電卓 ===&quot;);
            Console.WriteLine(&quot;使用方法: 数値1 演算子 数値2 の形で入力&quot;);
            Console.WriteLine(&quot;例: 10 + 5&quot;);
            Console.WriteLine(&quot;終了するには &apos;exit&apos; と入力してください&quot;);

            string input = &quot;&quot;;

            while (input != &quot;exit&quot;)
            {
                Console.WriteLine(&quot;&quot;);
                Console.WriteLine(&quot;計算式を入力してください:&quot;);
                input = Console.ReadLine();

                if (input == &quot;exit&quot;)
                {
                    Console.WriteLine(&quot;電卓を終了します。&quot;);
                    break;
                }

                // 簡易的な計算処理（実際はもっと複雑な解析が必要）
                string[] parts = input.Split(&apos; &apos;);

                if (parts.Length == 3)
                {
                    double num1 = double.Parse(parts[0]);
                    string op = parts[1];
                    double num2 = double.Parse(parts[2]);
                    double result = 0;

                    if (op == &quot;+&quot;)
                        result = num1 + num2;
                    else if (op == &quot;-&quot;)
                        result = num1 - num2;
                    else if (op == &quot;*&quot;)
                        result = num1 * num2;
                    else if (op == &quot;/&quot;)
                        result = num1 / num2;
                    else
                    {
                        Console.WriteLine(&quot;未対応の演算子です&quot;);
                        continue;
                    }

                    Console.WriteLine(&quot;結果: &quot; + result);
                }
                else
                {
                    Console.WriteLine(&quot;正しい形式で入力してください&quot;);
                }
            }
        }
    }
}

    練習問題

    以下の機能を持つ「平均点計算プログラム」を作成してください： 1. 何科目分の点数を入力するか聞く 2. その回数だけ点数を入力してもらう 3. 平均点を計算して表示 4. 最高点と最低点も表示

    解答例

    using System;

namespace AverageCalculator
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine(&quot;=== 平均点計算プログラム ===&quot;);

            Console.WriteLine(&quot;何科目分の点数を入力しますか？&quot;);
            int subjectCount = int.Parse(Console.ReadLine());

            int total = 0;
            int highest = 0;
            int lowest = 100;

            for (int i = 1; i &lt;= subjectCount; i++)
            {
                Console.WriteLine(i + &quot;科目目の点数を入力してください:&quot;);
                int score = int.Parse(Console.ReadLine());

                total += score;

                if (score &gt; highest)
                    highest = score;

                if (score &lt; lowest)
                    lowest = score;
            }

            double average = (double)total / subjectCount;

            Console.WriteLine(&quot;&quot;);
            Console.WriteLine(&quot;=== 結果 ===&quot;);
            Console.WriteLine(&quot;科目数: &quot; + subjectCount);
            Console.WriteLine(&quot;合計点: &quot; + total + &quot;点&quot;);
            Console.WriteLine(&quot;平均点: &quot; + average.ToString(&quot;F1&quot;) + &quot;点&quot;);
            Console.WriteLine(&quot;最高点: &quot; + highest + &quot;点&quot;);
            Console.WriteLine(&quot;最低点: &quot; + lowest + &quot;点&quot;);
        }
    }
}

    do-while文

    do-while文とは

    条件チェックを「後で」行うループです。必ず最低1回は実行されます。 

    // while文：条件を先にチェック
while (条件)
{
    処理; // 条件がfalseなら1回も実行されない
}

// do-while文：処理を先に実行
do
{
    処理; // 必ず最低1回は実行される
} while (条件);

    活用例

    int number;
do
{
    Console.WriteLine(&quot;正の数を入力してください:&quot;);
    number = int.Parse(Console.ReadLine());
} while (number &lt;= 0);

    ネストしたループの活用例：パターン表示

    using System;

namespace PatternDisplay
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine(&quot;=== 三角形パターン ===&quot;);

            for (int i = 1; i &lt;= 5; i++)
            {
                // 空白を表示
                for (int space = 1; space &lt;= 5 - i; space++)
                {
                    Console.Write(&quot; &quot;);
                }

                // 星を表示
                for (int star = 1; star &lt;= i; star++)
                {
                    Console.Write(&quot;* &quot;);
                }

                Console.WriteLine(); // 改行
            }
        }
    }
}

    実践的な応用例：在庫管理システム

    using System;

namespace InventorySystem
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine(&quot;=== 簡易在庫管理システム ===&quot;);

            int totalItems = 0;
            string choice = &quot;&quot;;

            while (choice != &quot;4&quot;)
            {
                Console.WriteLine(&quot;&quot;);
                Console.WriteLine(&quot;=== メニュー ===&quot;);
                Console.WriteLine(&quot;1. 商品入荷&quot;);
                Console.WriteLine(&quot;2. 商品出荷&quot;);
                Console.WriteLine(&quot;3. 在庫確認&quot;);
                Console.WriteLine(&quot;4. 終了&quot;);
                Console.WriteLine(&quot;選択してください（1-4）:&quot;);

                choice = Console.ReadLine();

                if (choice == &quot;1&quot;)
                {
                    Console.WriteLine(&quot;入荷数を入力してください:&quot;);
                    int incoming = int.Parse(Console.ReadLine());
                    totalItems += incoming;
                    Console.WriteLine(incoming + &quot;個入荷しました。&quot;);
                }
                else if (choice == &quot;2&quot;)
                {
                    Console.WriteLine(&quot;出荷数を入力してください:&quot;);
                    int outgoing = int.Parse(Console.ReadLine());

                    if (outgoing &lt;= totalItems)
                    {
                        totalItems -= outgoing;
                        Console.WriteLine(outgoing + &quot;個出荷しました。&quot;);
                    }
                    else
                    {
                        Console.WriteLine(&quot;在庫が不足しています。&quot;);
                    }
                }
                else if (choice == &quot;3&quot;)
                {
                    Console.WriteLine(&quot;現在の在庫: &quot; + totalItems + &quot;個&quot;);
                }
                else if (choice == &quot;4&quot;)
                {
                    Console.WriteLine(&quot;システムを終了します。&quot;);
                }
                else
                {
                    Console.WriteLine(&quot;正しい選択肢を入力してください。&quot;);
                }
            }
        }
    }
}

    2. 商品出荷

    重要なポイント

    ループを使う前に考えること

    • 繰り返し回数が決まっている → for文が適している
    • 条件によって繰り返し回数が変わる → while文が適している
    • 最低1回は実行したい → do-while文を検討

    無限ループを防ぐために

    • while文では必ず条件を変更する処理を入れる（カウンタの更新など）
    • for文では増減式を正しく設定する
    • break文で確実に抜け出せる条件を用意する

    ループ制御文の使い分け

    • break文：ループ全体を終了したい時（目的達成、エラー回避など）
    • continue文：特定の条件の時だけスキップして続行したい時

    ネストしたループの注意点

    • 内側のループは外側のループの1回につき全回実行される
    • 変数名は区別する（外側i、内側j など）
    • 実行回数は掛け算になる（外側3回×内側4回＝合計12回）

    変数のスコープ（有効範囲）

    • for文で宣言した変数はそのfor文内でのみ使用可能
    • while文で使う変数は事前に宣言が必要

    まとめ

    ループの基本

    • 同じ処理を効率的に繰り返すための仕組み
    • 手動で同じコードを何度も書く必要がなくなる

    3つのループ文の特徴

    • for文：回数が決まっている繰り返しに最適（初期化、条件式、増減式が1箇所にまとまる）
    • while文：条件に基づく繰り返しに使用（繰り返し回数が事前に分からない場合）
    • do-while文：最低1回は実行したい場合に使用（条件チェックが後）

    ループ制御

    • break文でループを途中終了
    • continue文で現在の繰り返しをスキップして次へ
    • 適切な条件設定で無限ループを回避

    実践的な活用

    • ネストしたループで複雑なパターンや表を作成可能
    • ユーザー入力を繰り返し受け取る対話的プログラム
    • メニュー形式のアプリケーション作成
