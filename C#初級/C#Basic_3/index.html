    学習目標

    • Console.ReadLine()を使ってユーザーからの入力を受け取れるようになる
    • 文字列から数値への変換方法を理解する
    • 対話的なプログラムを作成できるようになる

    ユーザー入力の基本

    Console.ReadLine()

    プログラムを一時停止して、ユーザーがキーボードから文字を入力してEnterキーを押すまで待つメソッドです。 ユーザーが入力した内容を文字列として受け取ることができます。

    動作の詳細

    1.プログラムが止まる：Console.ReadLine()に到達すると、プログラムはユーザーの入力を待って一時停止する
    2.Enterキーが必要：ユーザーがEnterキーを押すまで入力は確定されない
    3.常にstring型：どんな内容を入力しても、結果は必ずstring（文字列）型になる 

    Console.WriteLine(&quot;名前を入力してください:&quot;);
// ↑ この行が実行されて画面に表示される

string name = Console.ReadLine();
// ↑ ここでプログラムが止まり、カーソルが点滅してユーザーの入力を待つ
// ユーザーが「太郎」と入力してEnterを押すと...
// name変数に「太郎」という文字列が保存される

Console.WriteLine(&quot;こんにちは&quot; + name + &quot;さん&quot;);
// ↑ ここで再びプログラムが動き出す

    型変換（文字列から数値へ）

    型変換とは？

    型変換とは、あるデータ型の値を別のデータ型に変えることです。

    Console.ReadLine()は常に文字列（string）を返すため、数値として使いたい場合は変換が必要です。

    int.Parse()（整数に変換）

    Console.WriteLine(&quot;年齢を入力してください:&quot;);
string ageText = Console.ReadLine();
int age = int.Parse(ageText);
Console.WriteLine(&quot;あなたは&quot; + age + &quot;歳ですね。&quot;);

    double.Parse()（小数に変換）

    Console.WriteLine(&quot;身長を入力してください（cm）:&quot;);
string heightText = Console.ReadLine();
double height = double.Parse(heightText);
Console.WriteLine(&quot;身長は&quot; + height + &quot;cmですね。&quot;);

    なぜ型変換が必要なのか

    Console.ReadLine()は、ユーザーが何を入力しても必ず文字列（string）として受け取ります。たとえユーザーが数字の「25」を入力したとしても、コンピュータにとっては「2という文字と5という文字が並んだもの」として認識されます。 

    // ユーザーが「25」と入力した場合
string ageText = Console.ReadLine(); // &quot;25&quot;（文字列）
// これは数字の25ではなく、「2」と「5」という文字の組み合わせ

int age = int.Parse(ageText); // 25（数値）
// int.Parse()で文字列の&quot;25&quot;を数値の25に変換

    計算するには数値に変換が必要

    文字列の"25"を数値の25に変換することで、初めて計算ができるようになります。 

    string input = Console.ReadLine(); // &quot;25&quot;（文字列）
int number = int.Parse(input);     // 25（数値）

int result = number + 10;          // 35（計算可能！）

    身近な例で理解する!

    これは、電卓で計算する時と似ています： 紙に「25」と書いてある → 文字として見ている状態
    その「25」を数値として理解して計算に使う → 数値に変換した状態

    よく使う型変換

    // 文字列 → 整数
string ageText = &quot;20&quot;;
int age = int.Parse(ageText);      // &quot;20&quot; → 20

// 文字列 → 小数
string heightText = &quot;175.5&quot;;
double height = double.Parse(heightText); // &quot;175.5&quot; → 175.5

// 数値 → 文字列
int score = 85;
string scoreText = score.ToString(); // 85 → &quot;85&quot;

    例題：簡単な計算機

    まずは下記のコードを手作業で書き写して流れを理解しながら出力を確認しましょう

    using System;

namespace SimpleCalculator
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine(&quot;=== 簡単な足し算計算機 ===&quot;);

            // 1つ目の数値を入力
            Console.WriteLine(&quot;1つ目の数値を入力してください:&quot;);
            string input1 = Console.ReadLine();
            int number1 = int.Parse(input1);

            // 2つ目の数値を入力
            Console.WriteLine(&quot;2つ目の数値を入力してください:&quot;);
            string input2 = Console.ReadLine();
            int number2 = int.Parse(input2);

            // 計算と結果表示
            int result = number1 + number2;
            Console.WriteLine(&quot;&quot;);
            Console.WriteLine(&quot;計算結果:&quot;);
            Console.WriteLine(number1 + &quot; + &quot; + number2 + &quot; = &quot; + result);
        }
    }
}

    実行結果

    === 簡単な足し算計算機 ===
    1つ目の数値を入力してください:
    50
    2つ目の数値を入力してください:
    5
    計算結果：
    50 + 5 = 55

    より実践的な例：BMI計算機

    using System;

namespace BMICalculator
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine(&quot;=== BMI計算機 ===&quot;);

            // 名前の入力
            Console.WriteLine(&quot;お名前を入力してください:&quot;);
            string name = Console.ReadLine();

            // 身長の入力
            Console.WriteLine(&quot;身長を入力してください（cm）:&quot;);
            double height = double.Parse(Console.ReadLine());

            // 体重の入力
            Console.WriteLine(&quot;体重を入力してください（kg）:&quot;);
            double weight = double.Parse(Console.ReadLine());

            // BMI計算（身長をメートルに変換）
            double heightInMeters = height / 100;
            double bmi = weight / (heightInMeters * heightInMeters);

            // 結果表示
            Console.WriteLine(&quot;&quot;);
            Console.WriteLine(&quot;=== 結果 ===&quot;);
            Console.WriteLine(name + &quot;さんのBMI: &quot; + bmi.ToString(&quot;F1&quot;));

            // BMIの判定
            Console.WriteLine(&quot;判定: &quot;);
            if (bmi &lt; 18.5)
                Console.WriteLine(&quot;低体重&quot;);
            else if (bmi &lt; 25)
                Console.WriteLine(&quot;普通体重&quot;);
            else
                Console.WriteLine(&quot;肥満&quot;);
        }
    }
}

    実行結果

    === BMI計算機 ===
    お名前を入力してください:
    A
    身長を入力してください（cm）：
    175
    体重を入力してください（kg）：
    68

    === 結果 ===
    AさんのBMI：22.2
    判定：
    普通体重

    練習問題

    以下の機能を持つ「買い物計算機」を作成してください： 1. 商品名を入力 2. 単価を入力 3. 個数を入力 4. 合計金額を計算して表示

    <br> <br> <br> <br>

    解答例

    using System;

namespace ShoppingCalculator
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine(&quot;=== 買い物計算機 ===&quot;);

            // 商品情報の入力
            Console.WriteLine(&quot;商品名を入力してください:&quot;);
            string productName = Console.ReadLine();

            Console.WriteLine(&quot;単価を入力してください（円）:&quot;);
            int unitPrice = int.Parse(Console.ReadLine());

            Console.WriteLine(&quot;個数を入力してください:&quot;);
            int quantity = int.Parse(Console.ReadLine());

            // 計算
            int totalPrice = unitPrice * quantity;

            // 結果表示
            Console.WriteLine(&quot;&quot;);
            Console.WriteLine(&quot;=== 計算結果 ===&quot;);
            Console.WriteLine(&quot;商品名: &quot; + productName);
            Console.WriteLine(&quot;単価: &quot; + unitPrice + &quot;円&quot;);
            Console.WriteLine(&quot;個数: &quot; + quantity + &quot;個&quot;);
            Console.WriteLine(&quot;合計金額: &quot; + totalPrice + &quot;円&quot;);
        }
    }
}
