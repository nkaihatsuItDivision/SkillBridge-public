    変数とデータ型の基礎

    text 変数の概念と基本的なデータ型について学習する。プログラムで情報を保存・操作する基礎を身につける。 レッスン2: 変数とデータ型の基礎 # レッスン2: 変数とデータ型の基礎

    学習目標

    • 変数の概念を理解する
    • 基本的なデータ型（int, string, double, bool）を使えるようになる
    • 変数に値を代入し、取得できるようになる

    変数とは

    変数は、プログラム内で値を保存するための「箱」のようなものです。箱には名前（変数名）があり、中に値を入れたり、取り出したりできます。

    なぜ変数が必要なのか

    同じ値を何度も使ったり、計算結果を保存したりするために 「名前付きの箱」として変数を使います。

    データ型とは

    コンピュータが値をどのように扱うかを決めるルールです。 数値なのか文字なのか、小数点があるのかないのかを プログラムに教えてあげる必要があります。

    基本的なデータ型

    1. int（整数型）

    名前の通り整数を扱う型
    下の画像のように足し算や引き算をすることも可能
    小数を入れるとエラーになるので、小数を扱いたいときはdouble型を使う 

    int Boys = 25; // OK
int Girls= 10; // OK
int Total = Boys + Girls; // Totalの値は35になる 

int Average = 2.5 // エラー！

    2. string（文字列型）

    文字列を扱う為の型
    +を使うことで文字同士を繋げることが可能 

    string Prefix = &quot;僕の名前は、&quot;;
string Name = &quot;竹内です。&quot;;

// 「僕の名前は、竹内です。」と出力される
Console.WriteLine(Prefix  + Name ):

    3. double（小数型）

    int型同様数値を扱う型
    こちらは小数点以下も扱うことが出来る 

    double Height = 1.75;
double Weight = 65.2;

// BMI = 体重(kg) ÷ 身長(m) ÷ 身長(ｍ)
double BMI = Weight/Height/Height;

    4. bool（真偽値型）

    ある条件などに当てはまっているか当てはまっていないかの判定結果を扱う型
    レッスン4で学ぶif文で使うことが多い 

    bool isStudent = true;
bool isWorking = false;

    変数の使い方

    宣言と初期化

    宣言とは

    宣言とは、コンピュータに「この名前の箱を用意してください」と伝えることです。箱の種類（データ型）と箱の名前（変数名）を決めます。この段階では、箱は空っぽの状態です。まだ何も入っていません。

    初期化とは

    初期化とは、用意した箱に最初の値を入れることです。 

    // 宣言と同時に初期化
int number = 10;

// 宣言後に値を代入
string greeting;
greeting = &quot;おはよう&quot;;

    値の変更

    値の変更とは、すでに値が入っている箱の中身を別の値に入れ替えることです。箱（変数）は同じですが、中身だけを新しいものに交換します。 

    int count = 5;
Console.WriteLine(count); // 5が表示
count = 10;
Console.WriteLine(count); // 10が表示

    例題：自己紹介プログラム

    まずは下記のコードを手作業で書き写して流れを理解しながら出力を確認しましょう

    using System;

namespace SelfIntroduction
{
    class Program
    {
        static void Main(string[] args)
        {
            // 変数の宣言と初期化
            string name = &quot;佐藤花子&quot;;
            int age = 20;
            double height = 165.0;
            bool isStudent = true;

            // 自己紹介の表示
            Console.WriteLine(&quot;=== 自己紹介 ===&quot;);
            Console.WriteLine(&quot;名前: &quot; + name);
            Console.WriteLine(&quot;年齢: &quot; + age + &quot;歳&quot;);
            Console.WriteLine(&quot;身長: &quot; + height + &quot;cm&quot;);
            Console.WriteLine(&quot;学生: &quot; + isStudent);
        }
    }
}

    練習問題

    あなた自身の情報（名前、年齢、趣味、出身地）を変数に保存し、表示するプログラムを作成してください。

    実行結果

    === 自己紹介 ===
    名前：佐藤花子
    年齢：20歳
    身長：165cm
    学生：True <br>

    解答例

    using System;

namespace MyInfo
{
    class Program
    {
        static void Main(string[] args)
        {
            // 自分の情報を変数に保存
            string myName = &quot;山田太郎&quot;;
            int myAge = 28;
            string hobby = &quot;読書&quot;;
            string hometown = &quot;東京&quot;;

            // 情報を表示
            Console.WriteLine(&quot;私について&quot;);
            Console.WriteLine(&quot;名前: &quot; + myName);
            Console.WriteLine(&quot;年齢: &quot; + myAge + &quot;歳&quot;);
            Console.WriteLine(&quot;趣味: &quot; + hobby);
            Console.WriteLine(&quot;出身地: &quot; + hometown);

            // 追加情報
            Console.WriteLine(&quot;&quot;);
            Console.WriteLine(&quot;よろしくお願いします！&quot;);
        }
    }
}

    実行結果

    私について
    名前：山田太郎
    年齢：28歳
    趣味：読書
    出身地：東京

    よろしくお願いします！

    重要なポイント

    • 変数名は意味のある名前を付ける（例：age、name）
    • 同じ型の値のみ代入可能
    • 文字列の連結は+記号を使用
    • 変数は使用前に必ず初期化する

    まとめ

    変数を使うことで、プログラム内で情報を保存・操作できるようになりました。次のレッスンでは、ユーザーからの入力を受け取る方法を学習します。
