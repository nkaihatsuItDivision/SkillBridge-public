    📋 ステップ3: 論理演算子による複合条件

    📋 ステップ4: LIKE演算子とパターンマッチング

    LIKE演算子の基本

    LIKE演算子は文字列のパターンマッチングに使用します。ワイルドカードと組み合わせて柔軟な検索が可能です。

    ワイルドカードの種類

    ワイルドカード 意味 使用例 マッチする例
    % 任意の0文字以上の文字列 '田%' 田中、田村、田
    _ 任意の1文字 '田_太郎' 田中太郎、田村太郎

    🔍 実践：LIKE演算子の活用

    %ワイルドカードによる部分一致検索

    -- 1. 名前が「田」で始まる学生
SELECT * FROM student WHERE name LIKE '田%';

    実行結果： 

    id | student_id | name     | age | department
1  | S001      | 田中太郎  | 20  | 情報工学部


    -- 2. 名前に「郎」が含まれる学生
SELECT * FROM student WHERE name LIKE '%郎%';

    実行結果： 

    id | student_id | name     | age | department

3  | S003      | 鈴木次郎  | 19  | 文学部
4  | S004      | 高橋一郎  | 22  | 理学部


    -- 3. 学部名に「工学」が含まれる学生
SELECT * FROM student WHERE department LIKE '%工学%';

    実行結果： 

    id | student_id | name     | age | department
1  | S001      | 田中太郎  | 20  | 情報工学部
5  | S005      | 山田美咲  | 20  | 工学部


    -- 4. 名前が「子」で終わる学生

    実行結果： 

    id | student_id | name     | age | department

    _ワイルドカードによる文字数指定検索

    📋 ステップ6: LIMITによる取得件数制限

    LIMITの基本構文

    SELECT フィールド名 FROM テーブル名 LIMIT 件数;
SELECT フィールド名 FROM テーブル名 LIMIT オフセット, 件数;
    • 件数: 取得するレコードの最大件数
    • オフセット: 開始位置（0から始まる）

    💡 よくある質問

    Q1: WHERE句でOR条件を使うとき、括弧は必要ですか？

    A: 複数の条件が混在する場合は括弧を使って明確にすることを強く推奨します：

    -- ❌ 分かりにくい（AND優先のため意図と異なる可能性）
SELECT * FROM student WHERE name LIKE '%田%' OR age = 20 AND department = '工学部';

-- ✅ 明確で理解しやすい
SELECT * FROM student WHERE name LIKE '%田%' OR (age = 20 AND department = '工学部');

    Q2: LIKE演算子で%を文字として検索したい場合は？

    A: エスケープ文字を使用します：

    -- 文字として「%」を検索
SELECT * FROM student WHERE department LIKE '%\%%' ESCAPE '\';
-- または
SELECT * FROM student WHERE department LIKE '%[%]%';

    Q3: ORDER BYで複数フィールドを指定する順序は重要ですか？

    A: 非常に重要です。左から順に優先度が決まります：

    -- 年齢優先 → 同じ年齢の中で名前順
SELECT * FROM student ORDER BY age ASC, name ASC;

-- 名前優先 → 同じ名前の中で年齢順（結果が大きく変わる）
SELECT * FROM student ORDER BY name ASC, age ASC;

    Q4: LIMITを使わずに最大値・最小値を取得する方法は？

    A: MAX()・MIN()関数を使用します（次のレッスンで詳細学習）：

    -- 最年長の年齢
SELECT MAX(age) FROM student;

-- 最年少の学生（サブクエリ使用）
SELECT * FROM student WHERE age = (SELECT MIN(age) FROM student);
