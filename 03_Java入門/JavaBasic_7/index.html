    1. なぜ int と String だけでは不十分なのか？

    現実世界のデータの多様性

    プログラムで扱うデータを考えてみてください：

    数値データの例： - 学生の年齢：20歳 → 小さな整数で十分 - 世界の人口：約80億人 → とても大きな数値が必要 - 商品の価格：298.5円 → 小数点が必要 - 文字コード：65 = 'A' → 文字1個を数値で表現

    現在のint型だけでは： 

    int age = 20;           // ✅ 年齢：問題なし
int worldPopulation = 8000000000;  // ❌ エラー！大きすぎる
int price = 298.5;      // ❌ エラー！小数点は使えない

    📘 なぜ int と String だけでは不十分なのか？

    このように、様々な種類・サイズのデータに対応するため、Javaには8つの基本データ型が用意されています！

    2. Javaの8つの基本データ型（プリミティブ型）

    データ型の分類

    Javaの基本データ型は、大きく4つのカテゴリに分類されます：

    📘 データ型の一覧表

    カテゴリ データ型 サイズ 説明
    整数型 byte 8bit 非常に小さな整数（-128〜127）
    short 16bit 小さな整数（-32,768〜32,767）
    int 32bit 一般的な整数（約-21億〜21億）
    long 64bit 非常に大きな整数（約-922京〜922京）
    浮動小数点型 float 32bit 単精度小数点数（約7桁精度）
    double 64bit 倍精度小数点数（約15桁精度）
    文字型 char 16bit 文字1個（'A', 'あ', '1'など）
    論理型 boolean 1bit 真偽値（true/false）

    太字は今まで使ってきたものです！

    3. 整数型（4種類）の詳細

    なぜ4種類も整数型があるのか？

    メモリ効率と表現範囲のバランスを取るためです！

    📘 メモリ領域の比較図

    s

    各整数型の特徴

    byte型：最小の整数型

    byte smallNumber = 100;    // -128 から 127 まで
byte temperature = -5;     // 気温など小さな数値に適している

    使用場面： - ファイルのデータ処理 - 色の値（RGB: 0-255） - 温度、点数など小さな範囲の数値

    short型：小さな整数型

    short year = 2024;         // -32,768 から 32,767 まで
short population = 15000;  // 小さな町の人口など

    使用場面： - 年数、月数 - 小規模なカウント値 - 座標値（画面上の位置）

    int型：標準的な整数型（既習）

    int age = 25;              // 約-21億 から 21億 まで
int salary = 3000000;      // 年収など一般的な数値

    使用場面： - 年齢、給与、点数 - 一般的な計算 - 最も頻繁に使用される整数型

    long型：大きな整数型

    long worldPopulation = 8000000000L;  // 約-922京 から 922京 まで
long timeMillis = 1642678800000L;    // 時間のミリ秒表現

    重要：long型のリテラルには末尾にLをつける！

    実際にプログラムで確認してみよう

    public class IntegerTypes {
    public static void main(String[] args) {
        // 各整数型の宣言と表示
        byte byteValue = 127;           // byte型の最大値
        short shortValue = 32767;       // short型の最大値
        int intValue = 2147483647;      // int型の最大値
        long longValue = 9223372036854775807L; // long型の最大値

        System.out.println("byte型: " + byteValue);
        System.out.println("short型: " + shortValue);
        System.out.println("int型: " + intValue);
        System.out.println("long型: " + longValue);

        // サイズの確認（後で詳しく学習）
        System.out.println("byte型のサイズ: " + Byte.BYTES + "バイト");
        System.out.println("short型のサイズ: " + Short.BYTES + "バイト");
        System.out.println("int型のサイズ: " + Integer.BYTES + "バイト");
        System.out.println("long型のサイズ: " + Long.BYTES + "バイト");
    }
}

    4. 浮動小数点型（2種類）の詳細

    小数点を扱う型

    整数型では表現できない小数値を扱うための型です。

    float型：単精度浮動小数点

    float price = 298.5f;      // 末尾にfが必要
float average = 85.7f;     // 約7桁の精度

    重要：float型のリテラルには末尾にfをつける！

    double型：倍精度浮動小数点

    double pi = 3.141592653589793;    // 高精度な円周率
double temperature = 36.8;        // 体温など精密な値

    特徴： - デフォルト：小数点のある数値は自動的にdouble型 - 高精度：約15桁の精度 - 推奨：小数点を使う場合は基本的にdouble型を使用

    浮動小数点型のプログラム例

    public class FloatTypes {
    public static void main(String[] args) {
        // float型とdouble型の比較
        float floatPi = 3.141592653589793f;   // float（精度制限あり）
        double doublePi = 3.141592653589793;  // double（高精度）

        System.out.println("float型のπ:  " + floatPi);
        System.out.println("double型のπ: " + doublePi);

        // 計算例
        double radius = 5.0;
        double area = doublePi * radius * radius;
        System.out.println("半径" + radius + "の円の面積: " + area);
    }
}

    5. 文字型（char）と論理型（boolean）

    char型：文字1個を表現

    char grade = 'A';          // 成績のA
char initial = '田';       // 日本語の文字
char digit = '5';          // 数字の文字（数値の5とは異なる）
char symbol = '@';         // 記号

    重要ポイント： - シングルクォート（'）で囲む - 1文字のみ格納可能 - 内部的には数値（Unicode）で管理

    boolean型：真偽値（既習の復習）

    boolean isPass = true;     // 合格したかどうか
boolean isAdult = false;   // 成人かどうか

    char型とboolean型のプログラム例

    public class CharBoolean {
    public static void main(String[] args) {
        // char型の使用例
        char grade = 'A';
        char firstLetter = '田';
        char number = '7';

        System.out.println("成績: " + grade);
        System.out.println("名前の最初の文字: " + firstLetter);
        System.out.println("数字文字: " + number);

        // char型の数値的側面
        System.out.println("'A'の数値: " + (int)grade);  // 65が表示される

        // boolean型の使用例
        boolean hasLicense = true;
        boolean isWeekend = false;

        System.out.println("免許を持っている: " + hasLicense);
        System.out.println("週末である: " + isWeekend);
    }
}

    💡 よくある質問

    Q1: なぜlong型やfloat型には末尾にLやfが必要なの？

    A: Javaのデフォルト動作との区別のためです：

    // デフォルトの動作
int defaultInt = 123;        // 整数はデフォルトでint型
double defaultDouble = 1.23; // 小数はデフォルトでdouble型

// 明示的な指定が必要
long longValue = 123L;       // L がないとint型と解釈される
float floatValue = 1.23f;    // f がないとdouble型と解釈される

    Q2: char型とString型の違いは何ですか？

    A: 文字数の違いです：

    char singleChar = 'A';       // 文字1個のみ（シングルクォート）
String multiChar = "ABC";    // 文字列（0個以上の文字）（ダブルクォート）
String oneChar = "A";        // これも文字列（文字1個だが文字列型）

    Q3: どのデータ型を選べばいいか迷います

    A: 迷ったときの鉄則： 1. 整数: とりあえずint 2. 小数: とりあえずdouble 3. 文字1個: char 4. 文字列: String 5. 真偽値: boolean

    慣れてきたら効率を考えて他の型を検討しましょう。

    Q4: byte型の範囲はなぜ-128から127なの？

    A: 8bit（1バイト）の2進数表現のためです： - 8bitで表現できる数値：2⁸ = 256通り - 符号あり整数：-128 〜 +127 = 256通り - 詳しくは後のレッスンで学習します

    Q5: データ型を間違えるとどうなりますか？

    A: コンパイルエラーになることが多いです：

    byte tooLarge = 200;         // エラー！byteの範囲外
int decimal = 3.14;          // エラー！intに小数は入らない
char multiple = 'AB';        // エラー！charは1文字のみ
