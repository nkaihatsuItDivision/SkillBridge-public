    1. プログラムに「選択」させる

    日常生活での条件分岐

    私たちは日常的に「もし〜なら、〜する」という判断をしています：

    • もし雨が降っているなら、傘を持って行く
    • もし信号が赤なら、止まる
    • もし時間があるなら、コーヒーを飲む

    プログラムでも同じように、条件によって処理を変えることができます！

    プログラムでの条件分岐

    Javaではif文を使って条件分岐を実現します：

    if (条件) {
    // 条件がtrueの時に実行される処理
}

    📘 条件分岐の概念

    2. if文の基本構文

    基本的な書き方

    if (条件式) {
    // 条件がtrueの時に実行される処理
}

    重要なポイント： - ifの後に丸括弧で条件を囲む - 条件の後に波括弧で処理をまとめる - 条件式の結果はboolean型（true または false）

    実際の例

    public class ConditionalBasic {
    public static void main(String[] args) {
        int age = 20;

        if (age >= 18) {
            System.out.println("成人です");
        }

        System.out.println("プログラム終了");
    }
}

    実行結果： 

    成人です
プログラム終了

    条件がfalseの場合

    public class ConditionalBasic {
    public static void main(String[] args) {
        int age = 15;  // 18未満に変更

        if (age >= 18) {
            System.out.println("成人です");
        }

        System.out.println("プログラム終了");
    }
}

    実行結果： 

    プログラム終了

    条件がfalseの場合、if文の中身は実行されません！

    4. else文で「そうでなければ」を表現

    else文の基本構文

    if (条件式) {
    // 条件がtrueの時の処理
} else {
    // 条件がfalseの時の処理
}

    実際の例

    public class IfElseExample {
    public static void main(String[] args) {
        int score = 75;

        if (score >= 60) {
            System.out.println("合格です！");
        } else {
            System.out.println("不合格です");
        }

        System.out.println("成績: " + score + "点");
    }
}

    実行結果： 

    合格です！
成績: 75点

    条件がfalseの場合

    public class IfElseExample {

            System.out.println("合格です！");
        } else {
            System.out.println("不合格です");
        }

        System.out.println("成績: " + score + "点");

}

    不合格です
成績: 45点

    5. 実践的な条件分岐プログラム

    複数の判定を組み合わせる

    public class PracticalCondition {
    public static void main(String[] args) {
        // 年齢チェック
        int age = 22;
        if (age >= 20) {
            System.out.println("成人なのでお酒が飲めます");
        } else {
            System.out.println("未成年なのでお酒は飲めません");
        }

        // 天気チェック
        boolean isSunny = false;
        if (isSunny) {
            System.out.println("洗濯物を外に干しましょう");
        } else {
            System.out.println("洗濯物は室内に干しましょう");
        }

        // 予算チェック
        int budget = 800;
        int itemPrice = 1200;
        if (budget >= itemPrice) {
            System.out.println("商品を購入できます");
        } else {
            System.out.println("予算が足りません");
        }
    }
}

    実行結果： 

    成人なのでお酒が飲めます
洗濯物は室内に干しましょう
予算が足りません

    if (条件) {
    // 処理
} else {
    // 処理

    💡 ヒント2: 条件の書き方 - 25度以上かどうかを判定する必要があります - 比較演算子（>=, <）を使いましょう

    💡 ヒント3: temperature変数の使い方 

    int temperature = 28;
if (temperature >= ○○) {
    // 25度以上の場合の処理
}

    課題2：パスワード長さチェックプログラム

    パスワードの長さが適切かどうかをチェックするプログラムを作成して見ましょう。

    要件： - パスワードの長さが8文字以上の場合：「安全なパスワードです」 - パスワードの長さが8文字未満の場合：「パスワードが短すぎます」 - パスワードを"abc123"として設定 - 文字列の長さは.length()で取得できます

    まずは自分で考えてみよう！

    💡 ヒント1: 文字列の長さ取得

int length = password.length();  // 文字数を取得

    💡 ヒント2: lengthを使った条件

    // 8文字以上の場合

    // 8文字未満の場合
}

    💡 ヒント3: 完全な構造 

    String password = "abc123";
int length = password.length();
System.out.println("パスワード: " + password);
System.out.println("文字数: " + length);
// ここにif文を書く

    🎯 期待される実行結果 

    パスワード: abc123
文字数: 6
パスワードが短すぎます

    完成例

    public class PasswordCheck {
    public static void main(String[] args) {
        String password = "abc123";
        int length = password.length();  // 文字列の長さを取得

        System.out.println("パスワード: " + password);
        System.out.println("文字数: " + length);

        if (length >= 8) {
            System.out.println("安全なパスワードです");
        } else {
            System.out.println("パスワードが短すぎます");
        }
    }
}

    7. よくあるエラーと対処法

    エラー1: セミコロン忘れ

    // ❌ 間違い
if (age >= 18) {
    System.out.println("成人です")  // セミコロンがない
}

// ✅ 正しい
if (age >= 18) {
    System.out.println("成人です");  // セミコロンを追加
}

    エラー2: 波括弧の対応が間違い

    // ❌ 間違い
if (age >= 18) {
    System.out.println("成人です");
// else文の波括弧が足りない

// ✅ 正しい
if (age >= 18) {
    System.out.println("成人です");
} else {
    System.out.println("未成年です");
}

    エラー3: 条件式の括弧忘れ

    // ❌ 間違い
if age >= 18 {  // 括弧がない

// ✅ 正しい
if (age >= 18) {  // 括弧で条件を囲む

    💡 よくある質問

    Q1: if文の中に別のif文を書けますか？

    A: はい、可能です！これを「ネスト（入れ子）」と呼びます。ただし、今回は基本を覚えることに集中しましょう。

    if (age >= 18) {
    if (hasLicense) {  // if文の中にif文
        System.out.println("運転できます");
    }
}

    Q2: else文は必ず書く必要がありますか？

    A: いいえ、必須ではありません。条件がtrueの時だけ処理したい場合は、if文だけで十分です。

    Q3: 条件に複数の条件を組み合わせられますか？

    A: はい！&amp;&amp;（かつ）や||（または）を使って組み合わせできます。詳細は後のレッスンで学習します。
