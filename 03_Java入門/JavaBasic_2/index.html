    1. 変数とは何か？

    身近な例で理解しよう

    変数を理解するために、身近な例から考えてみましょう。

    例1: 財布の中のお金 

    今日の朝：財布に 5000円 入っている
昼食後：財布に 4500円 入っている（500円使った）
夕方：財布に 3500円 入っている（さらに1000円使った）

    この「財布」は、常に現在の金額を覚えてくれています。

    例2: スマホの連絡先 

    田中さんの電話番号：090-1234-5678
山田さんの電話番号：080-9876-5432

    「田中さん」という名前に、電話番号というデータが結びついています。

    📘 変数の基本概念①

    プログラムでの変数

    プログラムの世界でも、同じ考え方です：

    • 変数は「データを入れる箱」
    • 箱には名前がついている
    • 中身（データ）は変更できる
    // 変数のイメージ
int age;        // 「age」という名前の箱を用意
age = 25;       // その箱に「25」を入れる

    📘 変数の基本概念②

    2. int型 - 整数を扱う変数

    int型とは？

    intは「integer（整数）」の略で、数値（整数）を扱うための変数の種類です。

    int age = 25;           // 年齢
int score = 85;         // テストの点数
int count = 100;        // カウント数

    int型の基本的な使い方

    ステップ1: 変数の宣言

    int age;    // 「age」という名前のint型変数を用意

    ステップ2: 値の代入

    age = 25;   // age変数に25を入れる

    ステップ3: 宣言と代入を同時に

    int age = 25;   // 宣言と代入を1行で

    📘 int型の具体例

    int型で扱える数値

    int positive = 100;      // 正の数
int negative = -50;      // 負の数
int zero = 0;           // ゼロ

    注意: int型は整数のみです。小数点は扱えません。

    int score = 85;     // ✅ 正しい
int price = 150;    // ✅ 正しい
int average = 3.5;  // ❌ エラー！小数点は使えない

    3. String型 - 文字列を扱う変数

    String型とは？

    Stringは文字列（文字の集まり）を扱うための変数の種類です。

    String name = "田中太郎";
String message = "こんにちは";
String email = "tanaka@example.com";

    String型の基本的な使い方

    文字列は必ずダブルクォート（"）で囲む

    String name = "田中太郎";      // ✅ 正しい
String city = "東京";          // ✅ 正しい
String name = 田中太郎;        // ❌ エラー！クォートがない
String city = &apos;東京&apos;;          // ❌ エラー！シングルクォートは使えない

    空の文字列も使える

    String emptyString = "";      // 空の文字列
String message = "Hello";     // 文字列

    📘 文字列リテラルと引用符の説明

    数字も文字列として扱える

    int age = 25;           // 数値として扱う
String ageText = "25";  // 文字列として扱う

    この2つは全く別のものです！

        // 変数の内容を表示

        System.out.println("年齢: " + age);
        System.out.println("住所: " + city);
        System.out.println("点数: " + score);
    }
}

    7. 変数の命名規則

    変数に名前をつけるときのルールがあります。

    必須ルール（守らないとエラー）

    // ✅ 正しい命名
int age;
String userName;
int score1;
String email_address;

// ❌ エラーになる命名
int 123name;        // 数字から始まる
String user-name;   // ハイフンは使えない
int class;          // Javaの予約語は使えない
String 名前;         // 日本語は避ける

    推奨ルール（読みやすいコード）

    キャメルケース（camelCase）

    // ✅ 推奨：キャメルケース
String firstName = "太郎";
int totalScore = 300;
String emailAddress = "test@example.com";

// ❌ 非推奨：読みにくい
String firstname = "太郎";
String FIRSTNAME = "太郎";
String first_name = "太郎";

    意味のある名前をつける

    // ✅ 良い例：何の変数かわかる
int studentAge = 20;
String customerName = "田中";
int testScore = 85;

// ❌ 悪い例：何の変数かわからない
int a = 20;
String x = "田中";
int num = 85;

    💡 よくある質問

    Q1: int と String の違いがよくわかりません

    A: 簡単に言うと： - int: 計算できる数値（年齢、点数、個数など） - String: 文字として扱うもの（名前、住所、メッセージなど）

    int age = 25;        // 25という数値（計算可能）
String name = "25";  // "25"という文字列（計算不可）

    Q2: 変数名に日本語は使えないのですか？

    A: 技術的には使えますが、強く非推奨です。理由： - 文字化けの可能性 - チーム開発で問題になる - 業界標準は英語

    Q3: System.out.println の + はどういう意味ですか？

    A: 文字列連結の意味です。文字列と他の値をつなげて1つの文字列にします。

    "こんにちは" + "世界"  // → "こんにちは世界"
"年齢は" + 25 + "歳"   // → "年齢は25歳"

    Q4: 変数を宣言しただけで値を入れないとどうなりますか？

    A: エラーになります。Javaでは使用前に必ず値を代入する必要があります。

    int age;                    // 宣言のみ
System.out.println(age);    // ❌ エラー！値が入っていない

int score = 0;              // ✅ 正しい：値を代入してから使用
System.out.println(score);
