    1. なぜ美しい出力が重要なのか？

    プログラムの価値を決める「見た目」

    プログラムは 「動けばいい」だけではありません。ユーザーが実際に使うとき、最も重要なのは 「使いやすさ」 です。

    例：ATMの画面 

    // ❌ 機能的だが見にくい出力
残高1234567円引き出し10000円手数料110円新残高1224457円

// ✅ 見やすく整理された出力
================================
        ATM 取引明細
================================
残高:           1,234,567円
引き出し:          10,000円
手数料:               110円
--------------------------------
新残高:         1,224,457円
================================

    どちらが 安心して使える でしょうか？

    プログラマーとしての品質意識

    美しい出力を作ることで、以下のスキルが身につきます： - ユーザー目線 での設計思考 - 細部への注意力 - プロフェッショナル意識 - 保守しやすいコード の作成能力

    2. System.out.print vs println の使い分け

    基本的な違い

    これまで主に使っていた println と、新しく学ぶ print の違いを確認しましょう。

    public class PrintComparison {
    public static void main(String[] args) {
        // println：改行あり（これまで使っていた方法）
        System.out.println("Hello");
        System.out.println("World");

        System.out.println("---"); // 区切り線

        // print：改行なし（新しい方法）
        System.out.print("Hello");
        System.out.print("World");
        System.out.println(); // 手動で改行

        System.out.println("---"); // 区切り線

        // print を活用した連続出力
        System.out.print("商品: ");
        System.out.print("ノートPC ");
        System.out.print("価格: ");
        System.out.println("89,800円");
    }
}

    実行結果: 

    Hello
World
---
HelloWorld
---
商品: ノートPC 価格: 89,800円

    📘 printとprintlnの違い

    実用的な使い分け

    表形式の出力で威力を発揮

    public class TableExample {
    public static void main(String[] args) {
        // 表のヘッダー
        System.out.println("商品名        価格      数量   小計");

    }

    3. エスケープシーケンス：特殊文字の活用

    エスケープシーケンスとは？

    エスケープシーケンス は、通常の文字では表現できない 特殊文字 を表現する方法です。

    📘 エスケープシーケンスの視覚化

    基本的なエスケープシーケンス

    public class EscapeSequenceExample {
    public static void main(String[] args) {
        // \n：改行（newline）
        System.out.println("1行目\n2行目\n3行目");

        System.out.println("---");

        // \t：タブ（tab）
        System.out.println("名前\t年齢\t職業");
        System.out.println("田中\t25\t会社員");
        System.out.println("佐藤\t30\tエンジニア");

        System.out.println("---");

        // \"：ダブルクォート
        System.out.println("彼は\"こんにちは\"と言いました");

        // \\：バックスラッシュ
        System.out.println("ファイルパス: C:\\Users\\Documents\\file.txt");
    }
}

    4. System.out.printf：プロ級のフォーマット出力

    printfとは？

    printf は 「print formatted」 の略で、フォーマット指定 を使って美しい出力を作る専用メソッドです。

    基本的な使い方

    public class PrintfBasic {
    public static void main(String[] args) {
        String name = "田中太郎";
        int age = 25;
        double height = 170.5;

        // 基本的なprintf
        System.out.printf("名前: %s\n", name);
        System.out.printf("年齢: %d歳\n", age);
        System.out.printf("身長: %.1fcm\n", height);

        // 複数の値を同時に出力
        System.out.printf("こんにちは、%sさん！あなたは%d歳ですね。\n", name, age);
    }
}

    実行結果: 

    名前: 田中太郎
年齢: 25歳
身長: 170.5cm
こんにちは、田中太郎さん！あなたは25歳ですね。

    📘 printfの書式指定子

    主要なフォーマット指定子

    指定子 意味 使用例 結果
    %s 文字列（String） printf("名前: %s", "田中") 名前: 田中
    %d 整数（decimal） printf("年齢: %d", 25) 年齢: 25
    %f 小数（float） printf("価格: %.2f", 1234.5) 価格: 1234.50
    %c 文字（character） printf("等級: %c", &A&) 等級: A
    %x 16進数（hexadecimal） printf("色: #%06x", 255) 色: #0000ff

    5. printfの高度な活用：桁数指定と桁揃え

    桁数指定で美しい表を作成

    public class AdvancedPrintf {
    public static void main(String[] args) {
        System.out.println("=== 売上レポート ===");

        // ヘッダー（右揃えで統一）
        System.out.printf("%10s %8s %6s %10s\n", "商品名", "単価", "数量", "売上");

        System.out.printf("%-10s %8d %6d %10d\n", "キーボード", 5400, 18, 97200);

        System.out.println("------------------------------------------");
        System.out.printf("%-10s %8s %6s %10d\n", "合計", "", "", 1249300);
    }

    public class DecimalFormatting {
    public static void main(String[] args) {
        double[] prices = {1234.567, 89.9, 12345.0, 0.99};

        System.out.println("価格一覧（小数点以下2桁）:");
        System.out.println("------------------------");

        for (int i = 0; i < prices.length; i++) {
            System.out.printf("商品%d: %8.2f円\n", i + 1, prices[i]);

        System.out.println("------------------------");

        // 平均価格の計算と表示
        double total = 0;
        for (double price : prices) {
            total += price;
        }
        double average = total / prices.length;

        System.out.printf("平均価格: %.2f円\n", average);
    }
}

    完成例

    public class BeautifulShoppingSystem {
    public static void main(String[] args) {
        // 商品データ
        String product1 = "ノートPC";
        int price1 = 89800;
        int quantity1 = 2;

        String product2 = "マウス";
        int price2 = 2980;
        int quantity2 = 1;

        // 小計計算
        int subtotal1 = price1 * quantity1;
        int subtotal2 = price2 * quantity2;
        int totalSubtotal = subtotal1 + subtotal2;

        // 税計算
        double taxRate = 0.10;
        int tax = (int) (totalSubtotal * taxRate);
        int finalTotal = totalSubtotal + tax;

        // 美しいレシート出力
        System.out.println("========================================");
        System.out.println("           ショッピングレシート");
        System.out.println("========================================");

        // ヘッダー
        System.out.printf("%-15s %8s %6s %10s\n", "商品名", "単価", "数量", "小計");
        System.out.println("----------------------------------------");

        // 商品明細
        System.out.printf("%-15s %,8d %6d %,10d円\n", product1, price1, quantity1, subtotal1);
        System.out.printf("%-15s %,8d %6d %,10d円\n", product2, price2, quantity2, subtotal2);

        System.out.println("----------------------------------------");

        // 集計部分（右揃え）
        System.out.printf("%32s %,10d円\n", "小計:", totalSubtotal);
        System.out.printf("%32s %9.0f%%\n", "税率:", taxRate * 100);
        System.out.printf("%32s %,10d円\n", "税額:", tax);

        System.out.println("========================================");
        System.out.printf("%32s %,10d円\n", "合計:", finalTotal);
        System.out.println("========================================");

        System.out.println("ありがとうございました！");
    }
}

    3. %8d: 右揃えで幅8文字

    💡 よくある質問

    Q1: printfとprintlnはどう使い分けるべきですか？

    A: 以下のように使い分けましょう：

    • printf: フォーマットが必要な場合（桁揃え、小数点制御など）
    • println: シンプルな出力で十分な場合
    // printfが適している場面
System.out.printf("価格: %,8d円\n", price);

// printlnで十分な場面
System.out.println("処理が完了しました");

    Q2: printfで改行はどうやって入れるのですか？

    A: printfでは \n を使って改行します：

    System.out.printf("1行目\n2行目\n");
// または
System.out.printf("1行目%n2行目%n"); // %nも使える

    Q3: 文字列の中にダブルクォートを表示したいのですが？

    A: \" エスケープシーケンスを使います：

    System.out.println("彼は\"こんにちは\"と言いました");
// 結果: 彼は"こんにちは"と言いました

    Q4: 数値の桁数が固定されていない場合、どうやって桁揃えをしますか？

    A: printfの幅指定機能を使います：

    // 右揃えで8桁確保
System.out.printf("%8d\n", 123);     // "     123"
System.out.printf("%8d\n", 45678);   // "   45678"

// 左揃えで8桁確保
System.out.printf("%-8d\n", 123);    // "123     "

    Q5: カンマ区切りの数値表示ができません

    A: %,d を使います：

    System.out.printf("%,d円\n", 1234567);  // 1,234,567円
