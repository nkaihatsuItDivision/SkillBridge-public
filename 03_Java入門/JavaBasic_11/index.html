    個別学習 vs 統合実践の違い

    今まで個別に学習してきた技術を、実際のプログラムでは組み合わせて使います：

    // 個別学習の例（レッスン7-10）
int age = 25;                    // データ型
String name = "田中";             // String型
boolean isAdult = (age >= 18);   // 比較演算子
age++;                          // インクリメント

// 統合実践の例（今回）
public static String checkDiscountEligibility(String customerName, int age, double purchaseAmount) {
    // 複数の技術を組み合わせた実用的な処理
    boolean isAdult = (age >= 18);
    boolean isSenior = (age >= 65);
    boolean isLargePurchase = (purchaseAmount >= 10000.0);

    String result = (customerName != null && !customerName.equals("")) ? customerName : "お客様";
    result += isAdult ? "（成人）" : "（未成年）";

    double discountRate = isSenior ? 0.15 : (isLargePurchase ? 0.10 : 0.05);

    return result + " - 割引率: " + (int)(discountRate * 100) + "%";
}

    実際の開発での重要性

    プログラマーは個別の技術を組み合わせる判断力が最も重要です：

    • データ型選択: どの型が最適？（int vs long, float vs double）
    • 演算子の組み合わせ: 効率的で読みやすい条件式の作成
    • 型変換のタイミング: 精度を保ちながら適切に変換
    • エラー対応: 統合した際に起こりやすい問題の解決

    練習1: データ型選択の実践判断

    以下の要件に最適なデータ型を選択し、その理由を考えてみましょう：

    public class DataTypeSelection {
    public static void main(String[] args) {
        // 要件1: 人の年齢（0-150歳程度）
        int age = 25;  // int で十分（byteでも可能だが、計算を考慮してintが安全）

        // 要件2: 銀行残高（円単位、数億円まで）
        long balance = 150000000L;  // intの範囲を超える可能性があるためlong

        // 要件3: 体温（小数点2桁の精度）
        double bodyTemperature = 36.5;  // 精度を考慮してdouble

        // 要件4: 商品の在庫数（0-10000個程度）
        int stockCount = 500;  // int で十分

        // 要件5: 割引率（パーセント、小数点1桁）
        float discountRate = 15.5f;  // float で十分な精度

        System.out.println("年齢: " + age + "歳");
        System.out.println("残高: " + balance + "円");
        System.out.println("体温: " + bodyTemperature + "度");
        System.out.println("在庫: " + stockCount + "個");
        System.out.println("割引: " + discountRate + "%");
    }
}

    選択のポイント: - 範囲: 扱う数値の最大・最小値を考慮 - 精度: 小数点が必要か、どの程度の精度が必要か - パフォーマンス: 不要に大きな型を使わない - 将来性: 要件変更に対応できる余裕

    練習2: 演算子の効果的な組み合わせ

    複数の演算子を組み合わせて、効率的な条件判定を作成しましょう：

    public class OperatorCombination {
    public static void main(String[] args) {
        int score = 85;
        int attendance = 90;
        boolean isSubmitted = true;

        // 複数条件の効果的な組み合わせ
        boolean isPassing = (score >= 60) && (attendance >= 80) && isSubmitted;

        // 三項演算子で簡潔な条件分岐
        String grade = (score >= 90) ? "A" : 
                      (score >= 80) ? "B" : 
                      (score >= 70) ? "C" : 
                      (score >= 60) ? "D" : "F";

        // 複合代入演算子で効率的な更新
        score += 5;  // ボーナス点加算
        attendance *= 1.1;  // 10%増しで評価

        // 論理演算子の短絡評価を活用
        boolean isExcellent = (score >= 85) && (attendance >= 95) && isSubmitted;

        System.out.println("合格: " + isPassing);
        System.out.println("成績: " + grade);
        System.out.println("優秀: " + isExcellent);
        System.out.println("最終点数: " + score);
        System.out.println("最終出席率: " + (int)attendance + "%");
    }
}

    練習3: 型変換の実践活用

    実際の場面で発生する型変換を適切に処理しましょう：

    public class TypeConversionPractice {
    public static void main(String[] args) {
        // 文字列として受け取った数値データの処理
        String priceText = "1234.56";
        String quantityText = "10";

        try {
            // String → 数値型変換
            double price = Double.parseDouble(priceText);
            int quantity = Integer.parseInt(quantityText);

            // 計算処理（型変換が自動的に発生）
            double total = price * quantity;  // int → double への暗黙的変換

            // 表示用の型変換
            int totalInt = (int)total;  // 明示的キャスト（小数点切り捨て）
            long totalLong = Math.round(total);  // 四捨五入

            System.out.println("単価: " + price + "円");
            System.out.println("数量: " + quantity + "個");
            System.out.println("合計: " + total + "円");
            System.out.println("合計（整数）: " + totalInt + "円");
            System.out.println("合計（四捨五入）: " + totalLong + "円");

            // パーセント計算での型変換活用
            double taxRate = 0.10;
            double tax = total * taxRate;
            double finalAmount = total + tax;

            System.out.println("税額: " + (int)tax + "円");
            System.out.println("税込み総額: " + (int)finalAmount + "円");

        } catch (NumberFormatException e) {
            System.out.println("数値変換エラー: 正しい数値を入力してください");
        }
    }
}

    完成例

    public class ShoppingCalculator {
    public static void main(String[] args) {
        // 顧客情報の設定
        String customerName = "田中花子";
        int age = 68;
        int memberRank = 3;

        // 商品情報の設定
        String productName = "ノートパソコン";
        double unitPrice = 89800.0;
        int quantity = 1;

        // 基本計算
        double subtotal = unitPrice * quantity;

        // 割引計算
        double discountRate = calculateMaxDiscount(age, memberRank, subtotal);
        double discountAmount = subtotal * discountRate;
        double total = subtotal - discountAmount;

        // 結果表示
        displayShoppingResult(customerName, age, memberRank, productName, 
                            unitPrice, quantity, subtotal, discountRate, 
                            discountAmount, total);
    }

    // 最大割引率を計算するメソッド
    public static double calculateMaxDiscount(int age, int memberRank, double amount) {
        double seniorDiscount = (age >= 65) ? 0.15 : 0.0;
        double vipDiscount = (memberRank >= 4) ? 0.10 : 0.0;
        double largePurchaseDiscount = (amount >= 10000.0) ? 0.05 : 0.0;

        // 最大割引率を選択（三項演算子の活用）
        double maxDiscount = (seniorDiscount > vipDiscount) ? seniorDiscount : vipDiscount;
        maxDiscount = (maxDiscount > largePurchaseDiscount) ? maxDiscount : largePurchaseDiscount;

        return maxDiscount;
    }

    // 結果表示メソッド
    public static void displayShoppingResult(String name, int age, int rank,
                                           String product, double price, int qty,
                                           double subtotal, double discountRate,
                                           double discountAmount, double total) {
        System.out.println("=== お買い物計算システム ===");
        System.out.println("お客様: " + name + "さん（" + age + "歳、会員ランク: " + rank + "）");
        System.out.println("商品: " + product);
        System.out.println("単価: " + formatCurrency(price) + " × " + qty + "個");
        System.out.println();

        System.out.println("小計: " + formatCurrency(subtotal));
        System.out.println("適用割引: " + getDiscountName(age, rank, subtotal) + 
                          "（" + (int)(discountRate * 100) + "%）");
        System.out.println("割引額: " + formatCurrency(discountAmount));
        System.out.println("合計金額: " + formatCurrency(total));
        System.out.println();

        // 割引判定詳細
        displayDiscountDetails(age, rank, subtotal, discountRate);
    }

    // 通貨フォーマット（カンマ区切り）
    public static String formatCurrency(double amount) {
        return String.format("%,.0f円", amount);
    }

    // 適用された割引名を取得
    public static String getDiscountName(int age, int rank, double amount) {
        if (age >= 65) return "シニア割引";
        if (rank >= 4) return "VIP割引";
        if (amount >= 10000.0) return "大口割引";
        return "割引なし";
    }

    // 割引判定詳細表示
    public static void displayDiscountDetails(int age, int rank, double amount, double appliedRate) {
        System.out.println("割引判定詳細:");

        boolean isSenior = (age >= 65);
        boolean isVIP = (rank >= 4);
        boolean isLargePurchase = (amount >= 10000.0);

        System.out.println("- シニア割引（65歳以上）: " + 
                          (isSenior ? "適用 ✓" : "対象外"));
        System.out.println("- VIP割引（ランク4以上）: " + 
                          (isVIP ? "適用 ✓" : "対象外"));
        System.out.println("- 大口割引（1万円以上）: " + 
                          (isLargePurchase ? "適用 ✓" : "対象外"));
        System.out.println("- 最大割引適用: " + getDiscountName(age, rank, amount) + 
                          " " + (int)(appliedRate * 100) + "%");
    }
}

    エラー1: 型変換での精度損失

    // ❌ 問題のあるコード
public class PrecisionLossExample {
    public static void main(String[] args) {
        double price = 1234.56;
        int priceInt = (int)price;  // 小数点以下切り捨て
        System.out.println("元の価格: " + price);
        System.out.println("変換後: " + priceInt);  // 1234（.56が失われる）
    }
}

    // ✅ 改善されたコード
public class PrecisionSafeExample {
    public static void main(String[] args) {
        double price = 1234.56;

        // 四捨五入での変換
        int priceRounded = (int)Math.round(price);

        // 小数点以下の情報も保持
        int priceYen = (int)price;
        int priceSen = (int)((price - priceYen) * 100);

        System.out.println("元の価格: " + price + "円");
        System.out.println("四捨五入: " + priceRounded + "円");
        System.out.println("詳細: " + priceYen + "円" + priceSen + "銭");
    }
}

    エラー2: null安全性の問題

    // ❌ 問題のあるコード
public class NullUnsafeExample {
    public static void main(String[] args) {
        String customerName = null;  // nullの可能性

        // NullPointerException の危険性

    // ✅ 改善されたコード
public class NullSafeExample {
    public static void main(String[] args) {
        String customerName = null;

        // null安全な比較
        if (customerName != null && customerName.equals("田中")) {
            System.out.println("田中さんです");
        }

        // より安全な方法
        if ("田中".equals(customerName)) {  // 定数を先に書く
            System.out.println("田中さんです");
        }

        // 三項演算子での安全な処理
        String displayName = (customerName != null) ? customerName : "お客様";
        System.out.println("こんにちは、" + displayName);
    }
}

    エラー3: 演算子優先順位の誤解

    // ❌ 意図しない動作
public class OperatorPriorityError {
    public static void main(String[] args) {
        boolean isAdult = true;
        boolean hasLicense = false;
        boolean hasExperience = true;

        // 意図：「成人で、かつ（免許があるか経験がある）」
        // 実際：「（成人で免許がある）か経験がある」
        boolean canDrive = isAdult && hasLicense || hasExperience;  // 危険！

        System.out.println("運転可能: " + canDrive);  // true（意図と異なる）
    }
}

    // ✅ 正しい書き方
public class OperatorPriorityCorrect {
    public static void main(String[] args) {
        boolean isAdult = true;
        boolean hasLicense = false;
        boolean hasExperience = true;

        // 括弧で明確に意図を示す
        boolean canDrive = isAdult && (hasLicense || hasExperience);

        System.out.println("運転可能: " + canDrive);  // true（正しい結果）

        // より読みやすく分割する方法
        boolean hasQualification = hasLicense || hasExperience;
        boolean canDriveAlternative = isAdult && hasQualification;

        System.out.println("運転可能（別方法）: " + canDriveAlternative);
    }
}

    💡 よくある質問

    Q1: どのデータ型を選べばいいか迷います

    A: 以下の基準で判断しましょう： - int: 一般的な整数（年齢、個数、点数など） - long: intを超える可能性がある大きな数（ID、金額など） - double: 小数点が必要な数値（価格、割合、測定値など） - String: 文字として扱うもの（名前、住所など）

    迷ったときは「将来の拡張性」も考慮しましょう。

    Q2: 演算子を組み合わせると読みにくくなります

    A: 複雑になったら積極的に分割しましょう：

    // 読みにくい
boolean result = (age >= 18) && (score >= 60) && !isAbsent && (rank >= 3 || hasSpecialPermission);

// 読みやすい
boolean isAdult = (age >= 18);
boolean passesTest = (score >= 60);
boolean attendsClass = !isAbsent;
boolean hasPermission = (rank >= 3) || hasSpecialPermission;
boolean result = isAdult && passesTest && attendsClass && hasPermission;

    Q3: 型変換エラーが頻繁に発生します

    A: try-catch文を使用して安全に処理しましょう：

    String input = "abc123";  // 数値でない文字列
try {
    int number = Integer.parseInt(input);
    System.out.println("数値: " + number);
} catch (NumberFormatException e) {
    System.out.println("数値変換に失敗しました: " + input);
}

    Q4: nullエラーを防ぐ良い方法は？

    A: 常にnullチェックを習慣化しましょう：

    // 基本パターン
if (text != null && !text.equals("")) {
    // 安全な処理
}

// メソッド化して再利用
public static boolean isNotEmpty(String str) {
    return str != null && !str.equals("");
}
