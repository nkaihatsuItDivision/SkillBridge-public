    1. なぜエラーハンドリングが必要なのか？

    現実のプログラムで起こる問題

    前回学んだ基本的な入力処理には、まだ解決されていない問題があります：

    public class BasicInputProblem {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("年齢を入力してください: ");
        int age = scanner.nextInt();  // ここで問題が発生する可能性

        System.out.println("あなたは" + age + "歳ですね");
        scanner.close();
    }
}

    このプログラムで「abc」と入力すると... 

    Exception in thread "main" java.util.InputMismatchException

    プログラムがクラッシュしてしまいます！

    エラーハンドリングの効果

    同じプログラムに例外処理を追加すると：

    public class SafeInputExample {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("年齢を入力してください: ");
        try {
            int age = scanner.nextInt();
            System.out.println("あなたは" + age + "歳ですね");
        } catch (InputMismatchException e) {
            System.out.println("エラー: 数値を入力してください");
        }

        scanner.close();
    }
}

    「abc」と入力した場合： 

    年齢を入力してください: abc
エラー: 数値を入力してください

    プログラムがクラッシュせずに、適切なエラーメッセージを表示できました！

    2. try-catch文の基本構文

    try-catch文とは？

    try-catch文は、プログラムでエラー（例外）が発生した時に、プログラムをクラッシュさせずに適切に対処する仕組みです。

    基本的な書き方

    try {
    // エラーが発生する可能性のあるコード
} catch (例外の種類 変数名) {
    // エラーが発生した時の処理
}

    実際の例

    public class TryCatchBasic {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("数値を入力してください: ");

        try {
            int number = scanner.nextInt();
            System.out.println("入力された数値: " + number);
        } catch (InputMismatchException e) {
            System.out.println("エラー: 数値以外が入力されました");
        }

        scanner.close();
    }
}

    try-catch文の流れ

    📘 try-catchのフローチャート

    1. tryブロック内のコードを実行
    2. エラーが発生しなかった場合: catchブロックをスキップして次の処理へ
    3. エラーが発生した場合: 該当するcatchブロックの処理を実行

    3. InputMismatchException の対応

    InputMismatchException とは？

    InputMismatchExceptionは、Scannerで数値を期待している時に、数値以外のデータが入力された場合に発生する例外です。

    具体的な対応方法

    public class InputMismatchHandling {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("点数を入力してください (0-100): ");

        try {
            int score = scanner.nextInt();

            // 正常に入力された場合の処理
            if (score >= 0 && score <= 100) {
                System.out.println("入力された点数: " + score);

                if (score >= 80) {
                    System.out.println("評価: 優秀");
                } else if (score >= 60) {
                    System.out.println("評価: 合格");
                } else {
                    System.out.println("評価: 不合格");
                }
            } else {
                System.out.println("エラー: 0から100の範囲で入力してください");
            }

        } catch (InputMismatchException e) {
            System.out.println("エラー: 数値を入力してください");
            scanner.nextLine(); // 不正な入力をクリア
        }

        scanner.close();
    }
}

    重要なポイント

    scanner.nextLine()でバッファクリア

    catch (InputMismatchException e) {
    System.out.println("エラー: 数値を入力してください");
    scanner.nextLine(); // ← これが重要！
}

    例外が発生した場合、Scannerのバッファに不正な入力が残ってしまいます。scanner.nextLine()で不正な入力をクリアすることで、次の入力処理に影響しないようにします。

    4. NumberFormatException の対応

    NumberFormatException とは？

    NumberFormatExceptionは、文字列を数値に変換しようとした時に、変換できない形式の文字列が渡された場合に発生する例外です。

    Integer.parseInt() での例外処理

    public class NumberFormatHandling {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("年齢を入力してください: ");
        String input = scanner.nextLine();

        try {
            int age = Integer.parseInt(input);

            if (age >= 0 && age <= 150) {
                System.out.println("年齢: " + age + "歳");
            } else {
                System.out.println("エラー: 現実的な年齢を入力してください (0-150)");
            }

        } catch (NumberFormatException e) {
            System.out.println("エラー: 数値を入力してください");
            System.out.println("入力された値: \"" + input + "\"");
        }

        scanner.close();
    }
}

    Double.parseDouble() での例外処理

    public class DoubleParseHandling {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("身長を入力してください (cm): ");
        String input = scanner.nextLine();

        try {
            double height = Double.parseDouble(input);

            if (height > 0 && height < 300) {
                System.out.printf("身長: %.1f cm%n", height);

                // BMI計算のための体重入力
                System.out.print("体重を入力してください (kg): ");
                String weightInput = scanner.nextLine();

                try {
                    double weight = Double.parseDouble(weightInput);
                    if (weight > 0 && weight < 500) {
                        double bmi = weight / ((height / 100) * (height / 100));
                        System.out.printf("BMI: %.1f%n", bmi);
                    } else {
                        System.out.println("エラー: 現実的な体重を入力してください");
                    }
                } catch (NumberFormatException e) {
                    System.out.println("エラー: 体重は数値で入力してください");
                }

            } else {
                System.out.println("エラー: 現実的な身長を入力してください");
            }

        } catch (NumberFormatException e) {
            System.out.println("エラー: 身長は数値で入力してください");
        }

        scanner.close();
    }
}

    5. do-while文を使った再試行ループ

    正しい入力まで繰り返す仕組み

    実用的なプログラムでは、エラーが発生しても再度入力を求めることが一般的です。

    do-while文の基本構文

    do {
    // 繰り返したい処理
} while (条件);

    実践例：堅牢な年齢入力システム

    public class RobustAgeInput {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        int age = -1;  // 無効な初期値
        boolean validInput = false;

        do {
            System.out.print("年齢を入力してください (0-120): ");

            try {
                age = scanner.nextInt();

                // 年齢の範囲チェック
                if (age >= 0 && age <= 120) {
                    validInput = true;  // 正常な入力
                    System.out.println("✓ 年齢: " + age + "歳が入力されました");
                } else {
                    System.out.println("❌ エラー: 0から120の範囲で入力してください");
                }

            } catch (InputMismatchException e) {
                System.out.println("❌ エラー: 数値を入力してください");
                scanner.nextLine(); // バッファクリア
            }

            // エラーの場合は空行を追加して見やすくする
            if (!validInput) {
                System.out.println();
            }

        } while (!validInput);

        // 年齢に応じたメッセージ
        if (age < 18) {
            System.out.println("未成年の方ですね");
        } else if (age < 65) {
            System.out.println("成人の方ですね");
        } else {
            System.out.println("シニアの方ですね");
        }

        scanner.close();
    }
}

    より洗練された再試行システム

    public class AdvancedRetrySystem {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        int score = getValidScore(scanner);
        displayGrade(score);

        scanner.close();
    }

    // 有効な点数を取得するメソッド
    public static int getValidScore(Scanner scanner) {
        int score = -1;
        boolean validInput = false;
        int attemptCount = 0;

        do {
            attemptCount++;

            if (attemptCount > 1) {
                System.out.printf("再入力 (%d回目): ", attemptCount);
            } else {
                System.out.print("テストの点数を入力してください (0-100): ");
            }

            try {
                score = scanner.nextInt();

                if (score >= 0 && score <= 100) {
                    validInput = true;
                    System.out.println("✓ 点数: " + score + "点が入力されました");
                } else {
                    System.out.println("❌ エラー: 0から100の範囲で入力してください");
                }

            } catch (InputMismatchException e) {
                System.out.println("❌ エラー: 数値を入力してください");
                scanner.nextLine(); // バッファクリア
            }

            if (!validInput && attemptCount >= 3) {
                System.out.println("💡 ヒント: 0から100までの整数を入力してください");
            }

            if (!validInput) {
                System.out.println();
            }

        } while (!validInput);

        return score;
    }

    // 点数に応じた評価を表示するメソッド
    public static void displayGrade(int score) {
        System.out.println();
        System.out.println("=== 評価結果 ===");
        System.out.printf("点数: %d点%n", score);

        if (score >= 90) {
            System.out.println("評価: S (優秀)");
        } else if (score >= 80) {
            System.out.println("評価: A (良好)");
        } else if (score >= 70) {
            System.out.println("評価: B (普通)");
        } else if (score >= 60) {
            System.out.println("評価: C (合格)");
        } else {
            System.out.println("評価: D (不合格)");
        }
    }
}

    6. カスタムバリデーション（独自の検証ルール）

    業務ロジックレベルの検証

    例外処理は「型のエラー」だけでなく、業務ルールに基づいた検証にも活用できます。

    実践例：商品注文システム

    public class OrderValidationSystem {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.println("=== 商品注文システム ===");

        // 商品価格の入力
        double price = getValidPrice(scanner);

        // 注文数量の入力
        int quantity = getValidQuantity(scanner);

        // 年齢の入力（割引判定用）
        int age = getValidAge(scanner);

        // 注文処理
        processOrder(price, quantity, age);

        scanner.close();
    }

    // 有効な価格を取得
    public static double getValidPrice(Scanner scanner) {
        double price = -1;
        boolean validInput = false;

        do {
            System.out.print("商品価格を入力してください（円）: ");

            try {
                price = scanner.nextDouble();

                if (price >= 0) {
                    if (price > 1000000) {
                        System.out.println("❌ 警告: 100万円を超える商品は取り扱いできません");
                    } else {
                        validInput = true;
                        System.out.printf("✓ 価格: %.0f円%n", price);
                    }
                } else {
                    System.out.println("❌ エラー: 価格は0円以上で入力してください");
                }

            } catch (InputMismatchException e) {
                System.out.println("❌ エラー: 数値を入力してください");
                scanner.nextLine(); // バッファクリア
            }

            if (!validInput) {
                System.out.println();
            }

        } while (!validInput);

        return price;
    }

    // 有効な数量を取得
    public static int getValidQuantity(Scanner scanner) {
        int quantity = -1;
        boolean validInput = false;

        do {
            System.out.print("注文数量を入力してください（個）: ");

            try {
                quantity = scanner.nextInt();

                if (quantity > 0) {
                    if (quantity > 100) {
                        System.out.println("❌ 警告: 一度に注文できる数量は100個までです");
                    } else {
                        validInput = true;
                        System.out.println("✓ 数量: " + quantity + "個");
                    }
                } else {
                    System.out.println("❌ エラー: 数量は1個以上で入力してください");
                }

            } catch (InputMismatchException e) {
                System.out.println("❌ エラー: 整数を入力してください");
                scanner.nextLine(); // バッファクリア
            }

            if (!validInput) {
                System.out.println();
            }

        } while (!validInput);

        return quantity;
    }

    // 有効な年齢を取得
    public static int getValidAge(Scanner scanner) {
        int age = -1;
        boolean validInput = false;

        do {
            System.out.print("年齢を入力してください（割引判定用）: ");

            try {
                age = scanner.nextInt();

                if (age >= 0 && age <= 120) {
                    validInput = true;
                    System.out.println("✓ 年齢: " + age + "歳");
                } else {
                    System.out.println("❌ エラー: 現実的な年齢を入力してください (0-120)");
                }

            } catch (InputMismatchException e) {
                System.out.println("❌ エラー: 数値を入力してください");
                scanner.nextLine(); // バッファクリア
            }

            if (!validInput) {
                System.out.println();
            }

        } while (!validInput);

    }

    public static void processOrder(double price, int quantity, int age) {
        System.out.println();
        System.out.println("=== 注文確認 ===");

        double subtotal = price * quantity;
        double discount = 0;

        // 年齢による割引
        if (age >= 65) {
            discount = subtotal * 0.1; // シニア割引10%
            System.out.println("シニア割引(10%)が適用されます");
        } else if (age < 18) {
            discount = subtotal * 0.05; // 学生割引5%
            System.out.println("学生割引(5%)が適用されます");
        }

        double total = subtotal - discount;

        System.out.printf("商品価格: %.0f円%n", price);
        System.out.printf("数量: %d個%n", quantity);
        System.out.printf("小計: %.0f円%n", subtotal);

        if (discount > 0) {
            System.out.printf("割引: -%.0f円%n", discount);
        }

        System.out.printf("合計: %.0f円%n", total);
        System.out.println();

    }

    import java.util.InputMismatchException;
import java.util.Scanner;

public class RobustCalculator {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.println("=== 堅牢な計算機 ===");

        // 最初の数値を取得

        // 演算子を取得

    // 有効な数値を取得するメソッド
    public static double getValidNumber(Scanner scanner, String prompt) {
        double number = 0;
        boolean validInput = false;

        do {
            System.out.print(prompt);

            try {
                number = scanner.nextDouble();
                validInput = true;
                System.out.println("✓ 数値: " + number);
            } catch (InputMismatchException e) {
                System.out.println("❌ エラー: 数値を入力してください");
                scanner.nextLine(); // バッファクリア
                System.out.println();
            }

        } while (!validInput);

        return number;
    }

    // 有効な演算子を取得するメソッド
    public static char getValidOperator(Scanner scanner) {
        char operator = ' ';
        boolean validInput = false;

        do {
            System.out.print("演算子を入力してください (+, -, *, /): ");
            String input = scanner.next();

            if (input.length() == 1) {
                char inputChar = input.charAt(0);

                if (inputChar == '+' || inputChar == '-' || 
                    inputChar == '*' || inputChar == '/') {
                    operator = inputChar;
                    validInput = true;
                    System.out.println("✓ 演算子: " + operator);
                } else {
                    System.out.println("❌ エラー: +, -, *, / のいずれかを入力してください");
                    System.out.println();
                }
            } else {
                System.out.println("❌ エラー: 1文字の演算子を入力してください");
                System.out.println();
            }

        } while (!validInput);

        return operator;
    }

    // 2番目の数値を取得（ゼロ除算対応）
    public static double getValidSecondNumber(Scanner scanner, char operator) {
        double number = 0;
        boolean validInput = false;

        do {
            System.out.print("2番目の数値を入力してください: ");

            try {
                number = scanner.nextDouble();

                // ゼロ除算チェック
                if (operator == '/' && number == 0) {
                    System.out.println("❌ エラー: ゼロで割ることはできません");
                    System.out.println();
                } else {
                    validInput = true;
                    System.out.println("✓ 数値: " + number);
                }

            } catch (InputMismatchException e) {
                System.out.println("❌ エラー: 数値を入力してください");
                scanner.nextLine(); // バッファクリア
                System.out.println();
            }

        } while (!validInput);

        return number;
    }

    // 計算を実行して結果を表示
    public static void performCalculation(double num1, double num2, char operator) {
        System.out.println();
        System.out.println("=== 計算結果 ===");

        double result = 0;
        boolean validCalculation = true;

        switch (operator) {
            case '+':
                result = num1 + num2;
                break;
            case '-':
                result = num1 - num2;
                break;
            case '*':
                result = num1 * num2;
                break;
            case '/':
                result = num1 / num2;
                break;
            default:
                validCalculation = false;
                System.out.println("予期しないエラーが発生しました");
        }

        if (validCalculation) {
            System.out.printf("%.1f %c %.1f = %.6f%n", num1, operator, num2, result);
        }
    }
}

    import java.util.InputMismatchException;
import java.util.Scanner;

public class StudentGradeManager {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.println("=== 学生成績管理システム ===");

        // 学生名を取得
        String name = getValidStudentName(scanner);

        // 3科目の点数を取得
        int[] scores = new int[3];
        String[] subjects = {"国語", "数学", "英語"};

        for (int i = 0; i < subjects.length; i++) {
            scores[i] = getValidScore(scanner, subjects[i]);
        }

        // 結果表示
        displayResults(name, scores);

        scanner.close();
    }

    // 有効な学生名を取得するメソッド
    public static String getValidStudentName(Scanner scanner) {
        String name = "";
        boolean validInput = false;

        do {
            System.out.print("学生名を入力してください: ");
            name = scanner.nextLine().trim();

            if (name.isEmpty()) {
                System.out.println("❌ エラー: 名前を空にすることはできません");
                System.out.println();
            } else {
                validInput = true;
                System.out.println("✓ 学生名: " + name);
            }
        } while (!validInput);

        return name;
    }

    // 有効な点数を取得するメソッド
    public static int getValidScore(Scanner scanner, String subject) {
        int score = -1;
        boolean validInput = false;

        do {
            System.out.print(subject + "の点数を入力してください (0-100): ");
            try {
                score = scanner.nextInt();

                if (score < 0 || score > 100) {
                    System.out.println("❌ エラー: 0から100の範囲で入力してください");
                    System.out.println();
                } else {
                    validInput = true;
                    System.out.println("✓ " + subject + ": " + score);
                }
            } catch (InputMismatchException e) {
                System.out.println("❌ エラー: 数値を入力してください");
                scanner.nextLine(); // バッファクリア
                System.out.println();
            }
        } while (!validInput);

        return score;
    }

    // 結果を表示するメソッド
    public static void displayResults(String name, int[] scores) {
        int total = 0;
        for (int score : scores) {
            total += score;
        }
        double average = total / 3.0;

        String grade;
        if (average >= 80) {
            grade = "優秀";
        } else if (average >= 60) {
            grade = "合格";
        } else {

        System.out.println("学生名: " + name);
        System.out.printf("平均点: %.2f%n", average);
        System.out.println("評価: " + grade);
    }
}

    💡 よくある質問

    Q1: try-catch文とif文による事前チェックの違いは？

    A: - if文（事前チェック）: hasNextInt()のように、エラーが起きる前に確認 - try-catch文（例外処理）: エラーが起きてから対処

    // 事前チェック方式
if (scanner.hasNextInt()) {
    int number = scanner.nextInt();
} else {
    System.out.println("数値を入力してください");
}

// 例外処理方式
try {
    int number = scanner.nextInt();
} catch (InputMismatchException e) {
    System.out.println("数値を入力してください");
}

    どちらも有効ですが、例外処理の方がより汎用的で実用的です。

    Q2: catch文でscanner.nextLine()が必要な理由は？

    A: InputMismatchExceptionが発生すると、不正な入力がScannerのバッファに残ったままになります。これをクリアしないと、次の入力処理でも同じエラーが繰り返されます。

    // バッファクリアなし（問題のあるコード）
catch (InputMismatchException e) {
    System.out.println("エラー");
    // 不正な入力が残ったまま
}

// 正しいコード
catch (InputMismatchException e) {
    System.out.println("エラー");
    scanner.nextLine(); // 不正な入力をクリア
}

    Q3: なぜInteger.parseInt()とscanner.nextInt()を使い分けるのですか？

    A: - scanner.nextInt(): 直接入力から整数を読み取る - Integer.parseInt(): 文字列を整数に変換

    // scanner.nextInt()の場合
try {
    int age = scanner.nextInt();
} catch (InputMismatchException e) {
    // 処理
}

// Integer.parseInt()の場合
try {
    String input = scanner.nextLine();
    int age = Integer.parseInt(input);
} catch (NumberFormatException e) {
    // 処理
}

    Integer.parseInt()の方が文字列操作と組み合わせやすく、より細かい制御が可能です。

    Q4: do-while文とwhile文の違いは？

    A: - while文: 条件をチェックしてから実行（0回の場合あり） - do-while文: 実行してから条件をチェック（必ず1回は実行）

    // while文
while (条件) {
    // 条件がfalseなら1回も実行されない
}

// do-while文
do {
    // 必ず1回は実行される
} while (条件);

    入力処理では必ず1回は入力を求める必要があるため、do-while文が適しています。

    Q5: 例外処理はどこまで使うべきですか？

    A: 予期しないエラーや外部要因によるエラーに対して使用します。

    // 適切な使用例
try {
    int number = Integer.parseInt(userInput); // ユーザー入力は予測不可能
    File file = new File(filename); // ファイルが存在しない可能性
} catch (NumberFormatException | FileNotFoundException e) {
    // エラー処理
}

// 不適切な使用例
try {
    int result = 10 / 0; // 明らかにエラーになる処理
} catch (ArithmeticException e) {
    // このようなコードは書かない
}
