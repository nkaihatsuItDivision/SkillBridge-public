    1. 成績管理システムの要件定義

    💼 システム概要

    目的: クラスの学生の成績を管理し、統計情報を提供するシステム

    主要機能: 1. 学生管理: 学生の登録・表示・検索 2. 成績管理: 3科目（数学・英語・国語）の成績入力・編集 3. 統計機能: 平均点、合格率、成績分布の表示 4. 検索機能: 学生名・学籍番号での検索

    📊 管理するデータ

    学生情報: - 学籍番号（例：S001） - 学生名（例：山田太郎） - 数学の点数（0-100点） - 英語の点数（0-100点） - 国語の点数（0-100点）

    統計情報: - 各科目の平均点 - 総合平均点 - 合格者数・合格率（3科目すべて60点以上） - 各科目の最高点・最低点

    🎯 使用する技術（Week 2の総復習）

    • 配列: 学生データの管理

    • 条件分岐: 成績判定・メニュー処理
    • 繰り返し: データ処理・統計計算
    • Scanner: ユーザー入力
    • printf: 美しい表形式出力
    • try-catch: エラーハンドリング

    🚀 ステップ1: プロジェクトの基本構造

    新しいクラス ScoreManager を作成しましょう。

    import java.util.Scanner;

public class ScoreManager {
    // クラス変数（全メソッドで共有）
    private static String[][] students = new String[20][5];
    private static int studentCount = 0;
    private static Scanner scanner = new Scanner(System.in);

    public static void main(String[] args) {
        System.out.println("成績管理システムを開始します...");

        while (true) {
            showMenu();
            int choice = getChoice();

            if (choice == 7) {
                System.out.println("システムを終了します。");
                break;
            }

            processChoice(choice);
        }

        scanner.close();
    }

    // メニュー表示
    private static void showMenu() {
        System.out.println("\n========================================");
        System.out.println("      学生成績管理システム v1.0");
        System.out.println("========================================");
        System.out.println("1. 学生を登録する");
        System.out.println("2. 全学生を表示する");
        System.out.println("3. 成績を編集する");
        System.out.println("4. 学生を検索する");
        System.out.println("5. 統計情報を表示する");
        System.out.println("6. 成績分布を表示する");
        System.out.println("7. 終了");
        System.out.println("========================================");
        System.out.print("選択してください (1-7): ");
    }

    // メニュー選択の取得
    private static int getChoice() {
        try {
            return scanner.nextInt();
        } catch (Exception e) {
            scanner.nextLine(); // 不正入力をクリア
            return -1; // 無効な選択として返す
        }
    }

    // 選択に応じた処理
    private static void processChoice(int choice) {
        switch (choice) {
            case 1:
                addStudent();
                break;
            case 2:
                showAllStudents();
                break;
            case 3:
                editScore();
                break;
            case 4:
                searchStudent();
                break;
            case 5:
                showStatistics();
                break;
            case 6:
                showDistribution();
                break;
            default:
                System.out.println("無効な選択です。1-7の数字を入力してください。");
        }
    }
}

    🚀 ステップ2: 学生登録機能の実装

    学生を登録する addStudent() メソッドを実装しましょう：

    // 学生を登録する
private static void addStudent() {
    System.out.println("\n--- 学生登録 ---");

    if (studentCount >= 20) {
        System.out.println("これ以上学生を登録できません（最大20人）");
        return;
    }

    scanner.nextLine(); // 改行文字をクリア

    // 学籍番号の入力
    System.out.print("学籍番号を入力してください (例: S001): ");
    String studentId = scanner.nextLine();

    // 重複チェック
    if (isDuplicateId(studentId)) {
        System.out.println("この学籍番号は既に登録されています。");
        return;
    }

    // 学生名の入力
    System.out.print("学生名を入力してください: ");
    String studentName = scanner.nextLine();

    // 成績の入力
    int mathScore = getValidScore("数学");
    int englishScore = getValidScore("英語");
    int japaneseScore = getValidScore("国語");

    // データの保存
    students[studentCount][0] = studentId;
    students[studentCount][1] = studentName;
    students[studentCount][2] = String.valueOf(mathScore);
    students[studentCount][3] = String.valueOf(englishScore);
    students[studentCount][4] = String.valueOf(japaneseScore);

    studentCount++;

    System.out.println("学生「" + studentName + "」を登録しました。");
}

// 学籍番号の重複チェック
private static boolean isDuplicateId(String studentId) {
    for (int i = 0; i < studentCount; i++) {
        if (students[i][0].equals(studentId)) {
            return true;
        }
    }
    return false;
}

// 有効な点数の入力を取得
private static int getValidScore(String subject) {
    while (true) {
        try {
            System.out.print(subject + "の点数を入力してください (0-100): ");
            int score = scanner.nextInt();

            if (score >= 0 && score <= 100) {
                return score;
            } else {
                System.out.println("0から100の間で入力してください。");
            }
        } catch (Exception e) {
            System.out.println("数字を入力してください。");
            scanner.nextLine(); // 不正入力をクリア
        }
    }
}

    ✅ 確認: 学生を1-2人登録してみて、正常に動作することを確認してください。

    🚀 ステップ3: 学生一覧表示機能の実装

    登録した学生を表示する showAllStudents() メソッドを実装しましょう：

    // 全学生を表示する
private static void showAllStudents() {
    System.out.println("\n--- 全学生一覧 ---");

    if (studentCount == 0) {
        System.out.println("登録されている学生はいません。");
        return;
    }

    // ヘッダー表示
    System.out.println("========================================================================");
    System.out.printf("%-8s %-12s %6s %6s %6s %8s %6s\n", 
                      "学籍番号", "学生名", "数学", "英語", "国語", "平均点", "判定");
    System.out.println("========================================================================");

    // 学生データ表示
    for (int i = 0; i < studentCount; i++) {
        String studentId = students[i][0];
        String studentName = students[i][1];
        int mathScore = Integer.parseInt(students[i][2]);
        int englishScore = Integer.parseInt(students[i][3]);
        int japaneseScore = Integer.parseInt(students[i][4]);

        double average = (mathScore + englishScore + japaneseScore) / 3.0;
        String result = (mathScore >= 60 && englishScore >= 60 && japaneseScore >= 60) ? "合格" : "不合格";

        System.out.printf("%-8s %-12s %6d %6d %6d %8.1f %6s\n",
                          studentId, studentName, mathScore, englishScore, japaneseScore, average, result);
    }

    System.out.println("========================================================================");
    System.out.println("登録学生数: " + studentCount + " 人");
}

    🚀 ステップ4: 成績編集機能の実装

    学生の成績を編集する editScore() メソッドを実装しましょう：

    // 成績を編集する
private static void editScore() {
    System.out.println("\n--- 成績編集 ---");

    if (studentCount == 0) {
        System.out.println("登録されている学生はいません。");
        return;
    }

    scanner.nextLine(); // 改行文字をクリア

    System.out.print("編集する学生の学籍番号を入力してください: ");
    String targetId = scanner.nextLine();

    // 学生を検索
    int studentIndex = findStudentIndex(targetId);
    if (studentIndex == -1) {
        System.out.println("学籍番号「" + targetId + "」の学生は見つかりませんでした。");
        return;
    }

    // 現在の成績を表示
    String studentName = students[studentIndex][1];
    int currentMath = Integer.parseInt(students[studentIndex][2]);
    int currentEnglish = Integer.parseInt(students[studentIndex][3]);
    int currentJapanese = Integer.parseInt(students[studentIndex][4]);

    System.out.println("\n" + studentName + " さんの現在の成績:");
    System.out.println("数学: " + currentMath + "点");
    System.out.println("英語: " + currentEnglish + "点");
    System.out.println("国語: " + currentJapanese + "点");

    // 編集する科目の選択
    System.out.println("\n編集する科目を選択してください:");
    System.out.println("1. 数学");
    System.out.println("2. 英語");
    System.out.println("3. 国語");
    System.out.println("4. 全科目");
    System.out.print("選択 (1-4): ");

    int choice = getChoice();

    switch (choice) {
        case 1:
            editSingleScore(studentIndex, 2, "数学");
            break;
        case 2:
            editSingleScore(studentIndex, 3, "英語");
            break;
        case 3:
            editSingleScore(studentIndex, 4, "国語");
            break;
        case 4:
            editAllScores(studentIndex);
            break;
        default:
            System.out.println("無効な選択です。");
            return;
    }

    System.out.println(studentName + " さんの成績を更新しました。");
}

// 学生のインデックスを検索
private static int findStudentIndex(String studentId) {
    for (int i = 0; i < studentCount; i++) {
        if (students[i][0].equals(studentId)) {
            return i;
        }
    }
    return -1; // 見つからない場合
}

// 単一科目の成績編集
private static void editSingleScore(int studentIndex, int subjectIndex, String subjectName) {
    int newScore = getValidScore(subjectName);
    students[studentIndex][subjectIndex] = String.valueOf(newScore);
}

// 全科目の成績編集
private static void editAllScores(int studentIndex) {
    int mathScore = getValidScore("数学");
    int englishScore = getValidScore("英語");
    int japaneseScore = getValidScore("国語");

    students[studentIndex][2] = String.valueOf(mathScore);
    students[studentIndex][3] = String.valueOf(englishScore);
    students[studentIndex][4] = String.valueOf(japaneseScore);
}

    クラス全体の統計情報を表示する showStatistics() メソッドを実装しましょう：

    // 統計情報を表示する
private static void showStatistics() {
    System.out.println("\n--- 統計情報 ---");

    if (studentCount == 0) {
        System.out.println("登録されている学生はいません。");
        return;
    }

    // 各科目の合計・平均・最高・最低を計算
    int[] mathScores = new int[studentCount];
    int[] englishScores = new int[studentCount];
    int[] japaneseScores = new int[studentCount];

    for (int i = 0; i < studentCount; i++) {
        mathScores[i] = Integer.parseInt(students[i][2]);
        englishScores[i] = Integer.parseInt(students[i][3]);
        japaneseScores[i] = Integer.parseInt(students[i][4]);
    }

    // 統計計算
    StatResult mathStats = calculateStats(mathScores);
    StatResult englishStats = calculateStats(englishScores);
    StatResult japaneseStats = calculateStats(japaneseScores);

    // 合格者数の計算
    int passCount = 0;
    double totalAverage = 0;

    for (int i = 0; i < studentCount; i++) {
        int math = mathScores[i];
        int english = englishScores[i];
        int japanese = japaneseScores[i];

        if (math >= 60 && english >= 60 && japanese >= 60) {
            passCount++;
        }

        totalAverage += (math + english + japanese) / 3.0;
    }

    totalAverage /= studentCount;
    double passRate = (double) passCount / studentCount * 100;

    // 結果表示
    System.out.println("===============================================");
    System.out.println("              クラス統計情報");
    System.out.println("===============================================");
    System.out.printf("対象学生数: %d 人\n", studentCount);
    System.out.printf("総合平均: %.1f 点\n", totalAverage);
    System.out.printf("合格者数: %d 人 (%.1f%%)\n", passCount, passRate);
    System.out.println("===============================================");

    System.out.printf("%-8s %8s %8s %8s %8s\n", "科目", "平均点", "最高点", "最低点", "合格者");
    System.out.println("-----------------------------------------------");

    int mathPass = countPassingStudents(mathScores);
    int englishPass = countPassingStudents(englishScores);
    int japanesePass = countPassingStudents(japaneseScores);

    System.out.printf("%-8s %8.1f %8d %8d %8d\n", "数学", mathStats.average, mathStats.max, mathStats.min, mathPass);
    System.out.printf("%-8s %8.1f %8d %8d %8d\n", "英語", englishStats.average, englishStats.max, englishStats.min, englishPass);
    System.out.printf("%-8s %8.1f %8d %8d %8d\n", "国語", japaneseStats.average, japaneseStats.max, japaneseStats.min, japanesePass);

    System.out.println("===============================================");
}

// 統計計算用のクラス
private static class StatResult {
    double average;
    int max;
    int min;

    StatResult(double average, int max, int min) {
        this.average = average;
        this.max = max;
        this.min = min;
    }
}

// 統計値の計算
private static StatResult calculateStats(int[] scores) {
    int sum = 0;
    int max = scores[0];
    int min = scores[0];

    for (int score : scores) {
        sum += score;
        if (score > max) max = score;
        if (score < min) min = score;
    }

    double average = (double) sum / scores.length;
    return new StatResult(average, max, min);
}

// 合格者数をカウント
private static int countPassingStudents(int[] scores) {
    int count = 0;
    for (int score : scores) {
        if (score >= 60) {
            count++;
        }
    }
    return count;
}

    // 成績分布を表示する
private static void showDistribution() {
    System.out.println("\n--- 成績分布 ---");

    if (studentCount == 0) {
        System.out.println("登録されている学生はいません。");
        return;
    }

    // 点数帯の定義
    String[] ranges = {"90-100", "80-89", "70-79", "60-69", "50-59", "40-49", "30-39", "20-29", "10-19", "0-9"};
    int[][] distribution = new int[3][10]; // 3科目 × 10段階

    // 各学生の成績を分類
    for (int i = 0; i < studentCount; i++) {
        int mathScore = Integer.parseInt(students[i][2]);
        int englishScore = Integer.parseInt(students[i][3]);
        int japaneseScore = Integer.parseInt(students[i][4]);

        distribution[0][getScoreRange(mathScore)]++;
        distribution[1][getScoreRange(englishScore)]++;
        distribution[2][getScoreRange(japaneseScore)]++;
    }

    // 分布表示
    System.out.println("====================================================");
    System.out.println("                 成績分布表");
    System.out.println("====================================================");
    System.out.printf("%-10s %8s %8s %8s\n", "点数帯", "数学", "英語", "国語");
    System.out.println("----------------------------------------------------");

    for (int i = 0; i < 10; i++) {
        System.out.printf("%-10s %8d %8d %8d\n", 
                          ranges[i], distribution[0][i], distribution[1][i], distribution[2][i]);
    }

    System.out.println("====================================================");

    // 視覚的な分布グラフ（簡易版）
    System.out.println("\n数学の成績分布グラフ:");
    showSimpleGraph(distribution[0], ranges);

    System.out.println("\n英語の成績分布グラフ:");
    showSimpleGraph(distribution[1], ranges);

    System.out.println("\n国語の成績分布グラフ:");
    showSimpleGraph(distribution[2], ranges);
}

// 点数を範囲インデックスに変換
private static int getScoreRange(int score) {
    if (score >= 90) return 0;      // 90-100
    else if (score >= 80) return 1; // 80-89
    else if (score >= 70) return 2; // 70-79
    else if (score >= 60) return 3; // 60-69
    else if (score >= 50) return 4; // 50-59
    else if (score >= 40) return 5; // 40-49
    else if (score >= 30) return 6; // 30-39
    else if (score >= 20) return 7; // 20-29
    else if (score >= 10) return 8; // 10-19
    else return 9;                  // 0-9
}

// 簡易グラフ表示
private static void showSimpleGraph(int[] data, String[] ranges) {
    for (int i = 0; i < data.length; i++) {
        if (data[i] > 0) {
            System.out.printf("%s: ", ranges[i]);
            for (int j = 0; j < data[i]; j++) {
                System.out.print("■");
            }
            System.out.printf(" (%d人)\n", data[i]);
        }
    }
}

    💡 よくある質問

    Q1: 2次元配列の students[i][j] の使い方がよくわかりません

    A: 2次元配列は「表」のようなイメージです：

    // students[行][列] = データ
students[0][0] = "S001";     // 1人目の学籍番号
students[0][1] = "田中太郎";  // 1人目の名前  
students[0][2] = "85";       // 1人目の数学
students[1][0] = "S002";     // 2人目の学籍番号
students[1][1] = "佐藤花子"; // 2人目の名前

    行（i）: 学生番号、列（j）: データの種類と覚えましょう。

    Q2: String.valueOf() と Integer.parseInt() の使い分けは？

    A: - String.valueOf(数値): 数値 → 文字列への変換 - Integer.parseInt(文字列): 文字列 → 数値への変換

    int score = 85;
String scoreStr = String.valueOf(score);  // "85"

String str = "90";
int num = Integer.parseInt(str);          // 90

    Q3: try-catch を使わない場合の問題は？

    A: 不正な入力でプログラムが終了してしまいます：

    // try-catch なし → 「abc」入力で異常終了
int score = scanner.nextInt();

// try-catch あり → エラーメッセージ表示後続行
try {
    int score = scanner.nextInt();
} catch (Exception e) {
    System.out.println("数字を入力してください。");
}
