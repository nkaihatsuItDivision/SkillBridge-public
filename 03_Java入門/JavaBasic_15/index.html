    1. なぜ対話型プログラムが重要なのか？

    単発処理の限界

    これまで作成したプログラムは、一度実行すると終了してしまいました：

    // 従来のプログラム
public static void main(String[] args) {
    // 何かの処理
    System.out.println(&quot;結果: &quot; + result);
    // プログラム終了
}

    問題点： - 別の処理をしたい場合、プログラムを再起動する必要がある - 複数の機能を使い分けることができない - 実用的なアプリケーションとは言えない

    対話型プログラムの価値

    // 対話型プログラム
public static void main(String[] args) {
    while (true) {
        // メニュー表示
        // ユーザーの選択に応じて処理
        // 継続するかユーザーに確認
    }
}

    メリット： - 一度起動すれば複数の処理を実行できる - ユーザーが必要な機能を選択できる - 実際のソフトウェアに近い使用感

    身近な対話型プログラムの例

    • ATM: メニューから「残高照会」「出金」「入金」を選択
    • コンビニレジ: 商品登録、会計、レシート印刷を繰り返し
    • カーナビ: 目的地設定、ルート検索、案内開始を選択

    2. while文による継続的な処理ループ

    while文の基本構文

    while (条件) {
    // 条件がtrueの間、繰り返し実行される処理
}

    for文との違い

    // for文（回数が決まっている場合）
for (int i = 1; i &lt;= 5; i++) {
    System.out.println(i + &quot;回目&quot;);
}

// while文（条件によって継続する場合）
boolean continueProgram = true;
while (continueProgram) {
    // 何かの処理
    // continueProgram = false; でループを終了
}

    実用的なwhile文の例

    public class SimpleLoop {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        String answer;

        do {
            System.out.println(&quot;簡単な計算をします&quot;);
            System.out.print(&quot;1つ目の数値: &quot;);
            int num1 = scanner.nextInt();
            System.out.print(&quot;2つ目の数値: &quot;);
            int num2 = scanner.nextInt();

            int sum = num1 + num2;
            System.out.println(&quot;合計: &quot; + sum);

            System.out.print(&quot;もう一度計算しますか？ (y/n): &quot;);
            scanner.nextLine(); // 改行文字を消費
            answer = scanner.nextLine();

        } while (answer.equalsIgnoreCase(&quot;y&quot;));

        System.out.println(&quot;プログラムを終了します&quot;);
        scanner.close();
    }
}

    3. switch-case文によるメニューシステム

    switch-case文の基本構文

    switch (変数) {
    case 値1:
        // 値1の場合の処理
        break;
    case 値2:
        // 値2の場合の処理
        break;
    default:
        // どのcaseにも該当しない場合の処理
        break;
}

    if-else文との比較

    // if-else文での選択処理
if (choice == 1) {
    System.out.println(&quot;機能1を実行&quot;);
} else if (choice == 2) {
    System.out.println(&quot;機能2を実行&quot;);
} else if (choice == 3) {
    System.out.println(&quot;機能3を実行&quot;);
} else {
    System.out.println(&quot;無効な選択&quot;);
}

// switch-case文での選択処理（推奨）
switch (choice) {
    case 1:
        System.out.println(&quot;機能1を実行&quot;);
        break;
    case 2:
        System.out.println(&quot;機能2を実行&quot;);
        break;
    case 3:
        System.out.println(&quot;機能3を実行&quot;);
        break;
    default:
        System.out.println(&quot;無効な選択&quot;);
        break;
}

    switch-case文の利点： - 複数の選択肢がある場合に読みやすい - 実行速度が速い - メニューシステムに最適

    メニューシステムの実装例

    public class MenuSystem {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        boolean running = true;

        while (running) {
            // メニュー表示
            System.out.println(&quot;\n=== 計算機アプリ ===&quot;);
            System.out.println(&quot;1. 足し算&quot;);
            System.out.println(&quot;2. 引き算&quot;);
            System.out.println(&quot;3. 掛け算&quot;);
            System.out.println(&quot;4. 割り算&quot;);
            System.out.println(&quot;0. 終了&quot;);
            System.out.print(&quot;選択してください: &quot;);

            int choice = scanner.nextInt();

            switch (choice) {
                case 1:
                    performAddition(scanner);
                    break;
                case 2:
                    performSubtraction(scanner);
                    break;
                case 3:
                    performMultiplication(scanner);
                    break;
                case 4:
                    performDivision(scanner);

                    break;

            }
        }

    }

    4. 配列の基本的な使用（Week 3への準備）

    なぜ配列が必要なのか？

    これまでは一つずつの値しか扱えませんでした：

    // 問題：複数の学生の点数を管理したい場合
int score1 = 85;
int score2 = 92;
int score3 = 78;
int score4 = 88;
int score5 = 95;

// 平均を計算
double average = (score1 + score2 + score3 + score4 + score5) / 5.0;

    問題点： - 学生数が増えると変数がどんどん増える - 処理が複雑になる - 柔軟性がない

    配列による解決

    // 解決：配列を使用
int[] scores = {85, 92, 78, 88, 95};

// 平均を計算
int sum = 0;
for (int i = 0; i &lt; scores.length; i++) {
    sum += scores[i];
}
double average = sum / (double) scores.length;

System.out.printf(&quot;平均点: %.1f点%n&quot;, average);

    配列の基本操作

    // 配列の宣言と初期化
int[] numbers = new int[5];  // 5つの要素を持つ配列
String[] names = {&quot;田中&quot;, &quot;佐藤&quot;, &quot;山田&quot;};  // 初期値付きで作成

// 要素へのアクセス
numbers[0] = 100;  // 最初の要素に100を代入
String firstName = names[0];  // 最初の要素を取得

// 配列の長さ
int length = numbers.length;  // 配列の要素数を取得

// 全要素の表示
for (int i = 0; i &lt; names.length; i++) {
    System.out.println((i + 1) + &quot;番目: &quot; + names[i]);
}

    今回のレッスンでの配列の使用範囲： - 基本的な宣言と初期化のみ - Week 3で詳細を学習する予定

                    break;

                    break;

                    break;

                    break;

    /**

        System.out.println(&quot;1. 学生一覧表示&quot;);
        System.out.println(&quot;2. 学生情報登録&quot;);
        System.out.println(&quot;3. 統計情報表示&quot;);
        System.out.println(&quot;4. 成績評価表示&quot;);
        System.out.println(&quot;0. 終了&quot;);
        System.out.println(&quot;=&quot;.repeat(30));
        System.out.print(&quot;選択してください: &quot;);
    }

    /**
     * 有効な選択肢を取得する（エラーハンドリング付き）
     */
    public static int getValidChoice(Scanner scanner) {
        while (true) {
            try {
                int choice = scanner.nextInt();
                scanner.nextLine(); // 改行文字を消費
                return choice;
            } catch (InputMismatchException e) {
                System.out.print(&quot;数値を入力してください: &quot;);
                scanner.nextLine(); // 不正入力をクリア
            }
        }
    }

    /**
     * 有効な点数を取得する（0-100の範囲チェック付き）
     */
    public static int getValidScore(Scanner scanner) {
        while (true) {
            try {
                System.out.print(&quot;点数を入力してください (0-100): &quot;);
                int score = scanner.nextInt();
                scanner.nextLine(); // 改行文字を消費

                if (score &lt; 0 || score &gt; 100) {
                    System.out.println(&quot;エラー: 0から100までの値を入力してください&quot;);
                    continue;
                }

                return score;
            } catch (InputMismatchException e) {
                System.out.println(&quot;エラー: 数値を入力してください&quot;);
                scanner.nextLine(); // 不正入力をクリア
            }
        }
    }

    /**
     * 学生一覧を表示する
     */
    public static void showStudentList(String[] names, int[] scores, int count) {
        System.out.println(&quot;\n=== 学生一覧 ===&quot;);

        if (count == 0) {
            System.out.println(&quot;登録されている学生はいません&quot;);
            return;
        }

        System.out.println(&quot;No. | 名前          | 点数 | 評価&quot;);
        System.out.println(&quot;----|---------------|------|------&quot;);

        for (int i = 0; i &lt; count; i++) {
            String grade = getGrade(scores[i]);
            System.out.printf(&quot;%3d | %-12s | %4d | %s%n&quot;, 
                            i + 1, names[i], scores[i], grade);
        }
    }

    /**
     * 学生情報を登録する
     */
    public static int addStudent(Scanner scanner, String[] names, int[] scores, int count) {
        System.out.println(&quot;\n=== 学生情報登録 ===&quot;);

        if (count &gt;= names.length) {
            System.out.println(&quot;エラー: これ以上学生を登録できません（最大&quot; + names.length + &quot;人）&quot;);
            return count;
        }

        System.out.print(&quot;学生名を入力してください: &quot;);
        String name = scanner.nextLine();

        if (name.trim().isEmpty()) {
            System.out.println(&quot;エラー: 学生名を入力してください&quot;);
            return count;
        }

        int score = getValidScore(scanner);

        names[count] = name;
        scores[count] = score;

        System.out.println(&quot;学生情報を登録しました&quot;);
        System.out.printf(&quot;登録内容: %s さん (%d点)%n&quot;, name, score);

        return count + 1;
    }

    /**
     * 統計情報を表示する
     */
    public static void showStatistics(String[] names, int[] scores, int count) {
        System.out.println(&quot;\n=== 統計情報 ===&quot;);

        if (count == 0) {
            System.out.println(&quot;統計を計算できません（学生が登録されていません）&quot;);
            return;
        }

        int sum = 0;
        int max = scores[0];
        int min = scores[0];
        String maxName = names[0];
        String minName = names[0];

        for (int i = 0; i &lt; count; i++) {
            sum += scores[i];

            if (scores[i] &gt; max) {
                max = scores[i];
                maxName = names[i];
            }

            if (scores[i] &lt; min) {
                min = scores[i];
                minName = names[i];
            }
        }

        double average = sum / (double) count;

        System.out.printf(&quot;総学生数: %d人%n&quot;, count);
        System.out.printf(&quot;平均点: %.1f点%n&quot;, average);
        System.out.printf(&quot;最高点: %d点 (%s さん)%n&quot;, max, maxName);
        System.out.printf(&quot;最低点: %d点 (%s さん)%n&quot;, min, minName);
        System.out.printf(&quot;合計点: %d点%n&quot;, sum);
    }

    /**
     * 成績評価を表示する
     */
    public static void showGradeEvaluation(String[] names, int[] scores, int count) {
        System.out.println(&quot;\n=== 成績評価 ===&quot;);

        if (count == 0) {
            System.out.println(&quot;評価する学生がいません&quot;);
            return;
        }

        int excellentCount = 0, goodCount = 0, passCount = 0, failCount = 0;

        System.out.println(&quot;名前          | 点数 | 評価 | 判定&quot;);
        System.out.println(&quot;--------------|------|------|--------&quot;);

        for (int i = 0; i &lt; count; i++) {
            String grade = getGrade(scores[i]);
            String status = getStatus(scores[i]);

            System.out.printf(&quot;%-12s | %4d | %4s | %s%n&quot;, 
                            names[i], scores[i], grade, status);

            if (scores[i] &gt;= 90) excellentCount++;
            else if (scores[i] &gt;= 75) goodCount++;
            else if (scores[i] &gt;= 60) passCount++;
            else failCount++;
        }

        System.out.println(&quot;\n評価別人数:&quot;);
        System.out.printf(&quot;優 (90点以上): %d人%n&quot;, excellentCount);
        System.out.printf(&quot;良 (75-89点): %d人%n&quot;, goodCount);
        System.out.printf(&quot;可 (60-74点): %d人%n&quot;, passCount);
        System.out.printf(&quot;不可 (60点未満): %d人%n&quot;, failCount);
    }

    /**
     * 点数から成績評価を取得する
     */
    public static String getGrade(int score) {
        if (score &gt;= 90) return &quot;優&quot;;
        else if (score &gt;= 75) return &quot;良&quot;;
        else if (score &gt;= 60) return &quot;可&quot;;
        else return &quot;不可&quot;;
    }

    /**
     * 点数から合否判定を取得する
     */
    public static String getStatus(int score) {
        return score &gt;= 60 ? &quot;合格&quot; : &quot;不合格&quot;;
    }

    💡 よくある質問

    Q1: while文とfor文の使い分けがわかりません

    A: 以下の基準で判断してください： - for文: 繰り返し回数が決まっている場合（1から10まで、配列の全要素など） - while文: 条件によって継続する場合（ユーザーが終了を選ぶまで、正しい入力まで）

    // for文：配列の全要素を処理（回数が決まっている）
for (int i = 0; i &lt; array.length; i++) {
    System.out.println(array[i]);
}

// while文：ユーザーが終了を選ぶまで（回数が不明）
boolean running = true;
while (running) {
    // メニュー処理
    if (choice == 0) running = false;
}

    Q2: switch-case文のbreakを忘れるとどうなりますか？

    A: フォールスルーという現象が起きて、次のcaseの処理も実行されてしまいます：

    switch (choice) {
    case 1:
        System.out.println(&quot;機能1&quot;);
        // breakがない！
    case 2:
        System.out.println(&quot;機能2&quot;);
        break;
}

// choice=1の場合の出力：
// 機能1
// 機能2  ← 意図しない出力

    対策: 必ず各caseの最後にbreak;を記述してください。

    Q3: 配列のインデックスを間違えやすいです

    A: 以下のことに注意してください： - 配列のインデックスは0から始まる - 最後の要素はarray[array.length - 1] - ArrayIndexOutOfBoundsExceptionに注意

    int[] scores = new int[5]; // インデックス0-4が有効

// ✅ 正しいアクセス
scores[0] = 100; // 最初の要素
scores[4] = 90;  // 最後の要素

// ❌ エラーになるアクセス
scores[5] = 80;  // ArrayIndexOutOfBoundsException

    Q4: mainメソッドが長くなってしまいます

    A: 以下の方針で改善してください： 1. 機能ごとにメソッドを作成 2. mainメソッドは処理の流れのみ記述 3. 一つのメソッドは20-30行程度に抑える

    // ✅ 良い例：mainは処理の流れのみ
public static void main(String[] args) {
    initializeSystem();
    while (isRunning()) {
        showMenuAndProcessChoice();
    }
    finalizeSystem();
}

    Q5: エラーハンドリングを忘れてしまいます

    A: 以下のチェックリストを使用してください： - [ ] Scanner.nextInt()を使う場所でInputMismatchExceptionを処理している - [ ] 数値の範囲チェックを行っている - [ ] 配列のインデックス範囲チェックを行っている - [ ] null文字列のチェックを行っている
