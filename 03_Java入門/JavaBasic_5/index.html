    1. なぜ繰り返し処理が必要なのか？

    問題：同じ処理を何度も書く非効率さ

    例えば、「Hello Java」を5回表示したい場合を考えてみましょう。 今までの知識だと、このように書く必要があります：

    public class RepeatExample {
    public static void main(String[] args) {
        System.out.println("Hello Java");
        System.out.println("Hello Java");
        System.out.println("Hello Java");
        System.out.println("Hello Java");
        System.out.println("Hello Java");
    }
}

    問題点: - 同じコードを5回書いている（非効率） - 100回表示したい場合はどうしますか？ - 表示回数を変更したい時、全部修正が必要

    解決策：for文による効率化

    同じ処理を何度も繰り返すときは、for文を使います！

    public class RepeatExample {
    public static void main(String[] args) {
        for (int i = 1; i <= 5; i++) {
            System.out.println("Hello Java");
        }
    }
}

    実行結果： 

    Hello Java
Hello Java
Hello Java
Hello Java
Hello Java

    たった3行で5回の処理が完了しました！

    📘 繰り返しの基本イメージ

    2. for文の基本構文

    for文の書き方

    for (初期化; 条件; 更新) {
    繰り返したい処理
}

    構文の詳細説明

    📘 for文の構文と各部分の役割

    各部分の役割: - 初期化: 繰り返し開始前に1回だけ実行 - 条件: この条件がtrueの間、繰り返しを続ける - 更新: 1回の処理が終わるたびに実行 - { }: 繰り返したい処理を囲む

    具体例で理解しよう

    for (int i = 1; i <= 5; i++) {
    System.out.println(i + "回目の処理");
}

    動作の流れ: 1. 初期化: int i = 1 （iに1を設定） 2. 条件チェック: i <= 5 （1 <= 5 → true） 3. 処理実行: System.out.println(i + "回目の処理"); 4. 更新: i++ （iが2になる） 5. 条件チェック: i <= 5 （2 <= 5 → true） 6. 3-5を繰り返す... 7. 条件チェック: i <= 5 （6 <= 5 → false） 8. 終了

    実行結果： 

    1回目の処理
2回目の処理
3回目の処理
4回目の処理
5回目の処理

    3. カウンター変数（i）について

    カウンター変数とは？

    カウンター変数は、現在何回目の処理なのかを記録する変数です。

    for (int i = 1; i <= 5; i++) {
    // iが現在の回数を表す
}

    なぜ「i」という名前？

    • 慣習: プログラミングでは繰り返し処理のカウンターにiを使うのが一般的
    • 意味: 「index（インデックス）」や「iteration（反復）」の頭文字
    • 短い: タイピングが楽で読みやすい

    i++の意味

    i++  // iの値を1増やす（i = i + 1と同じ意味）

    使用例: 

    int i = 1;
System.out.println(i); // 1が表示
i++;
System.out.println(i); // 2が表示

    例2: 1から10までの合計を計算

    これは非常に実用的な例です！

    public class SumCalculation {
    public static void main(String[] args) {
        int sum = 0; // 合計を記録する変数

        for (int i = 1; i <= 10; i++) {
            sum = sum + i; // 現在の数値を合計に加える
        }

        System.out.println("1から10までの合計: " + sum);
    }
}

    実行結果： 

    1から10までの合計: 55

    処理の流れ: 

    初期: sum = 0
1回目: sum = 0 + 1 = 1
2回目: sum = 1 + 2 = 3
3回目: sum = 3 + 3 = 6
4回目: sum = 6 + 4 = 10
5回目: sum = 10 + 5 = 15
...
10回目: sum = 45 + 10 = 55

    5. if文とfor文の組み合わせ

    前回学んだif文と組み合わせることで、より柔軟な処理が可能になります！

    例1: 偶数だけを表示

    public class EvenNumbers {
    public static void main(String[] args) {
        System.out.println("1から10までの偶数:");

        for (int i = 1; i <= 10; i++) {
            if (i % 2 == 0) { // 2で割り切れる（偶数）の場合
                System.out.println(i);
            }
        }
    }
}

    実行結果： 

    1から10までの偶数:
2
4
6
8
10

    例2: 条件によって異なるメッセージを表示

    public class NumberCheck {
    public static void main(String[] args) {
        for (int i = 1; i <= 5; i++) {
            System.out.print(i + ": ");

            if (i % 2 == 0) {
                System.out.println("偶数です");
            } else {
                System.out.println("奇数です");
            }
        }
    }
}

    実行結果： 

    1: 奇数です
2: 偶数です
3: 奇数です
4: 偶数です
5: 奇数です

    課題1: 掛け算の3の段を表示

    3 × 1 から 3 × 9 まで計算結果を表示するプログラムを作成してください。

    まずは自分で考えてみよう！

    💡 ヒント1: 基本構文 

    for (int i = 1; i <= 9; i++) {
    // ここに3 × i の計算を書く
}

    💡 ヒント2: 掛け算の結果 - 3 × i を計算して表示 - System.out.println() で「3 × 1 = 3」のような形式で表示

    課題2: 条件付き合計計算

    1から20までの数字のうち、5で割り切れる数字だけの合計を求めるプログラムを作成してください。

    まずは自分で考えてみよう！

    💡 ヒント1: 5で割り切れる条件 

    if (i % 5 == 0) {
    // 5で割り切れる場合の処理
}

    💡 ヒント2: 合計の計算方法 

    int sum = 0; // 最初に合計用の変数を用意
// for文の中で条件に合う数字をsumに加える

    💡 ヒント3: 該当する数字の表示 - 5で割り切れる数字も一緒に表示すると分かりやすい

    🎯 期待される実行結果 

    1から20までの数字のうち、5で割り切れる数字:
5
10
15
20
合計: 50

    完成例

    public class ConditionalSum {
    public static void main(String[] args) {
        System.out.println("1から20までの数字のうち、5で割り切れる数字:");

        int sum = 0; // 合計を記録する変数

        for (int i = 1; i <= 20; i++) {
            if (i % 5 == 0) { // 5で割り切れる場合
                System.out.println(i);
                sum = sum + i; // 合計に加える
            }
        }

        System.out.println("合計: " + sum);
    }
}

    💡 よくある質問

    Q1: なぜfor文でint i = 1から始めるのですか？

    A: 用途によって変わります。 - 1から始める: 人間が数える感覚（1回目、2回目...） - 0から始める: プログラミングでは0から始めることも多い - 任意の数: 必要に応じて任意の数から始められる

    for (int i = 0; i < 5; i++) { } // 0から4まで（5回）
for (int i = 1; i <= 5; i++) { } // 1から5まで（5回）
for (int i = 10; i <= 15; i++) { } // 10から15まで（6回）

    Q2: i++とは何ですか？他の書き方はありますか？

    A: i++はiを1増やす省略記法です。

    i++       // iを1増やす（推奨）
i = i + 1 // 同じ意味だが長い
i += 1    // これも同じ意味

    Q3: for文の中でiの値を変更してもいいですか？

    A: 技術的には可能ですが、初心者のうちは避けましょう。思わぬバグの原因になります。

    // 避けた方が良い例
for (int i = 1; i <= 5; i++) {
    i = i + 2; // この書き方は混乱を招く
}

    Q4: { }を忘れるとどうなりますか？

    A: 繰り返されるのは次の1行だけになります。

    // { }がない場合
for (int i = 1; i <= 3; i++)
    System.out.println("1行目"); // この行だけ繰り返される
    System.out.println("2行目"); // この行は1回だけ実行

// 正しい書き方
for (int i = 1; i <= 3; i++) {
    System.out.println("1行目"); // この行が繰り返される
    System.out.println("2行目"); // この行も繰り返される
}
