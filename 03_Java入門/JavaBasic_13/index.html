    1. なぜ入力機能が必要なのか？

    現在のプログラムの限界

    これまで作成してきたプログラムには、一つの大きな制約がありました：

    public class Calculator {
    public static void main(String[] args) {
        // 値がプログラム内で固定されている
        int a = 10;
        int b = 5;

        int result = a + b;
        System.out.printf("%d + %d = %d%n", a, b, result);
    }
}

    問題点： - 計算する数値がプログラムに固定されている - 別の計算をするにはコードを書き換える必要がある - ユーザーが自由に計算できない

    対話型プログラムの魅力

    入力機能があると、こんなプログラムが作れます：

    どのような計算をしますか？
1つ目の数値を入力してください: 25
2つ目の数値を入力してください: 13

=== 計算結果 ===
25 + 13 = 38

ありがとうございました！

    📘 対話型プログラムの概念

    対話型プログラムの利点： - ユーザーが自由にデータを入力できる - 同じプログラムで様々なケースに対応 - リアルタイムで結果を確認できる - 実用的なアプリケーションに近づく

    入力機能で実現できること

    • 電卓プログラム: ユーザーが自由に計算
    • アンケートシステム: 質問に回答を入力
    • データ管理: 学生情報の登録・検索
    • ゲーム: プレイヤーの選択を受け取る

    今回は、この入力機能の基礎をしっかり身につけましょう！

    2. Scannerクラスの基本概念

    Scannerクラスとは？

    Scannerは、Javaでユーザーからの入力を受け取るためのクラスです。

    import java.util.Scanner;  // ← Scannerを使うための準備

public class InputExample {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);  // ← 入力を受け取る準備

        System.out.print("お名前を入力してください: ");
        String name = scanner.nextLine();  // ← 実際に入力を受け取る

        System.out.println("こんにちは、" + name + "さん！");

        scanner.close();  // ← 使い終わったら閉じる（重要）
    }
}

    import文の重要性

    Scannerクラスを使用するには、必ずimport文が必要です：

    import java.util.Scanner;  // ← この行が必須！

    なぜimport文が必要？ - ScannerはJavaの標準ライブラリに含まれている - 使用前に「このクラスを使います」と宣言する必要がある - System.out.printlnは標準で使えるが、Scannerは明示的にimportする

    import文を忘れると： 

    // import文なしでScannerを使用
Scanner scanner = new Scanner(System.in);  // ❌ エラー！

    エラーメッセージ： 

    Scanner cannot be resolved to a type

    Scannerインスタンスの作成

    Scanner scanner = new Scanner(System.in);

    各部分の説明： - Scanner: クラス名（型） - scanner: 変数名（好きな名前をつけられる） - new Scanner(System.in): 新しいScannerオブジェクトを作成 - System.in: 標準入力（キーボードからの入力）を指定

    リソース管理（close()の重要性）

    scanner.close();  // ← 必ず最後に実行！

    なぜclose()が必要？ - Scannerはシステムリソースを使用する - 使い終わったら適切に解放する必要がある - close()を忘れるとメモリリークの原因になる

    良い習慣： 

    public static void main(String[] args) {
    Scanner scanner = new Scanner(System.in);

    // 入力処理

    scanner.close();  // ← main メソッドの最後で必ずclose()
}

    3. 文字列入力（nextLine()メソッド）

    nextLine()の基本使用法

    import java.util.Scanner;

public class StringInput {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("お名前を入力してください: ");
        String name = scanner.nextLine();

        System.out.print("お住まいの都道府県を入力してください: ");
        String prefecture = scanner.nextLine();

        // レッスン12で学んだprintf と組み合わせ
        System.out.printf("=== ご入力情報 ===%n");
        System.out.printf("お名前: %s%n", name);
        System.out.printf("都道府県: %s%n", prefecture);

        scanner.close();
    }
}

    実行例： 

    お名前を入力してください: 山田太郎
お住まいの都道府県を入力してください: 東京都
=== ご入力情報 ===
お名前: 山田太郎
都道府県: 東京都

    nextLine()の特徴

    利点： - スペースを含む文字列も入力できる - 日本語も正しく処理される - 改行までの全ての文字を取得

    System.out.print("好きな映画を入力してください: ");
String movie = scanner.nextLine();
// 「スター ウォーズ」のようにスペースを含む入力も可能

    注意点： - 入力の最後でEnterキーを押す必要がある - Enterキーが押されるまでプログラムは待機する - 空文字列（何も入力せずEnter）も有効な入力

    複数の文字列入力

    import java.util.Scanner;

public class MultipleStringInput {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.println("=== 自己紹介フォーム ===");

        System.out.print("お名前: ");
        String name = scanner.nextLine();

        System.out.print("職業: ");
        String job = scanner.nextLine();

        System.out.print("趣味: ");
        String hobby = scanner.nextLine();

        // 美しい出力でまとめて表示
        System.out.printf("%n=== 自己紹介 ===%n");
        System.out.printf("お名前: %s%n", name);
        System.out.printf("職業:   %s%n", job);
        System.out.printf("趣味:   %s%n", hobby);
        System.out.printf("よろしくお願いします！%n");

        scanner.close();
    }
}

    4. 数値入力（nextInt()とnextDouble()）

    nextInt()による整数入力

    import java.util.Scanner;

public class NumberInput {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("年齢を入力してください: ");
        int age = scanner.nextInt();

        System.out.print("身長を入力してください（cm）: ");
        int height = scanner.nextInt();

        // 入力された数値で計算
        int birth_year = 2025 - age;

        System.out.printf("%n=== 入力情報 ===%n");
        System.out.printf("年齢: %d歳%n", age);
        System.out.printf("身長: %dcm%n", height);
        System.out.printf("生まれ年: 約%d年%n", birth_year);

        scanner.close();
    }
}

    実行例： 

    年齢を入力してください: 25
身長を入力してください（cm）: 170

=== 入力情報 ===
年齢: 25歳
身長: 170cm
生まれ年: 約2000年

    nextDouble()による小数入力

    import java.util.Scanner;

public class DoubleInput {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

    5. 入力メソッドの使い分けと注意点

    主要な入力メソッド比較

    📘 入力手法の比較

    メソッド 用途 特徴 注意点
    nextLine() 文字列入力 スペース含む、改行まで全て取得 改行文字も処理対象
    next() 単語入力 スペースで区切られた単語 スペース以降は無視
    nextInt() 整数入力 int型の数値のみ 数値以外でエラー
    nextDouble() 小数入力 double型の数値のみ 数値以外でエラー

    next() vs nextLine()の違い

    import java.util.Scanner;

public class NextVsNextLine {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("next()で入力してください: ");
        String word1 = scanner.next();  // 最初の単語のみ取得

        System.out.print("nextLine()で入力してください: ");
        String line1 = scanner.nextLine();  // 残りの部分＋次の行

        System.out.printf("next()の結果: &%s&%n", word1);
        System.out.printf("nextLine()の結果: &%s&%n", line1);

        scanner.close();
    }
}

    入力例： 

    next()で入力してください: Hello World
nextLine()で入力してください: 
next()の結果: 'Hello'
nextLine()の結果: ' World'

    ポイント： - next()は最初の単語（Hello）のみ取得 - nextLine()は残りの部分（" World"）を取得 - 多くの場合、nextLine()の方が使いやすい

    nextInt()とnextLine()の組み合わせ注意

    import java.util.Scanner;

public class MixedInput {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("年齢を入力してください: ");
        int age = scanner.nextInt();

        // 改行文字が残っている問題の解決
        scanner.nextLine();  // ← この行が重要！

        System.out.print("お名前を入力してください: ");
        String name = scanner.nextLine();

        System.out.printf("年齢: %d歳, 名前: %s%n", age, name);

        scanner.close();
    }
}

    なぜscanner.nextLine()が必要？ - nextInt()は数値のみを読み取り、改行文字は残る - その後のnextLine()が残った改行文字を読み取ってしまう - 空のnextLine()で改行文字を消去する必要がある

    6. 基本的なエラー回避テクニック

    hasNextInt()による事前チェック

    import java.util.Scanner;

public class SafeNumberInput {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("整数を入力してください: ");

        if (scanner.hasNextInt()) {
            // 整数が入力された場合
            int number = scanner.nextInt();
            System.out.printf("入力された数値: %d%n", number);
        } else {
            // 整数以外が入力された場合
            String input = scanner.next();
            System.out.printf("&%s& は整数ではありません。%n", input);
        }

        scanner.close();
    }
}

    安全な入力処理のパターン

    import java.util.Scanner;

public class SafeInputPattern {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        // 整数入力の安全なパターン
        int age = 0;
        while (true) {
            System.out.print("年齢を入力してください: ");
            if (scanner.hasNextInt()) {
                age = scanner.nextInt();
                break;  // 正常入力で終了
            } else {
                System.out.println("整数を入力してください。");
                scanner.next();  // 不正入力を消去
            }
        }

        System.out.printf("年齢: %d歳で受け付けました。%n", age);

        scanner.close();
    }
}

    import java.util.Scanner;

public class InteractiveCalculator {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.printf("=== 対話型計算機 ===%n");

        // 1つ目の数値入力
        System.out.print("1つ目の数値: ");
        if (!scanner.hasNextDouble()) {
            System.out.println("数値を入力してください。");
            scanner.close();
            return;
        }
        double num1 = scanner.nextDouble();

        // 2つ目の数値入力
        System.out.print("2つ目の数値: ");
        if (!scanner.hasNextDouble()) {
            System.out.println("数値を入力してください。");
            scanner.close();
            return;
        }
        double num2 = scanner.nextDouble();

        // 計算実行
        double addition = num1 + num2;
        double subtraction = num1 - num2;
        double multiplication = num1 * num2;

        // 美しい結果表示
        System.out.printf("%n=== 計算結果 ===%n");
        System.out.printf("┌────────────┬──────────┐%n");
        System.out.printf("│ 演算       │   結果   │%n");
        System.out.printf("├────────────┼──────────┤%n");
        System.out.printf("│ %-10s │ %8.2f │%n", "足し算", addition);
        System.out.printf("│ %-10s │ %8.2f │%n", "引き算", subtraction);
        System.out.printf("│ %-10s │ %8.2f │%n", "掛け算", multiplication);

        // 割り算はゼロチェック
        if (num2 != 0) {
            double division = num1 / num2;
            System.out.printf("│ %-10s │ %8.2f │%n", "割り算", division);
        } else {
            System.out.printf("│ %-10s │ %8s │%n", "割り算", "不可");
        }

        System.out.printf("└────────────┴──────────┘%n");

        scanner.close();
    }
}

    💡 よくある質問

    Q1: import文を書く場所は決まっていますか？

    A: はい、import文はpackage文の後、class定義の前に書きます。

    // 1. package文（今回は使用しない）
// 2. import文
import java.util.Scanner;

// 3. class定義
public class Example {
    // ...
}

    Q2: Scanner変数の名前は「scanner」でないとダメですか？

    A: いいえ、任意の名前をつけられます。ただし、分かりやすい名前を推奨します。

    Scanner input = new Scanner(System.in);     // ✅ OK
Scanner sc = new Scanner(System.in);        // ✅ OK  
Scanner userInput = new Scanner(System.in); // ✅ OK
Scanner a = new Scanner(System.in);         // △ 分かりにくい

    Q3: close()を忘れるとどうなりますか？

    A: プログラムは動作しますが、リソースリークが発生する可能性があります。必ず実行しましょう。

    // ❌ 悪い例
Scanner scanner = new Scanner(System.in);
String input = scanner.nextLine();
// close()を忘れている

// ✅ 良い例
Scanner scanner = new Scanner(System.in);
String input = scanner.nextLine();
scanner.close();  // 必ず実行

    Q4: nextInt()の後にnextLine()を使うと空文字が返されます

    A: これは改行文字が残るためです。nextInt()の後に空のnextLine()を実行して改行文字を消去してください。

    int age = scanner.nextInt();
scanner.nextLine();  // ← 改行文字を消去
String name = scanner.nextLine();  // ← 正常に動作

    Q5: hasNextInt()はどのような場面で使いますか？

    A: ユーザーが間違った形式で入力する可能性がある場合に使用します。特に実用的なプログラムでは必須です。

    // 安全な入力処理
if (scanner.hasNextInt()) {
    int number = scanner.nextInt();
    // 正常処理
} else {
    System.out.println("整数を入力してください。");
    scanner.next();  // 不正入力を消去
}
