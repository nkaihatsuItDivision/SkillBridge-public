    📋 1. @Controllerと画面表示の基礎

    @RestControllerと@Controllerの違い

    Week 8.7 で学んだ @RestController と今回の @Controller の違いを理解しましょう：

    📘 @RestController vs @Controller の比較図

    @RestController vs @Controller の比較図
- @RestController: JSON/XMLを直接返す（API専用）
- @Controller: HTMLテンプレートを返す（画面表示用）

    @Controllerの基本実装

    Week 7.10で準備済みの StudentViewController.java を見てみましょう：

    📘 StudentViewController.java のコード例

    @Controller<br>
@RequestMapping("/students")<br>
public class StudentViewController {<br>
    <br>
    @Autowired<br>
    private StudentService studentService;<br>
    <br>
    @GetMapping<br>
    public String list(Model model) {<br>
        // データベースから学生一覧を取得<br>
        List<Student> students = studentService.findAll();<br>
        <br>
        // Modelオブジェクトにデータを追加<br>
        model.addAttribute("students", students);<br>
        <br>
        // テンプレート名を返す<br>
        return "student-list";<br>
    }<br>
}

    重要なポイント解説

    1. @Controllerアノテーション - 画面表示専用のControllerであることを宣言 - Spring Bootが自動的に画面表示用として認識

    2. Modelオブジェクト - データをView（画面）に渡すためのオブジェクト - model.addAttribute("キー", 値) でデータを格納

    3. 戻り値の違い - @RestController: JavaオブジェクトやJSONを直接返す - @Controller: テンプレート名（文字列）を返す

    📘 Model-View連携の仕組み図

    Model-View連携の仕組み図
Controller → Model（データ） → View（テンプレート） → HTML

    Thymeleaf重要属性解説

    1. th:each - 繰り返し処理 

    <tr th:each="student : ${students}">
    - students リストの各要素を student として繰り返し - Java の拡張for文と同じ概念

    2. th:text - テキスト表示 

    <td th:text="${student.name}"></td>
    - ${student.name} でJavaオブジェクトのフィールドにアクセス - HTMLエスケープが自動的に実行される

    3. th:href - リンク生成 

    <link rel="stylesheet" th:href="@{/css/style.css}">
    - @{/css/style.css} で静的リソースのパスを生成

    📘 Thymeleaf属性の活用例図

    Thymeleaf属性の活用例

th:text → 動的テキスト表示
th:href → 動的リンク生成

    StudentViewController.java 完成版:

    @Controller
@RequestMapping("/students")
public class StudentViewController {

    @Autowired
    private StudentService studentService;

    @GetMapping
    public String list(Model model) {
        List<Student> students = studentService.findAll();
        model.addAttribute("students", students);
        return "student-list";
    }

    @GetMapping("/{id}")
    public String detail(@PathVariable Long id, Model model) {
        Student student = studentService.findById(id);
        model.addAttribute("student", student);
        return "student-detail";
    }

    @GetMapping("/search")
    public String search(@RequestParam(required = false) String name, Model model) {
        List<Student> students;
        if (name != null && !name.isEmpty()) {
            students = studentService.findByNameContaining(name);
        } else {
            students = studentService.findAll();
        }

        model.addAttribute("students", students);
        model.addAttribute("searchName", name);

        return "student-list";
    }
}
