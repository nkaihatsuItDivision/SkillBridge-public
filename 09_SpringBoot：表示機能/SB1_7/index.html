    🌐 @RestControllerとHTTP基礎

    Web層の役割と重要性

    Spring Boot MVCアーキテクチャにおいて、ControllerはWeb層を担当し、外部からのHTTPリクエストを受信して、適切なレスポンスを返す重要な役割を持ちます。

    📘 Web層のアーキテクチャ

    @RestControllerアノテーションの威力

    @RestController
@RequestMapping("/api/students")
public class StudentController {

    @Autowired
    private StudentService studentService;

    @GetMapping
    public List<Student> getAllStudents() {
        return studentService.findAll();
    }
}

    @RestControllerの特徴

    1. HTTP通信の自動処理: HTTPリクエストを自動的にJavaメソッドにマッピング
    2. JSON変換の自動化: Javaオブジェクトを自動的にJSONに変換してレスポンス
    3. アノテーション駆動: 設定ファイル不要で直感的な実装
    4. RESTful対応: REST APIの標準パターンを簡単に実装

    HTTP基礎の重要ポイント

    参考リンク： 📘 HTTPメソッドとその操作

    HTTPメソッドと操作の対応

    HTTPメソッド 操作 説明 例
    GET 取得 データの取得（副作用なし） /api/students
    POST 作成 新しいデータの作成 /api/students
    PUT 更新 既存データの完全更新 /api/students/1
    DELETE 削除 データの削除 /api/students/1

        // Week 8.6のMapper技術を透過的に活用

    }

    package com.example.demo.controller;

import com.example.demo.entity.Student;
import com.example.demo.service.StudentService;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.http.HttpStatus;
import org.springframework.http.ResponseEntity;
import org.springframework.web.bind.annotation.*;

import java.util.List;
import java.util.Optional;

@RestController
@RequestMapping("/api/students")
@CrossOrigin(origins = "*") // CORS対応
public class StudentController {

    @Autowired
    private StudentService studentService;

    /**
     * 全学生取得 API
     * GET /api/students
     */
    @GetMapping
    public ResponseEntity<List<Student>> getAllStudents() {
        try {
            List<Student> students = studentService.findAll();
            return ResponseEntity.ok(students);
        } catch (Exception e) {
            return ResponseEntity.status(HttpStatus.INTERNAL_SERVER_ERROR).build();
        }
    }

    /**
     * 特定学生取得 API
     * GET /api/students/{id}
     */
    @GetMapping("/{id}")
    public ResponseEntity<Student> getStudentById(@PathVariable Long id) {
        try {
            Optional<Student> student = studentService.findById(id);
            if (student.isPresent()) {
                return ResponseEntity.ok(student.get());
            } else {
                return ResponseEntity.notFound().build();
            }
        } catch (Exception e) {
            return ResponseEntity.status(HttpStatus.INTERNAL_SERVER_ERROR).build();
        }
    }

    /**
     * 学生作成 API
     * POST /api/students
     */
    @PostMapping
    public ResponseEntity<Student> createStudent(@RequestBody Student student) {
        try {
            Student createdStudent = studentService.save(student);
            return ResponseEntity.status(HttpStatus.CREATED).body(createdStudent);
        } catch (Exception e) {
            return ResponseEntity.status(HttpStatus.BAD_REQUEST).build();
        }
    }

    /**
     * 学生更新 API
     * PUT /api/students/{id}
     */
    @PutMapping("/{id}")
    public ResponseEntity<Student> updateStudent(@PathVariable Long id, @RequestBody Student student) {
        try {
            student.setId(id);
            Student updatedStudent = studentService.save(student);
            return ResponseEntity.ok(updatedStudent);
        } catch (Exception e) {
            return ResponseEntity.status(HttpStatus.BAD_REQUEST).build();
        }
    }

    /**
     * 学生削除 API
     * DELETE /api/students/{id}
     */
    @DeleteMapping("/{id}")
    public ResponseEntity<Void> deleteStudent(@PathVariable Long id) {
        try {
            studentService.deleteById(id);
            return ResponseEntity.noContent().build();
        } catch (Exception e) {
            return ResponseEntity.status(HttpStatus.INTERNAL_SERVER_ERROR).build();
        }
    }
}

    Controller実装のポイント

    📘 コントローラー実装のポイント

    アノテーションの詳細解説

    @RestController

    • @Controller + @ResponseBodyの組み合わせ
    • メソッドの戻り値を自動的にJSONに変換
    • HTTPレスポンスのボディに直接出力

    @RequestMapping("/api/students")

    • クラス全体に共通のURLパスを設定
    • 各メソッドのパスはこのベースパスに追加される

    @GetMapping、@PostMapping等

    • HTTPメソッド別のリクエストマッピング
    • 短縮記法で読みやすいコード

    @PathVariable

    • URLパスの一部を変数として取得
    • /api/students/{id} の {id} 部分

    @RequestBody

    • HTTPリクエストのボディをJavaオブジェクトに変換
    • JSON → Studentオブジェクトの自動変換

    RESTful設計の基本原則

    REST（Representational State Transfer）は、Web APIの設計パターンの標準的な考え方です。Spring Bootを使用することで、RESTfulなAPIを簡単に実装できます。

    エラーハンドリングの重要性

    実際のWebアプリケーションでは、様々なエラーが発生する可能性があります。適切なエラーハンドリングにより、ユーザーに分かりやすいエラーメッセージを提供し、システムの堅牢性を向上させることができます。

    GlobalExceptionHandlerの実装

    package com.example.demo.exception;

import org.springframework.http.HttpStatus;
import org.springframework.http.ResponseEntity;
import org.springframework.web.bind.annotation.ExceptionHandler;
import org.springframework.web.bind.annotation.RestControllerAdvice;

import java.time.LocalDateTime;
import java.util.HashMap;
import java.util.Map;

@RestControllerAdvice
public class GlobalExceptionHandler {

    /**
     * データが見つからない場合の例外処理
     */
    @ExceptionHandler(StudentNotFoundException.class)
    public ResponseEntity<Map<String, Object>> handleStudentNotFound(StudentNotFoundException ex) {
        Map<String, Object> errorResponse = new HashMap<>();
        errorResponse.put("timestamp", LocalDateTime.now());
        errorResponse.put("status", HttpStatus.NOT_FOUND.value());
        errorResponse.put("error", "Not Found");
        errorResponse.put("message", ex.getMessage());
        errorResponse.put("path", "/api/students");

        return ResponseEntity.status(HttpStatus.NOT_FOUND).body(errorResponse);
    }

    /**
     * バリデーションエラーの例外処理
     */
    @ExceptionHandler(IllegalArgumentException.class)
    public ResponseEntity<Map<String, Object>> handleIllegalArgument(IllegalArgumentException ex) {
        Map<String, Object> errorResponse = new HashMap<>();
        errorResponse.put("timestamp", LocalDateTime.now());
        errorResponse.put("status", HttpStatus.BAD_REQUEST.value());
        errorResponse.put("error", "Bad Request");
        errorResponse.put("message", ex.getMessage());

        return ResponseEntity.status(HttpStatus.BAD_REQUEST).body(errorResponse);
    }

    /**
     * 一般的な例外処理
     */
    @ExceptionHandler(Exception.class)
    public ResponseEntity<Map<String, Object>> handleGenericException(Exception ex) {
        Map<String, Object> errorResponse = new HashMap<>();
        errorResponse.put("timestamp", LocalDateTime.now());
        errorResponse.put("status", HttpStatus.INTERNAL_SERVER_ERROR.value());
        errorResponse.put("error", "Internal Server Error");
        errorResponse.put("message", "予期しないエラーが発生しました");

        return ResponseEntity.status(HttpStatus.INTERNAL_SERVER_ERROR).body(errorResponse);
    }
}

    標準レスポンス形式の実装

    Week 7.10で準備したApiResponse.javaを活用した統一レスポンス形式：

    package com.example.demo.response;

import com.fasterxml.jackson.annotation.JsonInclude;
import java.time.LocalDateTime;

@JsonInclude(JsonInclude.Include.NON_NULL)
public class ApiResponse<T> {
    private boolean success;
    private String message;
    private T data;
    private LocalDateTime timestamp;

    // コンストラクタ
    public ApiResponse(boolean success, String message, T data) {
        this.success = success;
        this.message = message;
        this.data = data;
        this.timestamp = LocalDateTime.now();
    }

    // 成功レスポンス用の静的メソッド
    public static <T> ApiResponse<T> success(T data) {
        return new ApiResponse<>(true, "成功", data);
    }

    public static <T> ApiResponse<T> success(String message, T data) {
        return new ApiResponse<>(true, message, data);
    }

    // エラーレスポンス用の静的メソッド
    public static <T> ApiResponse<T> error(String message) {
        return new ApiResponse<>(false, message, null);
    }

    // ゲッター・セッター
    public boolean isSuccess() { return success; }
    public void setSuccess(boolean success) { this.success = success; }

    public String getMessage() { return message; }
    public void setMessage(String message) { this.message = message; }

    public T getData() { return data; }
    public void setData(T data) { this.data = data; }

    public LocalDateTime getTimestamp() { return timestamp; }
    public void setTimestamp(LocalDateTime timestamp) { this.timestamp = timestamp; }
}

    🎯 StudentController完成版 
    package com.example.demo.controller;

import com.example.demo.entity.Student;
import com.example.demo.service.StudentService;
import com.example.demo.exception.StudentNotFoundException;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.http.HttpStatus;
import org.springframework.http.ResponseEntity;
import org.springframework.web.bind.annotation.*;

import java.util.List;
import java.util.Optional;

@RestController
@RequestMapping("/api/students")
@CrossOrigin(origins = "*")
public class StudentController {

    @Autowired
    private StudentService studentService;

    /**
     * 全学生取得 API
     */
    @GetMapping
    public ResponseEntity<List<Student>> getAllStudents() {
        List<Student> students = studentService.findAll();
        return ResponseEntity.ok(students);
    }

    /**
     * 特定学生取得 API
     */
    @GetMapping("/{id}")
    public ResponseEntity<Student> getStudentById(@PathVariable Long id) {
        validateId(id);
        Optional<Student> student = studentService.findById(id);
        return student.map(ResponseEntity::ok)
                     .orElseThrow(() -> new StudentNotFoundException(id));
    }

    /**
     * 学生作成 API
     */
    @PostMapping
    public ResponseEntity<Student> createStudent(@RequestBody Student student) {
        validateStudent(student);
        Student createdStudent = studentService.save(student);
        return ResponseEntity.status(HttpStatus.CREATED).body(createdStudent);
    }

    /**
     * 学生更新 API
     */
    @PutMapping("/{id}")
    public ResponseEntity<Student> updateStudent(@PathVariable Long id, @RequestBody Student student) {
        validateId(id);

        // 存在確認
        if (!studentService.findById(id).isPresent()) {
            throw new StudentNotFoundException(id);
        }

        student.setId(id);
        validateStudent(student);
        Student updatedStudent = studentService.save(student);
        return ResponseEntity.ok(updatedStudent);
    }

    /**
     * 学生削除 API
     */
    @DeleteMapping("/{id}")
    public ResponseEntity<Void> deleteStudent(@PathVariable Long id) {
        validateId(id);

        // 存在確認
        if (!studentService.findById(id).isPresent()) {
            throw new StudentNotFoundException(id);
        }

        studentService.deleteById(id);
        return ResponseEntity.noContent().build();
    }

    /**
     * ID バリデーション
     */
    private void validateId(Long id) {
        if (id == null || id <= 0) {
            throw new IllegalArgumentException("IDは1以上の数値である必要があります");
        }
    }

    /**
     * 学生データ バリデーション
     */
    private void validateStudent(Student student) {
        if (student.getName() == null || student.getName().trim().isEmpty()) {
            throw new IllegalArgumentException("学生名は必須です");
        }

        if (student.getEmail() == null || !isValidEmail(student.getEmail())) {
            throw new IllegalArgumentException("有効なメールアドレスを入力してください");
        }

        if (student.getAge() < 15 || student.getAge() > 100) {
            throw new IllegalArgumentException("年齢は15歳以上100歳以下で入力してください");
        }

        if (student.getGrade() == null || student.getGrade().trim().isEmpty()) {
            throw new IllegalArgumentException("成績は必須です");
        }
    }

    /**
     * メールアドレス形式チェック
     */
    private boolean isValidEmail(String email) {
        return email.contains("@") && email.contains(".") && email.length() > 5;
    }
}
