    2. 新規プロジェクト作成：右クリック → New → Other...
    3. Spring Starter Project選択：Spring Boot → Spring Starter Projectを選択

    📘 Eclipseでの新規プロジェクト作成

    🛠️ 手順2: プロジェクト基本情報の設定

    プロジェクト作成ウィザードで以下の設定を行います：

    プロジェクト設定：
Name: StudentSpringBootMyBatisProject
Type: Maven
Language: Java
Packaging: Jar
Java Version: 17
Group: com.example
Artifact: demo
Version: 0.0.1-SNAPSHOT
Description: Student Management System using Spring Boot and MyBatis
Package: com.example.demo

    重要なポイント： - Name：StudentSpringBootMyBatisProject（Week 7.10のapplication.propertiesと一致） - Package：com.example.demo（Week 7.10のクラスパッケージと一致） - Java Version：17（Week 5のLombok動作確認済み）

    📘 プロジェクト設定ウィザード

    🛠️ 手順3: 依存関係の選択

    Spring Boot Starter依存関係として以下を選択します：

    必須依存関係： - ✅ Spring Boot DevTools：開発効率向上（自動再起動） - ✅ Spring Web：Web機能の基本（Controller、REST API） - ✅ Thymeleaf：テンプレートエンジン（動的HTML生成） - ✅ MySQL Driver：データベース接続（Week 7のMySQLに接続） - ✅ MyBatis Framework：ORM機能（Javaクラス⇔データベース自動連携）

    追加依存関係（pom.xmlで後から追加）： - Lombok：Week 5で習得済み（@Data等のアノテーション） - Validation：Week 9のフォームバリデーションで使用

    <?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE mapper PUBLIC "-//mybatis.org//DTD Mapper 3.0//EN" 
    "http://mybatis.org/dtd/mybatis-3-mapper.dtd">

<mapper namespace="com.example.demo.repository.StudentMapper">

    <!-- Week 8〜10で共通使用する結果マッピング -->
    <resultMap id="studentResultMap" type="com.example.demo.entity.Student">
        <id property="studentId" column="student_id"/>
        <result property="name" column="name"/>
        <result property="email" column="email"/>
        <result property="gpa" column="gpa"/>
        <result property="status" column="status"/>
        <result property="departmentId" column="department_id"/>
    </resultMap>

    <!-- Week 8.4で最初に使用：全学生の取得 -->
    <select id="findAll" resultMap="studentResultMap">
        SELECT student_id, name, email, gpa, status, department_id
        FROM students
        WHERE status = 'active'
        ORDER BY student_id
    </select>

    <!-- Week 8.5で使用：学生詳細取得 -->
    <select id="findById" resultMap="studentResultMap">
        SELECT student_id, name, email, gpa, status, department_id
        FROM students
        WHERE student_id = #{id} AND status = 'active'
    </select>

    <!-- Week 9〜10で使用するSQL（今回は準備のみ） -->
    <insert id="save">
        INSERT INTO students (name, email, gpa, department_id, status)
        VALUES (#{name}, #{email}, #{gpa}, #{departmentId}, 'active')
    </insert>

    <update id="update">
        UPDATE students 
        SET name = #{name}, email = #{email}, gpa = #{gpa}, department_id = #{departmentId}
        WHERE student_id = #{studentId}
    </update>

    <delete id="delete">
        UPDATE students SET status = 'deleted' WHERE student_id = #{id}
    </delete>

</mapper>
