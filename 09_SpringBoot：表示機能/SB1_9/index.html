    9.9 Thymeleaf表示機能：条件分岐と動的スタイリング

    text Week 8.8で習得したThymeleafの基本的な表示技術を一歩発展させ、条件分岐表示、動的スタイリング、Week 1 CSS技術との基本統合を学習。Week 8.8からの自然な進歩として、より豊かで実用的な表示機能を段階的に習得し、Week 9のフォーム処理学習への準備を整える。 9.Thymeleaf表示機能：条件分岐と動的スタイリング # 8.9 Thymeleaf表示機能：条件分岐と動的スタイリング

    🎯 このレッスンの目標

    Week 8.8で習得したThymeleafの基本表示技術を一歩発展させ、より豊かで実用的な表示機能を実現します。

    技術目標

    • 条件分岐表示: th:if・th:unlessによる表示制御
    • 動的スタイリング: th:classappendによる見た目の変化
    • Week 1技術統合: CSS技術との基本的な連携
    • 統計情報表示: データを分析して表示する機能
    • エラーハンドリング: 適切なメッセージ表示

    価値実感目標

    • 表示の柔軟性: 条件に応じた表示制御の便利さ
    • ユーザビリティ向上: 見やすく分かりやすい画面作り
    • 技術統合: Week 1からの学習成果の活用
    • Week 9への準備: フォーム処理に必要な表示技術の基盤

    学習時間: 約50分

    📚 Week 8.8からの発展

    習得済み基礎技術（Week 8.8）

    Week 8.8では、基本的な表示技術を完全マスターしました：

    • th:each: リスト形式でのデータ表示
    • th:text: テキストの動的挿入
    • th:href: リンクの動的生成
    • Model属性: ControllerからViewへのデータ受け渡し
    • 基本レイアウト: HTMLとCSSの基本的な組み合わせ

    今回学ぶ発展技術

    基礎技術を活用して、一歩進んだ表示機能を実装します：

    Week 8.8（基礎表示）        Week 8.9（発展表示）
├── th:each による表示   →   条件分岐による表示制御
├── th:text によるテキスト →   動的スタイルによる見た目変化
├── 基本的なリスト表示   →   統計情報付きリスト表示
├── 固定的な表示        →   エラーハンドリング表示
└── 基本CSS            →   Week 1技術との統合

    1. 条件分岐表示：th:ifとth:unlessの活用

    🎯 なぜ条件分岐表示が重要なのか

    実際のWebアプリケーションでは、データの状態に応じて表示を変えることが頻繁にあります。

    例: - データが存在しない場合は「データがありません」を表示 - 成績が良い学生には「優秀」マークを表示 - エラーがある場合はエラーメッセージを表示

    📘 条件付き表示の重要性

    🔧 ステップ1: 基本的な条件分岐

    4. エラーハンドリング：適切なメッセージ表示

    ⚠️ エラーハンドリングの重要性

    実際のWebアプリケーションでは、エラーが発生した場合に適切なメッセージを表示することが重要です。

    🔧 ステップ1: 基本的なエラー表示

    Controllerでのエラーハンドリング: 

    @Controller
@RequestMapping("/students")
public class StudentViewController {

    @Autowired
    private StudentService studentService;

    @GetMapping
    public String list(Model model) {
        try {
            List<Student> students = studentService.findAll();

            // 統計情報の計算
            double averageGpa = students.stream()
                .mapToDouble(Student::getGpa)
                .average()
                .orElse(0.0);

            long excellentCount = students.stream()
                .filter(s -> s.getGpa() >= 3.5)
                .count();

            model.addAttribute("students", students);
            model.addAttribute("averageGpa", averageGpa);
            model.addAttribute("excellentCount", excellentCount);

            // 成功メッセージ（データが存在する場合）
            if (students.size() > 0) {
                model.addAttribute("successMessage", 
                    "学生データを " + students.size() + " 件取得しました。");
            }

        } catch (Exception e) {
            // エラーが発生した場合
            model.addAttribute("errorMessage", 
                "学生データの取得中にエラーが発生しました。しばらく時間をおいて再試行してください。");
            model.addAttribute("students", new ArrayList<Student>());
            model.addAttribute("averageGpa", 0.0);
            model.addAttribute("excellentCount", 0L);
        }

        return "student-list";
    }

    @GetMapping("/{id}")
    public String detail(@PathVariable Long id, Model model) {
        try {
            Student student = studentService.findById(id);
            model.addAttribute("student", student);
            return "student-detail";
        } catch (StudentNotFoundException e) {
            model.addAttribute("errorMessage", 
                "指定された学生（ID: " + id + "）が見つかりません。");
            return "redirect:/students";
        } catch (Exception e) {
            model.addAttribute("errorMessage", 
                "学生情報の取得中にエラーが発生しました。");
            return "redirect:/students";
        }
    }
}

        });

        // 2. 表の行にホバー効果を追加
        const tableRows = document.querySelectorAll('.table tbody tr');
        tableRows.forEach(function(row) {
            row.addEventListener('mouseenter', function() {
                this.style.backgroundColor = '#f8f9fa';
            });

            row.addEventListener('mouseleave', function() {
                this.style.backgroundColor = '';
            });
        });

        // 3. 統計カードのクリック効果
        const statCards = document.querySelectorAll('.stats-card');
        statCards.forEach(function(card) {
            card.addEventListener('click', function() {
                // カードがクリックされた時の簡単なアニメーション
                this.style.transform = 'scale(0.98)';
                setTimeout(() => {
                    this.style.transform = '';
                }, 100);
            });
        });

        // 4. ページの読み込み完了通知
        console.log('学生管理システムが正常に読み込まれました');
    });
</script>
