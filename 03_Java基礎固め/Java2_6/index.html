    📝 ステップ1: 文字列の基本情報を取得しよう

    まずは、文字列の 長さ や 特定の位置の文字 を取得する基本メソッドから始めましょう。

    1-1: length() - 文字列の長さを知る

    public class StringBasics1 {
    public static void main(String[] args) {
        // 好きな名前で試してみてください
        String name = &quot;田中太郎&quot;;
        String nickname = &quot;たろう&quot;;

        System.out.println(&quot;=== 文字列の長さチェック ===&quot;);
        System.out.println(&quot;名前: &quot; + name);
        System.out.println(&quot;文字数: &quot; + name.length() + &quot;文字&quot;);

        System.out.println(&quot;ニックネーム: &quot; + nickname);
        System.out.println(&quot;文字数: &quot; + nickname.length() + &quot;文字&quot;);

        // 長さによる判定
        if (name.length() &gt; 5) {
            System.out.println(&quot;フルネームは長めですね&quot;);
        } else {
            System.out.println(&quot;フルネームは短めですね&quot;);
        }
    }
}

    実行結果： 

    === 文字列の長さチェック ===
名前: 田中太郎
文字数: 4文字
ニックネーム: たろう
文字数: 3文字
フルネームは短めですね

    1-2: charAt() - 特定の位置の文字を取得

    📘 文字列インデックスの概念図

    public class StringBasics2 {
    public static void main(String[] args) {
        String product = &quot;iPhone15&quot;;

        System.out.println(&quot;=== 文字の位置チェック ===&quot;);
        System.out.println(&quot;商品名: &quot; + product);
        System.out.println(&quot;最初の文字: &quot; + product.charAt(0));
        System.out.println(&quot;最後の文字: &quot; + product.charAt(product.length() - 1));

        // 一文字ずつ表示
        System.out.println(&quot;一文字ずつ表示:&quot;);
        for (int i = 0; i &lt; product.length(); i++) {
            System.out.println(i + &quot;番目: &quot; + product.charAt(i));
        }
    }
}

    実行結果： 

    === 文字の位置チェック ===
商品名: iPhone15
最初の文字: i
最後の文字: 5
一文字ずつ表示:
0番目: i
1番目: P
2番目: h
3番目: o
4番目: n
5番目: e
6番目: 1
7番目: 5

    ポイント： - 文字の位置は 0から始まる（上の図解を参照） - 最後の文字は length() - 1 の位置にある - 範囲外の位置を指定するとエラーになるので注意

    2-2: indexOf() - 文字列の位置を調べる

    public class StringSearch2 {
    public static void main(String[] args) {
        String filename = &quot;report_2024_final.pdf&quot;;

        System.out.println(&quot;=== ファイル名解析 ===&quot;);
        System.out.println(&quot;ファイル名: &quot; + filename);

        // 拡張子の位置を探す
        int dotPosition = filename.indexOf(&quot;.&quot;);
        if (dotPosition != -1) {
            String extension = filename.substring(dotPosition + 1);

        }

        int yearPosition = filename.indexOf(&quot;2024&quot;);

            System.out.println(&quot;年度情報の位置: &quot; + yearPosition + &quot;文字目&quot;);
            System.out.println(&quot;このファイルは2024年のものです&quot;);
        }
    }
}

    === ファイル名解析 ===

    ✂️ ステップ3: 文字列を切り取ろう

    文字列の 一部分を取り出す 方法を学びます。

    3-1: substring() - 文字列の一部を取得

    📘 部分文字列の範囲図

    public class StringSubstring {
    public static void main(String[] args) {
        String phoneNumber = &quot;090-1234-5678&quot;;
        String url = &quot;https://www.example.com/products&quot;;

        System.out.println(&quot;=== 文字列の部分取得 ===&quot;);

        // 電話番号の分割
        System.out.println(&quot;電話番号: &quot; + phoneNumber);
        String area = phoneNumber.substring(0, 3);        // 0〜2文字目
        String middle = phoneNumber.substring(4, 8);      // 4〜7文字目
        String last = phoneNumber.substring(9);           // 9文字目以降全て

        System.out.println(&quot;エリアコード: &quot; + area);
        System.out.println(&quot;中間番号: &quot; + middle);
        System.out.println(&quot;下4桁: &quot; + last);

        // URLからドメイン名を取得
        System.out.println(&quot;\nURL: &quot; + url);
        String domain = url.substring(8, url.indexOf(&quot;/&quot;, 8));  // https://の後から/まで
        System.out.println(&quot;ドメイン名: &quot; + domain);
    }
}

    実行結果： 

    === 文字列の部分取得 ===
電話番号: 090-1234-5678
エリアコード: 090
中間番号: 1234
下4桁: 5678

URL: https://www.example.com/products
ドメイン名: www.example.com

    重要ポイント： - substring(開始, 終了) - 終了位置は含まれません（図解参照） - substring(開始) - 開始位置から最後まで取得 - 範囲外を指定するとエラーになるので注意

    🔄 ステップ4: 文字列を変換しよう

    public class StringReplace {
    public static void main(String[] args) {
        String message = &quot;今日は雨ですが、雨のおかげで涼しいです&quot;;

    4-2: 大文字・小文字変換

    public class StringCase {
    public static void main(String[] args) {

        String input = &quot;tokyo software inc.&quot;;

    📋 ステップ5: 文字列を分割・結合しよう

    文字列を 分割 したり、複数の文字列を 結合 する方法を学びます。

    5-1: split() - 文字列を分割

    public class StringSplit {
    public static void main(String[] args) {
        String fruits = &quot;りんご,みかん,バナナ,ぶどう&quot;;
        String address = &quot;東京都-渋谷区-恵比寿-1-2-3&quot;;

        System.out.println(&quot;果物リスト: &quot; + fruits);

        System.out.println(&quot;\n住所: &quot; + address);
        String[] addressParts = address.split(&quot;-&quot;);
        System.out.println(&quot;都道府県: &quot; + addressParts[0]);
        System.out.println(&quot;市区町村: &quot; + addressParts[1]);
        System.out.println(&quot;町名: &quot; + addressParts[2]);
    }
}

    5-2: String.join() - 文字列を結合

    public class StringJoin {
    public static void main(String[] args) {
        String[] skills = {&quot;Java&quot;, &quot;HTML&quot;, &quot;CSS&quot;, &quot;JavaScript&quot;};
        String[] names = {&quot;田中&quot;, &quot;佐藤&quot;, &quot;鈴木&quot;};

        System.out.println(&quot;=== 文字列の結合 ===&quot;);

        // スキルリストの結合
        String skillList = String.join(&quot;, &quot;, skills);
        System.out.println(&quot;保有スキル: &quot; + skillList);

        // 名前リストの結合（異なる区切り文字）
        String nameList = String.join(&quot; &amp; &quot;, names);
        System.out.println(&quot;メンバー: &quot; + nameList);

        // HTML形式で結合
        String htmlList = &quot;&lt;li&gt;&quot; + String.join(&quot;&lt;/li&gt;&lt;li&gt;&quot;, skills) + &quot;&lt;/li&gt;&quot;;
        System.out.println(&quot;HTML形式: &quot; + htmlList);
    }
}

    実行結果： 

    === 文字列の結合 ===
保有スキル: Java, HTML, CSS, JavaScript
メンバー: 田中 &amp; 佐藤 &amp; 鈴木
HTML形式: &lt;li&gt;Java&lt;/li&gt;&lt;li&gt;HTML&lt;/li&gt;&lt;li&gt;CSS&lt;/li&gt;&lt;li&gt;JavaScript&lt;/li&gt;

    ✨ ステップ6: 文字列のチェック機能

    文字列が 特定のパターンで始まる・終わる かチェックする方法を学びます。

    6-1: startsWith() / endsWith()

    public class StringCheck {
    public static void main(String[] args) {
        String[] files = {
            &quot;image_001.jpg&quot;,
            &quot;document.pdf&quot;, 
            &quot;image_002.png&quot;,
            &quot;report.txt&quot;,
            &quot;photo_family.jpg&quot;
        };

        System.out.println(&quot;=== ファイル分類システム ===&quot;);

        int imageCount = 0;
        int documentCount = 0;

        for (String file : files) {
            System.out.print(&quot;ファイル: &quot; + file + &quot; → &quot;);

            if (file.startsWith(&quot;image_&quot;)) {
                System.out.print(&quot;画像ファイル（連番）&quot;);
                imageCount++;
            } else if (file.endsWith(&quot;.jpg&quot;) || file.endsWith(&quot;.png&quot;)) {
                System.out.print(&quot;画像ファイル&quot;);
                imageCount++;
            } else if (file.endsWith(&quot;.pdf&quot;) || file.endsWith(&quot;.txt&quot;)) {
                System.out.print(&quot;文書ファイル&quot;);
                documentCount++;
            }
            System.out.println();
        }

        System.out.println(&quot;\n=== 集計結果 ===&quot;);
        System.out.println(&quot;画像ファイル: &quot; + imageCount + &quot;個&quot;);
        System.out.println(&quot;文書ファイル: &quot; + documentCount + &quot;個&quot;);
    }
}

    🎨 ステップ7: 美しい出力を作ろう

    文字列フォーマットを使って 見やすい出力 を作る方法を学びます。

    7-1: String.format() - 文字列フォーマット

    public class StringFormat {
    public static void main(String[] args) {
        // 商品情報
        String productName = &quot;ノートパソコン&quot;;
        double price = 89800.0;
        int stock = 5;
        double discountRate = 0.15;

        System.out.println(&quot;=== 商品情報表示 ===&quot;);

        // 基本的なフォーマット
        String info = String.format(&quot;商品名: %s&quot;, productName);
        System.out.println(info);

        // 数値のフォーマット
        String priceInfo = String.format(&quot;価格: ¥%,.0f&quot;, price);
        System.out.println(priceInfo);

        // 在庫数（右詰め）
        String stockInfo = String.format(&quot;在庫: %3d個&quot;, stock);
        System.out.println(stockInfo);

        // 割引率（パーセント表示）
        String discountInfo = String.format(&quot;割引率: %.0f%%&quot;, discountRate * 100);
        System.out.println(discountInfo);

        // 全て一行で表示
        String summary = String.format(&quot;%s - ¥%,.0f (%d個在庫)&quot;, 
                                     productName, price, stock);
        System.out.println(&quot;\n要約: &quot; + summary);
    }
}

    実行結果： 

    === 商品情報表示 ===
商品名: ノートパソコン
価格: ¥89,800
在庫: 　5個
割引率: 15%

要約: ノートパソコン - ¥89,800 (5個在庫)

    🚀 ステップ8: 実践統合 - 簡単なユーザー情報処理

    今まで学んだメソッドを組み合わせて、実用的なプログラム を作ってみましょう。

    public class UserInfoProcessor {
    public static void main(String[] args) {
        // ユーザー情報（実際にはユーザー入力から取得）
        String rawUserInfo = &quot;  tanaka.taro@example.com  &quot;;
        String rawPhoneNumber = &quot;090-1234-5678&quot;;
        String rawMessage = &quot;こんにちは！よろしくお願いします。&quot;;

        System.out.println(&quot;=== ユーザー情報処理システム ===&quot;);

        // 1. メールアドレスの処理
        String email = processEmail(rawUserInfo);
        System.out.println(&quot;処理後メール: &quot; + email);

        // 2. 電話番号の処理
        String phone = processPhoneNumber(rawPhoneNumber);
        System.out.println(&quot;処理後電話番号: &quot; + phone);

        // 3. メッセージの処理
        String message = processMessage(rawMessage);
        System.out.println(&quot;処理後メッセージ: &quot; + message);

        // 4. 統合表示
        displayUserSummary(email, phone, message);
    }

    // メールアドレス処理：空白削除、小文字変換、検証
    public static String processEmail(String rawEmail) {
        String email = rawEmail.trim().toLowerCase();

        if (email.contains(&quot;@&quot;) &amp;&amp; email.contains(&quot;.&quot;)) {
            return email;
        } else {
            return &quot;無効なメールアドレス&quot;;
        }
    }

    // 電話番号処理：ハイフン削除、形式チェック
    public static String processPhoneNumber(String rawPhone) {
        String phone = rawPhone.replace(&quot;-&quot;, &quot;&quot;);

        if (phone.length() == 11 &amp;&amp; phone.startsWith(&quot;0&quot;)) {
            return phone.substring(0, 3) + &quot;-&quot; + 
                   phone.substring(3, 7) + &quot;-&quot; + 
                   phone.substring(7);
        } else {
            return &quot;無効な電話番号&quot;;
        }
    }

    // メッセージ処理：長さチェック、要約作成
    public static String processMessage(String rawMessage) {
        if (rawMessage.length() &gt; 20) {
            return rawMessage.substring(0, 17) + &quot;...&quot;;
        } else {
            return rawMessage;
        }
    }

    // ユーザー情報の統合表示
    public static void displayUserSummary(String email, String phone, String message) {
        System.out.println(&quot;\n=== ユーザー情報サマリー ===&quot;);
        System.out.printf(&quot;Email   : %s%n&quot;, email);
        System.out.printf(&quot;Phone   : %s%n&quot;, phone);
        System.out.printf(&quot;Message : %s%n&quot;, message);
        System.out.println(&quot;========================&quot;);
    }
}

    🎯 まとめ

    このレッスンで学んだStringメソッド：

    基本取得メソッド

    • length() - 文字列の長さ
    • charAt(index) - 指定位置の文字

    検索メソッド

    • contains(文字列) - 含まれているかチェック
    • indexOf(文字列) - 位置を取得

    変換メソッド

    • substring(開始, 終了) - 部分文字列を取得
    • replace(元, 新) - 文字列を置換
    • toUpperCase(), toLowerCase() - 大文字小文字変換

    分割・結合メソッド

    • split(区切り文字) - 文字列を分割
    • String.join(区切り文字, 配列) - 文字列を結合

    チェックメソッド

    • startsWith(文字列) - 指定文字列で開始するか
    • endsWith(文字列) - 指定文字列で終了するか

    フォーマットメソッド

    • String.format(形式, 値...) - 書式指定した文字列作成
