    まずは基本の検索メソッドをマスター

    新しいクラス AdvancedArrayListDemo を作成してください：

    import java.util.ArrayList;
import java.util.Scanner;

public class AdvancedArrayListDemo {
    public static void main(String[] args) {
        System.out.println("=== ArrayListの高度な検索技術 ===");

        // サンプルデータの準備
        ArrayList<String> students = new ArrayList<>();
        students.add("田中太郎");
        students.add("佐藤花子");  
        students.add("山田一郎");
        students.add("鈴木美咲");
        students.add("田中次郎");
        students.add("高橋洋子");

        System.out.println("学生リスト: " + students);

        // 基本検索メソッドの活用
        demonstrateBasicSearch(students);

        // カスタム検索の実装
        demonstrateCustomSearch(students);

        // フィルタリングの実装
        demonstrateFiltering(students);
    }

    // 基本検索メソッドの活用
    public static void demonstrateBasicSearch(ArrayList<String> students) {
        System.out.println("\n【基本検索メソッド】");

        // indexOf: 最初に見つかった位置を返す
        String target = "田中太郎";
        int index = students.indexOf(target);
        System.out.println(target + "の位置: " + index);

        // lastIndexOf: 最後に見つかった位置を返す
        students.add("田中太郎"); // 重複追加
        int lastIndex = students.lastIndexOf(target);
        System.out.println(target + "の最後の位置: " + lastIndex);

        // contains: 存在するかどうかの確認
        boolean exists = students.contains("佐藤花子");
        System.out.println("佐藤花子は存在する: " + exists);

        boolean notExists = students.contains("伊藤");
        System.out.println("伊藤は存在する: " + notExists);

        // 検索結果を使った処理
        if (students.contains("山田一郎")) {
            System.out.println("山田一郎を発見！位置: " + students.indexOf("山田一郎"));
        }
    }

    // カスタム検索の実装
    public static void demonstrateCustomSearch(ArrayList<String> students) {
        System.out.println("\n【カスタム検索機能】");

        // 部分一致検索（姓での検索）
        String searchLastName = "田中";
        System.out.println("\n「" + searchLastName + "」で始まる学生:");

        for (int i = 0; i < students.size(); i++) {
            String student = students.get(i);
            if (student.startsWith(searchLastName)) {
                System.out.println("  " + (i + 1) + ". " + student + " (位置: " + i + ")");
            }
        }

        // 文字列の長さでの検索
        int targetLength = 4;
        System.out.println("\n名前が" + targetLength + "文字の学生:");

        for (String student : students) {
            if (student.length() == targetLength) {
                System.out.println("  " + student);
            }
        }

        // 複数条件での検索
        System.out.println("\n複数条件検索（名前に「子」を含み、4文字以上）:");

    private static void sortByAvailability() {
        Collections.sort(books, new Comparator<String>() {
            @Override
            public int compare(String s1, String s2) {
                boolean s1Available = !borrowedBooks.contains(s1);
                boolean s2Available = !borrowedBooks.contains(s2);

                // 在庫ありを先に
                if (s1Available && !s2Available) return -1;
                if (!s1Available && s2Available) return 1;

                // 同じ状態なら名前順
                return s1.compareTo(s2);
            }
        });

        System.out.println("✅ 在庫状況順にソートしました（在庫あり → 貸出中）。");
        showAllBooks();
    }

    💡 よくある質問

    Q1: Comparatorの書き方が難しいです

    A: 基本パターンを覚えましょう：

    // 昇順（小さい順）
Collections.sort(list, new Comparator<String>() {
    @Override
    public int compare(String s1, String s2) {
        return s1.compareTo(s2);  // または Integer.compare(s1.length(), s2.length())
    }
});

// 降順（大きい順）
Collections.sort(list, new Comparator<String>() {
    @Override
    public int compare(String s1, String s2) {
        return s2.compareTo(s1);  // または Integer.compare(s2.length(), s1.length())
    }
});

    Q2: 検索結果が期待通りにならないことがあります

    A: よくある間違いをチェックしてください：

    // ❌ 間違い：大文字小文字を考慮していない
if (student.contains("java")) { ... }

// ✅ 正解：大文字小文字を考慮
if (student.toLowerCase().contains("java")) { ... }

// ❌ 間違い：前後の空白を考慮していない
if (student.equals("田中 太郎")) { ... }

// ✅ 正解：空白を除去
if (student.trim().equals("田中太郎")) { ... }

    Q3: toArray()とArrays.asList()の使い分けは？

    A: - toArray(): ArrayList → 配列（完全に独立） - Arrays.asList(): 配列 → ArrayList（追加・削除は不可）

    実用的には新しいArrayListを作成： 

    String[] array = {"A", "B", "C"};
ArrayList<String> list = new ArrayList<>(Arrays.asList(array)); // これが安全
