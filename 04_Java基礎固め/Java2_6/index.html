    💡 なぜString メソッドが重要なのか？

    プログラミングにおいて、文字列処理 は避けて通れません：

    日常的な処理例： - ユーザーの入力チェック（メールアドレス形式など） - ファイル名の拡張子取得 - 検索機能の実装 - データの整形・表示

    String email = "tanaka@example.com";

// → ドメイン: example.com

    今回学ぶメソッドを使えば、こうした処理が簡単にできるようになります！

    📝 ステップ1: 文字列の基本情報を取得しよう

    まずは、文字列の 長さ や 特定の位置の文字 を取得する基本メソッドから始めましょう。

    1-1: length() - 文字列の長さを知る

    public class StringBasics1 {
    public static void main(String[] args) {
        // 好きな名前で試してみてください
        String name = "田中太郎";
        String nickname = "たろう";

        System.out.println("=== 文字列の長さチェック ===");
        System.out.println("名前: " + name);
        System.out.println("文字数: " + name.length() + "文字");

        System.out.println("ニックネーム: " + nickname);
        System.out.println("文字数: " + nickname.length() + "文字");

        // 長さによる判定
        if (name.length() > 5) {
            System.out.println("フルネームは長めですね");
        } else {
            System.out.println("フルネームは短めですね");
        }
    }
}

    実行結果： 

    === 文字列の長さチェック ===
名前: 田中太郎
文字数: 4文字
ニックネーム: たろう
文字数: 3文字
フルネームは短めですね

    1-2: charAt() - 特定の位置の文字を取得

    📘 文字列インデックスの概念図

    public class StringBasics2 {
    public static void main(String[] args) {
        String product = "iPhone15";

        System.out.println("=== 文字の位置チェック ===");
        System.out.println("商品名: " + product);
        System.out.println("最初の文字: " + product.charAt(0));
        System.out.println("最後の文字: " + product.charAt(product.length() - 1));

        // 一文字ずつ表示
        System.out.println("一文字ずつ表示:");
        for (int i = 0; i < product.length(); i++) {
            System.out.println(i + "番目: " + product.charAt(i));
        }
    }
}

    実行結果： 

    === 文字の位置チェック ===
商品名: iPhone15
最初の文字: i
最後の文字: 5
一文字ずつ表示:
0番目: i
1番目: P
2番目: h
3番目: o
4番目: n
5番目: e
6番目: 1
7番目: 5

    ポイント： - 文字の位置は 0から始まる（上の図解を参照） - 最後の文字は length() - 1 の位置にある - 範囲外の位置を指定するとエラーになるので注意

    🔍 ステップ2: 文字列を検索してみよう

    次は、文字列の中に 特定の文字や文字列が含まれているか を調べる方法を学びます。

    2-1: contains() - 含まれているかチェック

    public class StringSearch1 {
    public static void main(String[] args) {
        String review = "この商品はとても使いやすく、デザインも良いです";

        System.out.println("=== レビュー分析 ===");
        System.out.println("レビュー: " + review);

        // ポジティブワードをチェック
        if (review.contains("良い")) {
            System.out.println("✅ ポジティブな評価を発見!");
        }

        if (review.contains("使いやすく")) {
            System.out.println("✅ 使いやすさを評価!");
        }

        if (review.contains("悪い")) {
            System.out.println("❌ ネガティブな評価を発見");
        } else {
            System.out.println("✅ ネガティブワードは見つかりませんでした");
        }
    }
}

    2-2: indexOf() - 文字列の位置を調べる

    public class StringSearch2 {
    public static void main(String[] args) {
        String filename = "report_2024_final.pdf";

        System.out.println("=== ファイル名解析 ===");
        System.out.println("ファイル名: " + filename);

        // 拡張子の位置を探す
        int dotPosition = filename.indexOf(".");
        if (dotPosition != -1) {
            String extension = filename.substring(dotPosition + 1);
            System.out.println("拡張子: " + extension);
        }

        // 年度の位置を探す
        int yearPosition = filename.indexOf("2024");
        if (yearPosition != -1) {
            System.out.println("年度情報の位置: " + yearPosition + "文字目");
            System.out.println("このファイルは2024年のものです");
        }
    }
}

    実行結果： 

    === ファイル名解析 ===
ファイル名: report_2024_final.pdf
拡張子: pdf
年度情報の位置: 7文字目
このファイルは2024年のものです

    ✂️ ステップ3: 文字列を切り取ろう

    文字列の 一部分を取り出す 方法を学びます。

    3-1: substring() - 文字列の一部を取得

    📘 部分文字列の範囲図

    public class StringSubstring {
    public static void main(String[] args) {
        String phoneNumber = "090-1234-5678";
        String url = "https://www.example.com/products";

        System.out.println("=== 文字列の部分取得 ===");

        // 電話番号の分割
        System.out.println("電話番号: " + phoneNumber);
        String area = phoneNumber.substring(0, 3);        // 0〜2文字目
        String middle = phoneNumber.substring(4, 8);      // 4〜7文字目
        String last = phoneNumber.substring(9);           // 9文字目以降全て

        System.out.println("エリアコード: " + area);
        System.out.println("中間番号: " + middle);
        System.out.println("下4桁: " + last);

        // URLからドメイン名を取得
        System.out.println("\nURL: " + url);
        String domain = url.substring(8, url.indexOf("/", 8));  // https://の後から/まで
        System.out.println("ドメイン名: " + domain);
    }
}

    実行結果： 

    === 文字列の部分取得 ===
電話番号: 090-1234-5678

    重要ポイント： - substring(開始, 終了) - 終了位置は含まれません（図解参照） - substring(開始) - 開始位置から最後まで取得 - 範囲外を指定するとエラーになるので注意

    public class StringCase {

    📋 ステップ5: 文字列を分割・結合しよう

    5-1: split() - 文字列を分割

    public class StringSplit {
    public static void main(String[] args) {
        String fruits = "りんご,みかん,バナナ,ぶどう";
        String address = "東京都-渋谷区-恵比寿-1-2-3";

        // 果物リストの分割
        System.out.println("果物リスト: " + fruits);
        String[] fruitArray = fruits.split(",");
        System.out.println("分割結果:");
        for (int i = 0; i < fruitArray.length; i++) {
            System.out.println((i + 1) + ". " + fruitArray[i]);
        }

        // 住所の分割
        System.out.println("\n住所: " + address);
        String[] addressParts = address.split("-");
        System.out.println("都道府県: " + addressParts[0]);
        System.out.println("市区町村: " + addressParts[1]);
        System.out.println("町名: " + addressParts[2]);
    }
}

    5-2: String.join() - 文字列を結合

    public class StringJoin {
    public static void main(String[] args) {
        String[] skills = {"Java", "HTML", "CSS", "JavaScript"};
        String[] names = {"田中", "佐藤", "鈴木"};

        System.out.println("=== 文字列の結合 ===");

        // スキルリストの結合
        String skillList = String.join(", ", skills);
        System.out.println("保有スキル: " + skillList);

        // 名前リストの結合（異なる区切り文字）
        String nameList = String.join(" & ", names);
        System.out.println("メンバー: " + nameList);

        // HTML形式で結合
        String htmlList = "<li>" + String.join("</li><li>", skills) + "</li>";
        System.out.println("HTML形式: " + htmlList);
    }
}

    ✨ ステップ6: 文字列のチェック機能

    文字列が 特定のパターンで始まる・終わる かチェックする方法を学びます。

    6-1: startsWith() / endsWith()

    public class StringCheck {
    public static void main(String[] args) {
        String[] files = {
            "image_001.jpg",
            "document.pdf", 
            "image_002.png",
            "report.txt",
            "photo_family.jpg"
        };

        System.out.println("=== ファイル分類システム ===");

        int imageCount = 0;
        int documentCount = 0;

        for (String file : files) {
            System.out.print("ファイル: " + file + " → ");

            if (file.startsWith("image_")) {
                System.out.print("画像ファイル（連番）");
                imageCount++;
            } else if (file.endsWith(".jpg") || file.endsWith(".png")) {
                System.out.print("画像ファイル");
                imageCount++;
            } else if (file.endsWith(".pdf") || file.endsWith(".txt")) {
                System.out.print("文書ファイル");
                documentCount++;
            }
            System.out.println();
        }

        System.out.println("\n=== 集計結果 ===");
        System.out.println("画像ファイル: " + imageCount + "個");
        System.out.println("文書ファイル: " + documentCount + "個");
    }
}

    🎨 ステップ7: 美しい出力を作ろう

    文字列フォーマットを使って 見やすい出力 を作る方法を学びます。

    7-1: String.format() - 文字列フォーマット

    public class StringFormat {
    public static void main(String[] args) {
        // 商品情報
        String productName = "ノートパソコン";
        double price = 89800.0;
        int stock = 5;
        double discountRate = 0.15;

        System.out.println("=== 商品情報表示 ===");

        // 基本的なフォーマット
        String info = String.format("商品名: %s", productName);
        System.out.println(info);

        // 数値のフォーマット
        String priceInfo = String.format("価格: ¥%,.0f", price);
        System.out.println(priceInfo);

        // 在庫数（右詰め）
        String stockInfo = String.format("在庫: %3d個", stock);
        System.out.println(stockInfo);

        // 割引率（パーセント表示）
        String discountInfo = String.format("割引率: %.0f%%", discountRate * 100);
        System.out.println(discountInfo);

        // 全て一行で表示
        String summary = String.format("%s - ¥%,.0f (%d個在庫)", 
                                     productName, price, stock);
        System.out.println("\n要約: " + summary);
    }
}

    実行結果： 

    === 商品情報表示 ===

価格: ¥89,800
在庫: 　5個
割引率: 15%

要約: ノートパソコン - ¥89,800 (5個在庫)

    🚀 ステップ8: 実践統合 - 簡単なユーザー情報処理

    今まで学んだメソッドを組み合わせて、実用的なプログラム を作ってみましょう。

    public class UserInfoProcessor {
    public static void main(String[] args) {
        // ユーザー情報（実際にはユーザー入力から取得）
        String rawUserInfo = "  tanaka.taro@example.com  ";
        String rawPhoneNumber = "090-1234-5678";
        String rawMessage = "こんにちは！よろしくお願いします。";

        System.out.println("=== ユーザー情報処理システム ===");

        // 1. メールアドレスの処理
        String email = processEmail(rawUserInfo);
        System.out.println("処理後メール: " + email);

        // 2. 電話番号の処理
        String phone = processPhoneNumber(rawPhoneNumber);
        System.out.println("処理後電話番号: " + phone);

        // 3. メッセージの処理
        String message = processMessage(rawMessage);
        System.out.println("処理後メッセージ: " + message);

        // 4. 統合表示
        displayUserSummary(email, phone, message);
    }

    // メールアドレス処理：空白削除、小文字変換、検証
    public static String processEmail(String rawEmail) {
        String email = rawEmail.trim().toLowerCase();

        if (email.contains("@") && email.contains(".")) {
            return email;
        } else {
            return "無効なメールアドレス";
        }
    }

    // 電話番号処理：ハイフン削除、形式チェック
    public static String processPhoneNumber(String rawPhone) {
        String phone = rawPhone.replace("-", "");

        if (phone.length() == 11 && phone.startsWith("0")) {
            return phone.substring(0, 3) + "-" + 
                   phone.substring(3, 7) + "-" + 
                   phone.substring(7);
        } else {
            return "無効な電話番号";
        }
    }

    // メッセージ処理：長さチェック、要約作成
    public static String processMessage(String rawMessage) {
        if (rawMessage.length() > 20) {
            return rawMessage.substring(0, 17) + "...";
        } else {
            return rawMessage;
        }
    }

    // ユーザー情報の統合表示
    public static void displayUserSummary(String email, String phone, String message) {
        System.out.println("\n=== ユーザー情報サマリー ===");
        System.out.printf("Email   : %s%n", email);
        System.out.printf("Phone   : %s%n", phone);
        System.out.printf("Message : %s%n", message);
        System.out.println("========================");
    }
}

処理後電話番号: 090-1234-5678
処理後メッセージ: こんにちは！よろしくお願いします。

=== ユーザー情報サマリー ===
