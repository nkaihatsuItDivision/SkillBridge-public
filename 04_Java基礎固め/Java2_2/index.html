    💡 ArrayList とは：配列の問題を解決する理想的なデータ構造

    レッスン1で体験した配列の3つの制限を思い出しましょう

    レッスン1で、配列には以下の制限があることを実際に体験しました：

    1. サイズ固定の制限: 一度作成すると要素数を変更できない
    2. 挿入・削除の困難: 途中への追加や削除が複雑で非効率
    3. メモリ効率の問題: 使わない領域も確保し続ける

    ArrayList は、これらの問題を全て解決します！

    // 配列の制限をもう一度確認
int[] fixedArray = new int[5];  // サイズ固定！変更不可！

// ArrayListなら...
ArrayList<Integer> dynamicList = new ArrayList<>();  // サイズ自由自在！

    ArrayList の特徴: - 動的サイズ: 必要に応じて自動的にサイズが変更される - 簡単な操作: 追加・削除・挿入が1行のコードで可能 - メモリ効率: 使用する分だけメモリを効率的に管理

    📘 配列とArrayListの比較

    🔧 ArrayList を使うための準備

    1. インポート文の追加

    ArrayListを使うには、まずインポート文が必要です：

    import java.util.ArrayList;

    2. ArrayList の宣言方法

    // 基本的な宣言（ジェネリクス使用）
ArrayList<String> nameList = new ArrayList<>();      // 文字列のリスト
ArrayList<Integer> numberList = new ArrayList<>();   // 整数のリスト
ArrayList<Double> priceList = new ArrayList<>();     // 小数のリスト

    ジェネリクス（<>）について: - <String> : このArrayListには文字列しか入らない - <Integer> : このArrayListには整数しか入らない - 型安全性を保ち、エラーを防ぐ重要な仕組み

    📘 ArrayList操作の視覚化

    🚀 実践1: ArrayList 基本操作の完全マスター

    ステップ1: 新しいJavaプロジェクトの作成

    1. Eclipse で「File」→「New」→「Java Project」
    2. プロジェクト名: ArrayListBasics
    3. 「Finish」をクリック

    ステップ2: ArrayListDemoクラスの作成

    新しいクラス ArrayListDemo を作成し、以下のコードをコピペしてください：

    import java.util.ArrayList;

public class ArrayListDemo {
    public static void main(String[] args) {
        System.out.println("=== ArrayList 基本操作デモ ===\n");

        // 1. ArrayList の作成
        ArrayList<String> fruits = new ArrayList<>();
        System.out.println("1. 空のArrayListを作成しました");
        System.out.println("現在のサイズ: " + fruits.size());

        // 2. 要素の追加（add）
        fruits.add("りんご");
        fruits.add("バナナ");
        fruits.add("オレンジ");
        System.out.println("\n2. 要素を追加しました");
        System.out.println("現在のサイズ: " + fruits.size());
        System.out.println("内容: " + fruits);

        // 3. 特定位置への追加（insert）
        fruits.add(1, "いちご");  // インデックス1に挿入
        System.out.println("\n3. インデックス1に「いちご」を挿入");
        System.out.println("内容: " + fruits);

        // 4. 要素の取得（get）
        String firstFruit = fruits.get(0);
        String secondFruit = fruits.get(1);
        System.out.println("\n4. 要素の取得");
        System.out.println("最初の果物: " + firstFruit);
        System.out.println("2番目の果物: " + secondFruit);

        // 5. 要素の変更（set）
        fruits.set(2, "ぶどう");  // インデックス2を「ぶどう」に変更
        System.out.println("\n5. インデックス2を「ぶどう」に変更");
        System.out.println("内容: " + fruits);

        // 6. 要素の削除（remove）
        fruits.remove(0);  // インデックス0を削除
        System.out.println("\n6. インデックス0を削除");
        System.out.println("内容: " + fruits);

        // 7. 要素の検索（contains）
        boolean hasOrange = fruits.contains("オレンジ");
        System.out.println("\n7. 「オレンジ」が含まれているか: " + hasOrange);

        // 8. 空かどうかの確認（isEmpty）
        System.out.println("\n8. リストの状態確認");
        System.out.println("空かどうか: " + fruits.isEmpty());
        System.out.println("最終的なサイズ: " + fruits.size());
        System.out.println("最終的な内容: " + fruits);
    }
}

    配列の走査 vs ArrayList の走査

    まず、従来の方法と新しい方法を比較してみましょう。

    新しいクラス ArrayListLoop を作成し、以下のコードをコピペしてください：

    import java.util.ArrayList;

public class ArrayListLoop {
    public static void main(String[] args) {
        System.out.println("=== ArrayList の走査方法比較 ===\n");

        // ArrayListの準備
        ArrayList<String> colors = new ArrayList<>();
        colors.add("赤");
        colors.add("青");
        colors.add("緑");
        colors.add("黄");
        colors.add("紫");

        System.out.println("走査するArrayList: " + colors + "\n");

        // 方法1: 従来のfor文（配列と同じ方式）
        System.out.println("【方法1: 従来のfor文】");
        for (int i = 0; i < colors.size(); i++) {
            System.out.println((i + 1) + "番目: " + colors.get(i));
        }

        // 方法2: 拡張for文（for-each文）- おすすめ！
        System.out.println("\n【方法2: 拡張for文（おすすめ！）】");
        int count = 1;
        for (String color : colors) {
            System.out.println(count + "番目: " + color);
            count++;
        }

        // 方法3: インデックス不要の場合の拡張for文
        System.out.println("\n【方法3: シンプルな拡張for文】");
        for (String color : colors) {
            System.out.println("色: " + color);
        }

        // 実用例: 条件に合う要素の処理
        System.out.println("\n【実用例: 条件処理】");
        for (String color : colors) {
            if (color.contains("赤") || color.contains("青")) {
                System.out.println("原色発見: " + color);
            }
        }
    }
}

    レッスン1の問題を思い出しましょう

    レッスン1で作成したToDoリスト管理プログラムでは、配列の制限により以下の問題がありました：

    1. 固定サイズ: 最大10個までしか登録できない
    2. 削除の困難: 削除後の詰め処理が複雑
    3. 空き領域の無駄: nullで埋められた領域

    ArrayList版ToDoリストで全ての問題を解決！

    新しいクラス TodoListImproved を作成し、以下のコードをコピペしてください：

    import java.util.ArrayList;
import java.util.Scanner;

public class TodoListImproved {
    private static ArrayList<String> todoList = new ArrayList<>();
    private static Scanner scanner = new Scanner(System.in);

    public static void main(String[] args) {
        System.out.println("=== 改良版ToDoリスト管理システム ===");
        System.out.println("※ArrayListで配列の制限を完全解決！\n");

        while (true) {
            showMenu();
            int choice = scanner.nextInt();
            scanner.nextLine(); // 改行文字を消費

            switch (choice) {
                case 1:
                    addTodo();
                    break;
                case 2:
                    showAllTodos();
                    break;
                case 3:
                    completeTodo();
                    break;
                case 4:
                    showStats();
                    break;
                case 5:
                    System.out.println("ToDoリスト管理システムを終了します。");
                    return;
                default:
                    System.out.println("無効な選択です。1-5を選んでください。");
            }
            System.out.println(); // 空行
        }
    }

    private static void showMenu() {
        System.out.println("【メニュー】");
        System.out.println("1. ToDoを追加");
        System.out.println("2. ToDoリスト表示");
        System.out.println("3. ToDoを完了（削除）");
        System.out.println("4. 統計情報");
        System.out.println("5. 終了");
        System.out.print("選択してください (1-5): ");
    }

    private static void addTodo() {
        System.out.print("新しいToDoを入力してください: ");
        String newTodo = scanner.nextLine();

        if (!newTodo.trim().isEmpty()) {
            todoList.add(newTodo);
            System.out.println("✅ ToDo「" + newTodo + "」を追加しました！");
            System.out.println("現在のToDo数: " + todoList.size() + "個");
        } else {
            System.out.println("❌ 空のToDoは登録できません。");
        }
    }

    private static void showAllTodos() {
        if (todoList.isEmpty()) {
            System.out.println("📝 ToDoリストは空です。");
            return;
        }

        System.out.println("📋 現在のToDoリスト (" + todoList.size() + "個):");
        System.out.println("----------------------------------------");

        int index = 1;
        for (String todo : todoList) {  // 拡張for文で簡潔に！
            System.out.println(index + ". " + todo);
            index++;
        }
        System.out.println("----------------------------------------");
    }

    private static void completeTodo() {
        if (todoList.isEmpty()) {
            System.out.println("📝 完了するToDoがありません。");
            return;
        }

        showAllTodos();
        System.out.print("完了するToDoの番号を入力してください (1-" + todoList.size() + "): ");

        int index = scanner.nextInt();
        scanner.nextLine(); // 改行文字を消費

        if (index >= 1 && index <= todoList.size()) {
            String completedTodo = todoList.remove(index - 1);  // 1行で削除完了！
            System.out.println("🎉 ToDo「" + completedTodo + "」を完了しました！");
            System.out.println("残りのToDo数: " + todoList.size() + "個");
        } else {
            System.out.println("❌ 無効な番号です。");
        }
    }

    private static void showStats() {
        System.out.println("📊 ToDoリスト統計情報");
        System.out.println("----------------------------------------");
        System.out.println("現在のToDo数: " + todoList.size() + "個");
        System.out.println("リストが空か: " + (todoList.isEmpty() ? "はい" : "いいえ"));

        if (!todoList.isEmpty()) {
            System.out.println("\n📝 最新のToDo: " + todoList.get(todoList.size() - 1));
            System.out.println("🔝 最初のToDo: " + todoList.get(0));

            // 文字数統計
            int totalChars = 0;
            for (String todo : todoList) {
                totalChars += todo.length();
            }
            double avgChars = (double) totalChars / todoList.size();
            System.out.printf("📏 平均文字数: %.1f文字\n", avgChars);
        }
        System.out.println("----------------------------------------");
    }
}

    同じ処理を配列とArrayListで比較

    最後に、同じ処理を配列とArrayListの両方で実装し、違いを直接比較してみましょう。

    新しいクラス ArrayVsArrayListComparison を作成し、以下のコードをコピペしてください：

    import java.util.ArrayList;

public class ArrayVsArrayListComparison {
    public static void main(String[] args) {
        System.out.println("=== 配列 vs ArrayList 比較デモ ===\n");

        // === 配列版：学生名簿管理 ===
        System.out.println("【配列版】学生名簿管理");
        System.out.println("----------------------------------------");

        // 配列の宣言（固定サイズ）
        String[] studentArray = new String[5];
        int arrayCount = 0;

        // 学生追加（配列）
        studentArray[arrayCount++] = "田中";
        studentArray[arrayCount++] = "佐藤";
        studentArray[arrayCount++] = "鈴木";

        System.out.println("学生を3人追加しました");

        // 表示（配列）- nullチェックが必要
        System.out.print("配列の内容: ");
        for (int i = 0; i < studentArray.length; i++) {
            if (studentArray[i] != null) {
                System.out.print(studentArray[i] + " ");
            }
        }
        System.out.println();

        // 途中への挿入（配列）- 複雑な処理
        System.out.println("\n「高橋」をインデックス1に挿入:");
        for (int i = arrayCount; i > 1; i--) {
            studentArray[i] = studentArray[i - 1];
        }
        studentArray[1] = "高橋";
        arrayCount++;

        System.out.print("挿入後: ");
        for (int i = 0; i < arrayCount; i++) {
            System.out.print(studentArray[i] + " ");
        }
        System.out.println();

        // === ArrayList版：学生名簿管理 ===
        System.out.println("\n【ArrayList版】学生名簿管理");
        System.out.println("----------------------------------------");

        // ArrayListの宣言（動的サイズ）
        ArrayList<String> studentList = new ArrayList<>();

        // 学生追加（ArrayList）
        studentList.add("田中");
        studentList.add("佐藤");
        studentList.add("鈴木");

        System.out.println("学生を3人追加しました");

        // 表示（ArrayList）- 簡潔
        System.out.println("ArrayListの内容: " + studentList);

        // 途中への挿入（ArrayList）- 1行で完了
        System.out.println("\n「高橋」をインデックス1に挿入:");
        studentList.add(1, "高橋");

        System.out.println("挿入後: " + studentList);

        // === 機能比較 ===
        System.out.println("\n【機能比較】");
        System.out.println("----------------------------------------");

        System.out.println("配列のサイズ: " + studentArray.length + " (固定)");
        System.out.println("配列の実際の要素数: " + arrayCount + " (手動管理)");

        System.out.println("ArrayListのサイズ: " + studentList.size() + " (自動管理)");
        System.out.println("ArrayListの容量: 動的に変化");

        // === 削除の比較 ===
        System.out.println("\n【削除操作の比較】");
        System.out.println("----------------------------------------");

        // 配列での削除（複雑）
        System.out.println("配列でインデックス1を削除:");
        for (int i = 1; i < arrayCount - 1; i++) {
            studentArray[i] = studentArray[i + 1];
        }
        studentArray[arrayCount - 1] = null;
        arrayCount--;

        System.out.print("削除後（配列）: ");
        for (int i = 0; i < arrayCount; i++) {
            System.out.print(studentArray[i] + " ");
        }
        System.out.println();

        // ArrayListでの削除（簡単）
        System.out.println("\nArrayListでインデックス1を削除:");
        String removed = studentList.remove(1);
        System.out.println("削除された要素: " + removed);
        System.out.println("削除後（ArrayList）: " + studentList);

        // === まとめ ===
        System.out.println("\n【結論】");
        System.out.println("----------------------------------------");
        System.out.println("配列: 高速だが制限が多く、コードが複雑");
        System.out.println("ArrayList: 柔軟で簡単、実用的な選択");
    }
}
