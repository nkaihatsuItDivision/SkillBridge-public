    1. 基本構造の準備（Week 1復習）

    🛠️ ステップ1: HTMLファイルの作成

    VSCodeで新しいファイル contact-form-final.html を作成しましょう。

    まずは、これまで学んだ基本構造から始めます：

    <!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>お問い合わせフォーム - 完成版</title>
</head>
<body>
    <h1>お問い合わせフォーム</h1>

    <form method="POST">
        <!-- ここにフォーム要素を追加していきます -->
    </form>
</body>
</html>

    ✅ 確認： ファイルを保存してLive Serverで開き、タイトルが表示されることを確認してください。

    🔄 振り返りポイント

    • <!DOCTYPE html> でHTML5文書を宣言
    • lang="ja" で日本語文書を指定
    • viewport でレスポンシブ対応
    • <form method="POST"> でフォームの基本設定

    2. 入力要素の追加（レッスン1・2の技術）

    📝 ステップ2: 基本的な入力項目を追加

    <!-- お名前 -->
<label for="name">お名前</label>
<input type="text" id="name" name="name" placeholder="例：山田太郎">

<br><br>

<!-- メールアドレス -->
<label for="email">メールアドレス</label>
<input type="email" id="email" name="email" placeholder="例：yamada@example.com">

    ✅ ステップ5: より詳細なバリデーション

    さらに詳細なバリデーションを追加しましょう：

    <!-- フリガナにカタカナパターンを追加 -->
<input type="text" id="kana" name="kana" placeholder="例：ヤマダタロウ" 
       pattern="[ァ-ヴー]+" title="カタカナで入力してください">

<!-- 電話番号にパターンを追加 -->
<input type="tel" id="phone" name="phone" placeholder="例：090-1234-5678" 
       pattern="[0-9]{2,4}-[0-9]{2,4}-[0-9]{3,4}" title="ハイフン区切りで入力してください">

<!-- 年齢に範囲を追加 -->
<input type="number" id="age" name="age" placeholder="例：25" 
       min="1" max="120" required>

<!-- 名前に文字数制限を追加 -->
<input type="text" id="name" name="name" placeholder="例：山田太郎" 
       maxlength="50" required>

<!-- メールアドレスに文字数制限を追加 -->
<input type="email" id="email" name="email" placeholder="例：yamada@example.com" 
       maxlength="100" required>

<!-- お問い合わせ内容に文字数制限を追加 -->
<textarea id="message" name="message" placeholder="お問い合わせ内容を詳しくお書きください（500文字以内）" 
          maxlength="500" required></textarea>

    📧 ステップ10: Formspree.io設定

    📘 Formspree導入ガイド

    実際にメールを送信できるようにしましょう。

    10-1. Formspree.ioアカウント作成

    1. Formspree.io(https://formspree.io/) にアクセス
    2. 「Get Started」をクリック
    3. メールアドレスとパスワードを入力してアカウント作成
    4. メール認証を完了
    5. アンケートがいくつかある為回答する

    10-2. フォーム作成

    1. ダッシュボードで「New Form」をクリック

    3. 通知先メールアドレスを入力
    4. 「Create Form」をクリック

    10-3. エンドポイントURLの取得

    作成後に表示されるエンドポイントURLをコピーします：

    https://formspree.io/f/YOUR_FORM_ID
