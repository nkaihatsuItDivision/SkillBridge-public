    2.6 HTMLの基本構造を理解する

    text HTMLファイルの正しい構造と必須要素を理解させる。初心者が混乱しやすいDOCTYPE、html、head、bodyの役割を明確に区別し、実際にファイルを作成しながら体験的に学習する。 6.HTMLの基本構造を理解する # HTMLの基本構造を理解する

    🎯 このレッスンで学ぶこと

    • HTML文書の正しい基本構造

    1. HTMLファイルの「決められた形」

    前回のレッスンで簡単なHTMLファイルを作成しましたが、実はHTMLには決められた基本構造があります。

    なぜ構造が重要なのか？

    • ブラウザが正しく理解できるようになる
    • 他の開発者が読みやすくなる
    • 検索エンジンが理解しやすくなる
    • エラーを防ぐことができる

    HTMLの基本構造（テンプレート）

    <!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <title>ページのタイトル</title>
</head>
<body>
    <!-- ここにページの内容を書く --></body>
</html>

    この形を必ず守ることで、どのブラウザでも正しく表示されるHTMLファイルを作ることができます。

    2. 各要素の役割を詳しく理解しよう

    🏷️ DOCTYPE宣言

    <!DOCTYPE html>

    役割：「このファイルはHTMLで書かれています」とブラウザに宣言する

    重要なポイント： - 必ず1行目に書く - 大文字・小文字どちらでもOK（<!doctype html>でも可） - この1行がないと、ブラウザが古いモードで表示してしまう場合がある

    例え：お店の看板のようなもの。「ここはラーメン屋です」と最初に伝える

    🌐 htmlタグ

    <html lang="ja">

    役割：HTMLファイル全体を囲む「容器」の役割

    lang="ja"の意味： - lang = language（言語） - ja = 日本語 - 英語の場合はlang="en"

    重要なポイント： - すべてのHTML要素の親になる - 言語を指定することで、検索エンジンや音声読み上げソフトが正しく認識する - 必ず閉じタグ </html> が必要

    🧠 headタグ

    <head>
    <meta charset="UTF-8">
    <title>ページのタイトル</title>
</head>

    役割：ページの「設定情報」を書く場所

    の中に書くもの： - ページタイトル（ブラウザのタブに表示される） - 文字コードの設定 - CSSの読み込み - その他の設定情報

    重要なポイント： - 画面には表示されない - ブラウザや検索エンジン向けの情報

    📌 meta charsetタグ

    <meta charset="UTF-8">
    役割：「このファイルは日本語で書かれています」と文字コードを指定

    これがないと： - 日本語が「文字化け」してしまう - 必ずの最初に書く

    🏷️ titleタグ

    <title>ここにタイトルを書く</title>
    役割：ページのタイトルを設定

    表示される場所： - ブラウザのタブ - 検索結果のタイトル - お気に入りの名前

    👤 bodyタグ

    <body>
    <h1>見出し</h1>
    <p>ここに本文を書きます</p>
</body>

    役割：実際に画面に表示される内容を書く場所

    の中に書くもの： - 見出し（h1、h2など） - 段落（p） - 画像（img） - リンク（a） - その他すべての表示内容

    ※上記のタグの詳細は後のレッスンで学習します

    3. 正しい構造と間違った構造の比較

    ✅ 正しい構造

    <!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <title>正しいHTMLページ</title>
</head>
<body>
    <h1>こんにちは</h1>
    <p>これは正しく構造化されたHTMLです。</p>
</body>
</html>

    ❌ 間違った構造の例

    <!-- DOCTYPEがない --><html>
<head>
<!-- charsetがない --><title>間違ったHTMLページ</title>
</head>
<!-- bodyタグがない --><h1>こんにちは</h1>
<p>これは間違った構造です。</p>
</html>

    何が問題？ - DOCTYPEがないと古いモードで表示される可能性 - charsetがないと日本語が文字化けする可能性 - bodyタグがないと構造が正しくない

    4. 実践課題

    課題1：正しい構造のHTMLファイルを作成しよう

    手順： 1. メモ帳またはテキストエディタを開く 2. 以下の内容を正確に入力する
    ※練習としてコピー&ペーストではなく実際に打つようにしてください

    <!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <title>私の自己紹介ページ</title>
</head>
<body>
    <h1>自己紹介</h1>
    <p>はじめまして！私の名前は◯◯です。</p>
    <p>趣味は◯◯で、好きな食べ物は◯◯です。</p>
</body>
</html>
    1. ファイル名を self-introduction.html で保存
    2. ダブルクリックしてブラウザで開く
    3. ブラウザのタブを確認する

    課題2：構造の確認をしよう

    確認項目： - [ ] ブラウザのタブに「私の自己紹介ページ」と表示されているか？ - [ ] 「自己紹介」が大きな見出しで表示されているか？ - [ ] 2つの段落が正しく表示されているか？ - [ ] 日本語が文字化けしていないか？

    ポイント： - <title>の内容がブラウザのタブに表示される - <h1>が大きな見出しとして表示される - <p>が段落として表示される - <meta charset="UTF-8">があるので日本語が正しく表示される

    5. よくある間違いと対処法

    間違い1：閉じタグを忘れる

    <!-- 間違い --><h1>見出し
<p>段落

<!-- 正しい --><h1>見出し</h1>
<p>段落</p>

    間違い2：タグの順序が間違っている

    <!-- 間違い --><body>
<head>
    <title>タイトル</title>
</head>
</body>

<!-- 正しい --><head>
    <title>タイトル</title>
</head>
<body>
</body>

    間違い3：文字コードの記述漏れ

    <!-- 間違い --><head>
    <title>タイトル</title>
</head>

<!-- 正しい --><head>
    <meta charset="UTF-8">
    <title>タイトル</title>
</head>

    6. HTMLタグの基本ルール

    ルール1：開始タグと終了タグは必ずセット

    <h1>見出し</h1>
<p>段落</p>

    ルール2：タグは正しい順序で閉じる

    <!-- 正しい --><h1><strong>重要な見出し</strong></h1>

<!-- 間違い --><h1><strong>重要な見出し</h1></strong>

    ルール3：必須要素は省略しない

    • <!DOCTYPE html>
    • <html>
    • <head>
    • <meta charset="UTF-8">
    • <title>
    • <body>

    7. まとめ

    学んだこと

    ✅ HTMLの基本構造：DOCTYPE、html、head、bodyの4つの基本要素
    ✅ ：ページの設定情報（表示されない）
    ✅ ：ページの内容（表示される）
    ✅ 必須要素：文字コード、タイトルは必ず設定する
    ✅ 正しい構造：決められた順序と形を守ることの重要性

    重要なポイント

    • テンプレートをコピペから始める：毎回基本構造を手入力せず、正しいテンプレートをコピーして使いましょう
    • タグを書いたらすぐに閉じタグも書く：<h1>と入力したら、内容を書く前に</h1>も書いてしまうとミスが防げます
    • 文字化けを防ぐためcharsetは必須：日本語サイトでは<meta charset="UTF-8">を忘れずに含めましょう
