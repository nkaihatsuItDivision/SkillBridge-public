    2.13 実践：シンプルな自己紹介ページを作成

    text レッスン1-7で学んだHTML/CSSの知識を総合して、完成度の高い自己紹介ページを一から作成する。段階的な手順とヒントにより、初心者でも美しく機能的なWebページを完成させ、Web開発の達成感と自信を得る総復習レッスン。 13.実践：シンプルな自己紹介ページを作成 # 実践：シンプルな自己紹介ページを作成

    🎯 このレッスンで学ぶこと

    • これまで学んだHTML/CSS技術の総合的な活用
    • プロフェッショナルな自己紹介ページの設計と実装
    • レイアウト設計から完成まで一貫した開発体験
    • 基礎技術のみで美しいWebページを作る技術
    • 自分だけのオリジナルWebページの完成

    1. これまでの学習内容の振り返り

    おめでとうございます！レッスン1から12を通じて、Web開発の基礎をしっかりと学習してきました。

    🔍 習得した技術一覧

    📝 HTML技術： - 基本構造：DOCTYPE、html、head、body - 見出し・段落：h1-h6、p、br - リスト・リンク：ul、ol、li、a - 画像・表：img、table、tr、th、td - 構造化：意味のあるマークアップ

    🎨 CSS技術： - 基本スタイル：color、font-size、background-color - レイアウト：margin、padding、border - 配置方法：text-align、margin: 0 auto - ボックスモデル：width、height、box-sizing - セレクタ：要素、クラス、IDセレクタ

    🎯 今回の目標

    これらの技術をすべて組み合わせて、プロフェッショナルな自己紹介ページを作成します。

    2. 作成するページの全体設計

    完成イメージ

    作成するページの構成： 1. ヘッダー：名前とキャッチフレーズ 2. プロフィール写真エリア：写真と基本情報 3. スキル・経験セクション：学習内容やスキル 4. 趣味・興味セクション：個人的な内容 5. 連絡先セクション：連絡方法 6. フッター：著作権表示

    デザインの特徴

    • 清潔で読みやすいレイアウト
    • 適切な余白で情報を整理
    • 色の統一感でプロフェッショナルな印象
    • レスポンシブ風の中央配置デザイン

    3. 段階的な作成手順

    ステップ1：プロジェクトの準備

    📁 ファイル構成： 

    my-profile/
├── index.html          # メインのHTMLファイル
├── images/            
│   └── profile.jpg     # プロフィール写真（任意）
└── README.md          # プロジェクト説明（任意）

    🔧 基本構造から開始： 

    <!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <title>あなたの名前 - プロフィール</title>
    <style>
        /* ここにCSSを書いていきます */
    </style>
</head>
<body>
    <!-- ここにコンテンツを書いていきます -->
</body>
</html>

    ステップ2：全体のスタイル設定

    💡 最初に設定すべきCSS： 

    /* 全体の基本設定 */
* {
    box-sizing: border-box;        /* サイズ計算を簡単に */
}

body {
    margin: 0;                     /* ブラウザデフォルトの余白削除 */
    font-family: "游ゴシック", Arial, sans-serif;
    line-height: 1.6;              /* 読みやすい行間 */
    color: #333333;                /* 基本の文字色 */
    background-color: #f8f9fa;     /* 薄いグレー背景 */
}

/* メインコンテナ */
.container {
    max-width: 900px;              /* 最大幅を制限 */
    margin: 0 auto;                /* 中央配置 */
    background-color: white;       /* 白い背景 */
    min-height: 100vh;             /* 画面の高さいっぱい */
}

    6. 作成のコツとベストプラクティス

    📋 開発の進め方

    1. 段階的な開発： 

    1. HTML構造を全て作成
2. 基本的なCSS（全体設定）を適用
3. セクションごとにスタイルを追加
4. 細かい調整と最終確認

    2. デバッグのコツ： - 一つずつ確認：セクションごとにブラウザで確認 - コメントアウト：問題のある部分を特定 - 開発者ツール：F12でスタイルを確認

    ⚠️ よくある問題と解決方法

    問題1：レイアウトが崩れる 

    /* 解決策：box-sizingを必ず設定 */
* {
    box-sizing: border-box;
}

    問題2：画像が表示されない 

    <!-- パスを確認 -->
<img src="images/profile.jpg" alt="プロフィール写真">

<!-- または仮の画像を使用 -->
<img src="https://via.placeholder.com/150" alt="プロフィール写真">

    問題3：色が意図通りにならない 

    /* より具体的なセレクタを使用 */
.section-container .section-title {
    color: #2c5282;
}

    7. 発展課題（外部スタイルシートの活用）

    💪 余裕のある人への挑戦

    完成したページをさらにプロフェッショナルにするため、外部スタイルシートを活用してみましょう。

    手順： 1. style.css ファイルを新規作成 2. HTMLから <style> 部分をコピーして style.css に移動 3. HTMLの <head> に <link rel="stylesheet" href="style.css"> を追加 4. Live Serverで正しく表示されるか確認

    詳しい手順は レッスン10「CSSの基本とスタイルの適用方法」 を参照してください。
