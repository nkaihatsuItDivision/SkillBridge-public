    レッスン1

    text PC初心者向けの基本操作習得レッスン。マウス・キーボード・ウィンドウ操作・ショートカットキーの完全習熟を目標とする。学習者の不安を解消し、PC操作に対する自信を獲得させることが最優先。 1.PC基本操作 # 0.1 PC基本操作

    🎯 このレッスンのゴール

    「PCを怖がらずに、基本操作を自信を持って行えるようになる」

    このレッスンを終えると、以下ができるようになります： - マウスを使って思い通りに操作できる - キーボードで文字・数字・記号を正確に入力できる - ウィンドウを自由に操作できる - 便利なショートカットキーを使える - PC操作に対する不安が解消される

    📚 レッスンの構成

    このレッスンは4つのセクションで構成されています：

    1. マウス操作の完全習熟 ← まずはここから！
    2. キーボード操作の完全習熟
    3. ウィンドウ操作の完全習熟
    4. 便利なショートカットキーの習得

    重要: 焦らず一つずつ確実に身につけましょう。「できない」ことは恥ずかしいことではありません。誰もが最初は初心者です！

    🖱️ セクション1: マウス操作の完全習熟

    マウスを正しく持ちましょう

    📘​ マウスの持ち方ガイド

    基本操作1: 左クリック（一番大切！）

    左クリックは最も基本的で、最も頻繁に使う操作です。

    何に使う？ - ボタンを押す - アイコンを選ぶ - テキストボックスにカーソルを置く

    練習してみましょう：

    1. デスクトップの空いている場所で左クリックしてみてください
    2. タスクバーの「スタート」ボタンを左クリックしてみてください
    3. 開いたメニューの外側を左クリックしてメニューを閉じてみてください

    📘​ 左クリック練習

    💡 コツ: 「カチッ」と軽く押すだけで十分です。強く押す必要はありません！

    基本操作2: 右クリック（便利メニューが出る！）

    右クリックをすると、その場所に応じた便利なメニューが表示されます。

    練習してみましょう：

    1. デスクトップの空いている場所で右クリックしてみてください
    2. どんなメニューが出ましたか？
    3. メニューの外側を左クリックしてメニューを閉じてください

    📘​ 右クリックメニュー

    💡 覚えておこう: 困ったときは「右クリック」！色々なメニューが出てきます。

    基本操作3: ダブルクリック（アプリを開く！）

    ダブルクリックは、左クリックを素早く2回行うことです。主にアプリケーションを起動するときに使います。

    練習してみましょう：

    1. デスクトップにある「ごみ箱」アイコンをダブルクリックしてみてください
    2. ごみ箱のウィンドウが開きましたか？

    📘​ ダブルクリック練習

    💡 コツ: - 2回のクリックの間隔は約0.5秒以内 - マウスを動かさずに、同じ場所をクリック - うまくいかないときは、もう一度ゆっくり試してみましょう

    よくある質問: ダブルクリックがうまくできません... 答え: 大丈夫です！最初は難しく感じますが、練習すれば必ずできるようになります。焦らず何度でも試してください。

    基本操作4: ドラッグ&ドロップ（物を移動！）

    ドラッグ&ドロップは、何かを「つかんで」「移動させて」「離す」操作です。

    手順: 1. 移動したいものの上で左クリックしたまま押し続ける（つかむ） 2. マウスを移動させる（運ぶ） 3. 目的の場所で左クリックを離す（離す）

    練習してみましょう：

    1. デスクトップに新しいフォルダを作ってみましょう：
    2. デスクトップの空いている場所で右クリック
    3. 「新規作成」→「フォルダー」を左クリック
    4. 「新しいフォルダー」ができましたね！

    📘​ フォルダーへのドラッグ＆ドロップ

    1. このフォルダをドラッグ&ドロップで移動してみましょう：
    2. フォルダの上で左クリックしたまま押し続ける
    3. マウスを別の場所に移動
    4. 左クリックを離す

    💡 コツ: 「つかんで、運んで、離す」のイメージで！実際に物を手で運ぶのと同じです。

    基本操作5: スクロール（ページを上下に動かす！）

    スクロールは、長いページや文書を上下に動かす操作です。

    方法は2つあります:

    方法1: マウスホイール（推奨） - マウスの真ん中にあるホイール（車輪のような部分）を回す - 手前に回す：上にスクロール - 奥に回す：下にスクロール

    方法2: スクロールバー - ウィンドウの右端にある縦の棒（スクロールバー）をドラッグ

    練習してみましょう：

    1. ブラウザ（インターネットを見るアプリ）を開いてみましょう：

    2. タスクバーにあるブラウザのアイコンをダブルクリック

    3. 適当なウェブサイトでスクロールを練習してみましょう：

    4. マウスホイールを回してみてください
    5. スクロールバーをドラッグしてみてください

    ​📘 スクロール操作

    💡 覚えておこう: 長い文章を読むときはスクロールが必須です！

    🎯 マウス操作チェックリスト

    以下の操作ができるようになったら、次のセクションに進みましょう：

    • [ ] 左クリック: ボタンやアイコンを選択できる
    • [ ] 右クリック: メニューを表示できる
    • [ ] ダブルクリック: アプリケーションを起動できる
    • [ ] ドラッグ&ドロップ: ファイルやフォルダを移動できる
    • [ ] スクロール: ページを上下に移動できる

    すべてチェックできましたか？ 素晴らしいです！マウス操作はばっちりですね！

    ⌨️ セクション2: キーボード操作の完全習熟

    キーボードの配置を確認しましょう

    📘​ キーボード配列

    キーボードは大きく分けて以下の領域があります： - 文字キー領域: アルファベット・ひらがなを入力 - 数字キー領域: 数字を入力 - 機能キー領域: 特別な機能を実行 - 矢印キー領域: カーソルを移動

    基本操作1: 文字入力の練習

    ひらがな入力

    メモ帳を開いて練習しましょう：

    1. スタートメニューを開く（Windowsキー または 左下のスタートボタンをクリック）
    2. 「メモ帳」と入力して検索
    3. 「メモ帳」アプリをクリックして起動

    📘​ メモ帳起動

    練習文を入力してみましょう：

    こんにちは
わたしのなまえは
がっこう

    カタカナ入力

    同じメモ帳に以下を入力してみましょう：

    コンピュータ
キーボード  
マウス

    💡 ひらがな↔カタカナの切り替え: - F6キー: ひらがなに変換 - F7キー: カタカナに変換 - F8キー: 半角カタカナに変換

    アルファベット入力

    アルファベットの練習もしましょう：

    computer
keyboard
mouse
programming

    💡 ひらがな↔英数字の切り替え: - 半角/全角キー（通常、数字の1の左にあります） - または Alt + 半角/全角キー

    基本操作2: 数字・記号の入力

    数字の入力

    以下の数字を入力してみましょう：

    12345
67890
2025

    よく使う記号の入力

    以下の記号を入力してみましょう：

    記号 読み方 キーの組み合わせ
    @ アットマーク Shift + 2
    # シャープ Shift + 3
    $ ドル Shift + 4
    % パーセント Shift + 5
    & アンド Shift + 6
    * アスタリスク Shift + 8
    ( 左括弧 Shift + 9
    ) 右括弧 Shift + 0

    練習してみましょう：

    user@example.com
100%
2024年8月25日
(注意)

    基本操作3: 重要な機能キー

    Enter（エンター）キー: 確定・実行 - 文字入力の確定 - 検索の実行 - 改行

    BackSpace（バックスペース）キー: 削除 - カーソルの左側の文字を1文字削除

    Delete（デリート）キー: 削除 - カーソルの右側の文字を1文字削除

    Space（スペース）キー: 空白 - 文字と文字の間に空白を入れる - 変換候補の切り替え

    練習してみましょう：

    1. メモ帳に「こんにちわ」と入力してください
    2. BackSpaceキーで「わ」を削除してください
    3. 「は」と入力して「こんにちは」にしてください
    4. Enterキーで改行してください

    基本操作4: タイピングの基本姿勢

    正しい指の配置

    📘​ ホームポジション入力

    基本ポジション（ホームポジション）: - 左手：A S D F キーに人差し指から小指を置く - 右手：J K L ; キーに人差し指から小指を置く - 両手の親指：スペースキーの上

    練習のコツ: 1. 正しい姿勢で座る: 背筋を伸ばして 2. 手首を浮かせる: 机につけない 3. 軽いタッチ: 強く叩かない 4. 見ないで打つ練習: 最初は遅くても大丈夫

    🎯 キーボード操作チェックリスト

    以下の操作ができるようになったら、次のセクションに進みましょう：

    • [ ] ひらがな入力: スムーズに日本語が入力できる
    • [ ] カタカナ入力: F6/F7キーで変換できる
    • [ ] アルファベット入力: 英数字が入力できる
    • [ ] 数字入力: 正確に数字が入力できる
    • [ ] 記号入力: @や%などの基本記号が入力できる
    • [ ] 機能キー: Enter、BackSpace、Deleteが使える
    • [ ] 正しい姿勢: ホームポジションを意識して入力できる

    すべてチェックできましたか？ 素晴らしいです！キーボード操作もマスターですね！

    🖥️ セクション3: ウィンドウ操作の完全習熟

    ウィンドウとは？

    ウィンドウとは、アプリケーション（ソフト）が動作している「画面上の四角い領域」のことです。

    例えば： - メモ帳のウィンドウ - ブラウザ（インターネット）のウィンドウ
    - ファイルを表示するエクスプローラーのウィンドウ

    📘​ ウィンドウとは

    基本操作1: ウィンドウの開き方・閉じ方

    ウィンドウを開く（アプリケーションを起動する）

    方法1: デスクトップアイコンから - デスクトップのアイコンをダブルクリック

    方法2: タスクバーから
    - タスクバーのアイコンを左クリック

    方法3: スタートメニューから 1. スタートボタンをクリック（またはWindowsキーを押す） 2. アプリ名を入力して検索 3. 見つかったアプリをクリック

    ウィンドウを閉じる

    方法1: ×ボタン（推奨） - ウィンドウの右上角にある× ボタンをクリック

    方法2: キーボード - Alt + F4 キーを同時押し

    練習してみましょう：

    基本操作2: ウィンドウサイズの変更

    最大化: ウィンドウを画面全体に広げる - ウィンドウのタイトルバーをダブルクリック - または ウィンドウ右上の□ボタンをクリック

    最小化: ウィンドウを隠す（閉じるのではない） - ウィンドウ右上の－ボタンをクリック - タスクバーのアイコンをクリックすると再表示

    元のサイズに戻す: 最大化を解除する - 最大化されたウィンドウのタイトルバーをダブルクリック - または 右上の元に戻すボタン（□が2つ重なったアイコン）をクリック

    手動でサイズ変更: 1. ウィンドウの端（境界線）にマウスカーソルを合わせる 2. マウスカーソルが矢印の形に変わったら 3. ドラッグ&ドロップでサイズを変更

    ⚡ セクション4: 便利なショートカットキーの習得

    ショートカットキーとは？

    ショートカットキーとは、キーボードのキーを組み合わせて押すことで、マウス操作を省略できる便利な機能です。

    メリット: - 時間短縮: マウスより速い - 効率アップ: 作業がスムーズになる - プロっぽい: できる人に見える！

    必須ショートカットキー TOP5

    覚えておくと必ず役立つ、厳選5つをご紹介します！

    1. Ctrl + C（コピー）

    何ができる？: 選択したテキストや画像を「コピー」する

    使い方: 1. コピーしたいテキストを選択（ドラッグして青くする） 2. Ctrl + C を同時押し 3. 「コピーされました」という状態になる

    練習してみましょう：

    1. メモ帳を開いてください
    2. 「プログラミング学習中」と入力してください
    3. この文字を全選択してください（Ctrl + A または ドラッグで選択）
    4. Ctrl + C でコピーしてください

    📘​ CTRL+C でコピー

    2. Ctrl + V（貼り付け）

    何ができる？: コピーしたものを「貼り付け」する

    使い方: 1. 貼り付けたい場所にカーソルを置く 2. Ctrl + V を同時押し 3. コピーした内容が貼り付けられる

    練習してみましょう：

    1. 上でコピーした「プログラミング学習中」を使います
    2. メモ帳の新しい行にカーソルを置いてください
    3. Ctrl + V で貼り付けてください
    4. 同じ文字が2行になりましたか？

    3. Ctrl + Z（元に戻す）

    何ができる？: 直前の操作を「なかったこと」にする

    使い方: 1. 間違いに気づいたら 2. Ctrl + Z を同時押し 3. 1つ前の状態に戻る

    練習してみましょう：

    1. メモ帳に何か文字を追加で入力してください
    2. Ctrl + Z を押してください
    3. 入力した文字が消えて、前の状態に戻りましたか？

    💡 覚え方: - Copy（コピー） - V（Ctrl+Cの隣だから貼り付け） - Z（最後のアルファベット → 最後を取り消し）

    4. Ctrl + A（全選択）

    何ができる？: ページやファイルの内容を「すべて選択」する

    使い方: 1. 全部選択したい画面で 2. Ctrl + A を同時押し 3. すべての内容が選択される（通常、青くなる）

    練習してみましょう：

    1. メモ帳に複数行の文字が入力されている状態で
    2. Ctrl + A を押してください
    3. すべての文字が青く選択されましたか？

    5. Alt + Tab（ウィンドウ切り替え）

    何ができる？: 開いているアプリケーションを素早く切り替え

    使い方: 1. Alt + Tab を同時押し 2. Tabを押しながら切り替えたいアプリを選択 3. 決まったらキーを離す

    練習してみましょう：

    1. メモ帳と電卓を両方開いてください
    2. Alt + Tab でアプリを切り替えてみてください
    3. 何度か練習してスムーズにできるようになりましょう
