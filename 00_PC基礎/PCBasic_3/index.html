    📚 セクション1: インターネットとブラウザって何？

    🌍 インターネットとは - 世界最大の情報の海

    インターネットとは、世界中のコンピュータがつながった巨大なネットワークです。

    📘 インターネットの仕組み図

    分かりやすい例え

    • インターネット = 世界中をつなぐ「道路網」
    • ウェブサイト = 道路沿いにある「お店」
    • ブラウザ = お店を巡るための「車」
    • URL（アドレス） = お店の「住所」

    🏪 ウェブサイト・ホームページとは

    ウェブサイト（ホームページ）は、インターネット上の「情報を提供する場所」です。

    身近な例

    • Googleの検索サイト → 情報を探すお手伝いをするお店
    • Amazonのサイト → 商品を販売するお店
    • ニュースサイト → 最新情報を教えてくれるお店
    • Wikipedia → 百科事典を提供する図書館

    🔗 URL（アドレス）の仕組み

    URLは、ウェブサイトの「住所」です。

    URLの構造例: https://www.google.com

    📘​ URL構造図

    • https:// → 安全な通信方式（後で詳しく説明）
    • www → インターネット上の場所を示す
    • google → サイトの名前
    • com → サイトの種類

    🌐 ブラウザとは - インターネットを見るためのアプリ

    ブラウザは、インターネット上のウェブサイトを見るためのアプリケーション（ソフトウェア）です。

    主要なブラウザ

    • Google Chrome → 世界で最も使われているブラウザ
    • Microsoft Edge → Windowsに標準でインストールされているブラウザ
    • Mozilla Firefox → プライバシーを重視したブラウザ
    • Safari → Appleのデバイス（Mac、iPhone）で使われるブラウザ

    このレッスンでは、ChromeとEdgeを中心に学習します。

    🎯 体験してみよう: ブラウザを起動してみる

    💭 考えてみましょう: あなたのパソコンには、どのブラウザがインストールされていますか？

    ステップ1: ブラウザを見つけて起動

    ChromeまたはEdgeを探して起動してください

    見つける方法: 1. デスクトップのアイコンを探す（Week 0.1のダブルクリック技術） 2. タスクバー（画面下部）のアイコンをクリック 3. スタートメニューから検索（Week 0.1のスタートメニュー技術）

    📘 ブラウザアイコン一覧

    <details> <summary>💡 ヒント: ブラウザが見つからない場合</summary>

    Windows 11の場合: - Edge: デフォルトでインストールされています - タスクバーに青い「e」のアイコンがあるはずです - またはスタートメニュー → 「Microsoft Edge」と検索

    Chromeをインストールしたい場合: - まずはEdgeを使って、「Chrome ダウンロード」で検索してインストールできます - ただし、このレッスンではEdgeでも十分学習できます

    </details>

    ステップ2: ブラウザ画面の確認

    ブラウザが起動したら、以下の部分を確認してください：

    📘​ ブラウザのレイアウト図

    ブラウザ画面の構成: - アドレスバー → URLを入力する場所（上部の長い枠） - 戻る・進むボタン → ページを前後に移動 - 更新ボタン → ページを再読み込み - タブ → 複数のサイトを同時に開くため - メニューボタン → 設定やブックマークへのアクセス

    📚 セクション2: ブラウザを使いこなそう

    🎯 基本操作をマスターしよう

    2-1: アドレスバーの使い方

    アドレスバーは、ウェブサイトの「住所」を入力する場所です。

    💭 Week 0.1の技術活用: キーボードでの文字入力技術がここで活躍します！

    実践してみましょう:

    🎯 目標: Googleのサイトにアクセスしてみる

    手順: 1. アドレスバーをクリック（Week 0.1のマウス左クリック技術） 2. 「google.com」と入力（Week 0.1のキーボード入力技術） 3. Enterキーを押す

    📘​ アドレスバーの操作説明

    <details> <summary>💡 ヒント: アドレスバーがよく分からない場合</summary>

    アドレスバーの見つけ方: - ブラウザウィンドウの上部にある細長い枠 - 「検索またはWebアドレスを入力」のような文字が薄く表示されている - クリックすると文字カーソルが表示される場所

    うまくいかない場合: - Ctrl + L （Week 0.1のショートカット応用！）を押すとアドレスバーに移動できます

    </details>

    2-2: 戻る・進む・更新の活用

    ナビゲーション操作を覚えましょう。

    📘​ ナビゲーションボタン図

    基本操作: - 戻るボタン（←）: 前のページに戻る - 進むボタン（→）: 次のページに進む（戻った後で使用） - 更新ボタン（🔄）: ページを再読み込み

    実践してみましょう:

    🎯 目標: ナビゲーション操作に慣れる

    手順: 1. google.comにアクセス 2. 何か検索してみる（例：「天気」） 3. 戻るボタンでGoogleのトップページに戻る 4. 進むボタンで検索結果に戻る

    <details> <summary>💡 ヒント: 便利なショートカット</summary>

    Week 0.1の技術発展: - Alt + ← : 戻る - Alt + → : 進む
    - F5 または Ctrl + R : 更新

    これらのショートカットキーを覚えると、マウスを使わずに素早く操作できます！

    </details>

    🗂️ タブ機能を使いこなそう

    タブ機能は、複数のウェブサイトを同時に開いて切り替える機能です。

    💭 Week 0.1の技術発展: ウィンドウ操作技術がタブ操作に発展します！

    2-3: タブの基本操作

    ​📘 タブ操作の図

    タブの操作方法:

    新しいタブを開く: - 「+」ボタンをクリック - Ctrl + T（Week 0.1のショートカット発展！）

    タブを切り替える: - タブをクリック - Ctrl + Tabで次のタブに移動 - Ctrl + Shift + Tabで前のタブに移動

    タブを閉じる: - タブの「×」ボタンをクリック - Ctrl + W

    実践してみましょう:

    🎯 目標: 複数のサイトを同時に開いて管理する

    手順: 1. 新しいタブを開く（Ctrl + T） 2. 「youtube.com」にアクセス 3. もう1つ新しいタブを開く
    4. 「wikipedia.org」にアクセス 5. 3つのタブを行き来してみる 6. 不要なタブを閉じる（Ctrl + W）

    <details> <summary>💡 ヒント: タブの効率的な使い方</summary>

    実用的なタブ活用法: - 調べ物をするとき → 複数の情報源を同時に開く - 学習するとき → 教材と参考資料を分けて表示 - 比較検討 → 複数のサイトを見比べる

    タブが多くなりすぎた場合: - Ctrl + Shift + T で間違って閉じたタブを復活できます

    </details>

    2-4: 右クリックメニューの活用

    💭 Week 0.1の技術活用: 右クリック技術がブラウザでも大活躍！

    リンクを右クリックすると、便利なメニューが表示されます。

    📘​ 右クリックメニューの構成

    よく使う右クリックメニュー: - 新しいタブで開く → 現在のページを残したまま新しいタブでリンクを開く - 新しいウィンドウで開く → 別ウィンドウでリンクを開く - リンクのURLをコピー → リンクのアドレスをコピー（Week 0.1のコピー技術！）

    実践してみましょう:

    🎯 目標: 効率的なリンク操作をマスターする

    🛡️ インターネットセキュリティの基本

    4-1: 安全なサイトと危険なサイトの見分け方

    安全なサイトの特徴を覚えましょう。

    📘​ セキュリティチェックの図

    URLで安全性を確認: ✅ 「https://」で始まる → 暗号化された安全な通信
    ⚠️ 「http://」（sなし） → 暗号化されていない通信

    ブラウザの安全表示: ✅ 鍵マーク（🔒） → 安全な接続
    ⚠️ 警告マーク（⚠️） → 安全でない可能性
    ❌ 「安全でない」表示 → アクセス注意

    実践してみましょう:

    🎯 目標: サイトの安全性を確認する習慣を身につける

    手順: 1. google.comにアクセス 2. アドレスバーの左側を確認 → 鍵マーク（🔒）があるか？ 3. 鍵マークをクリック → 接続情報を確認 4. 他のサイト（Amazon、Wikipedia等）でも同様に確認

    📘 セキュリティの基本情報

    4-2: 個人情報保護の基本原則

    個人情報は大切に守る必要があります。

    保護すべき個人情報: - 名前・住所・電話番号 - メールアドレス - パスワード - クレジットカード情報 - マイナンバー

    基本的な保護原則: 1. 必要でない限り入力しない 2. 信頼できるサイトでのみ入力 3. パスワードは複雑なものを使用 4. 同じパスワードを複数サイトで使い回さない

    危険なサイトの特徴（回避すべき）: ❌ 「個人情報を入力してください」という突然の表示
    ❌ 「おめでとうございます！賞金が当たりました！」などの怪しい表示
    ❌ SSL証明書（鍵マーク）がないサイトでの個人情報入力
    ❌ 日本語が不自然なサイト

    4-3: パスワード管理の基本

    強力なパスワードの作り方と管理方法を学びましょう。

    良いパスワードの特徴: ✅ 8文字以上 → 長いほど安全
    ✅ 大文字・小文字・数字・記号の組み合わせ
    ✅ 辞書にない文字列 → 予測されにくい
    ✅ サイトごとに異なる → 使い回しを避ける

    悪いパスワードの例: ❌ 「12345678」 → 単純すぎる
    ❌ 「password」 → よく使われる文字列
    ❌ 「tanaka123」 → 名前+数字（予測されやすい）
    ❌ 「qwertyui」 → キーボード配列

    パスワード作成のコツ: 💡 「好きな文章の頭文字 + 数字 + 記号」方式

    例：「私は毎日プログラミングを勉強している2025年」 → 「WmppWbS2025!」（Watashi mainichi programming wo benkyou Shiteiru 2025!）

    <details> <summary>💡 ヒント: パスワード管理の実用的な方法</summary>

    初心者におすすめのパスワード管理方法:

    1. 段階的アプローチ - 重要なサイト（銀行、メール等）→ 最強パスワード - 普通のサイト（学習サイト等）→ 中程度パスワード - あまり重要でないサイト→ 簡単なパスワード

    2. 基本パターン + サイト名方式 - 基本パターン：「MyBase123!」 - Google用：「MyBase123!Go」（Googleの略） - Amazon用：「MyBase123!Am」（Amazonの略）

    3. 覚え方のコツ - 紙に書いて安全な場所に保管（デジタルより安全な場合も） - 使用頻度の高いパスワードから覚える - 定期的な見直し（3-6ヶ月に一度）

    </details>
