    💡 クラス設計の思考プロセス

    実際のクラス設計では、以下の順序で考えていきます：

    ステップ1: 要件分析

    「何を表現したいか」を明確にする 

    今回の例：学生を表現したい
- どんな情報を持つべきか？
- どんな動作ができるべきか？
- どんな場面で使われるか？

    ステップ2: データ設計（フィールド）

    「どんなデータを持つか」を決める 

    学生に必要なデータ：
- 名前（name）
- 点数（score）
- 科目（subject）
- その他？

    ステップ3: 機能設計（メソッド）

    「どんな機能を提供するか」を決める 

    学生に必要な機能：
- 成績評価を取得（getGrade）
- 自己紹介をする（introduce）
- 合格判定をする（isPassed）
- その他？

    ステップ4: 実装

    実際にJavaコードでクラスを作成する

    ステップ5: テスト

    作成したクラスが正しく動作するか確認する

    この思考プロセスは、実務でも全く同じです！

    🚀 実践1: 基本的なStudentクラスの作成

    新しいJavaプロジェクトの作成

    1. Eclipse で「File」→「New」→「Java Project」
    2. プロジェクト名: StudentClassDesign
    3. 「Finish」をクリック

    Studentクラスの基本設計

    まず、シンプルなStudentクラスから始めましょう。

    ファイル名: Student.java

    public class Student {
    // フィールド（学生が持つデータ）
    String name;      // 名前
    int score;        // 点数
    String subject;   // 科目

    // メソッド（学生ができる動作）

    // 成績評価を取得するメソッド
    String getGrade() {
        if (score >= 90) {
            return "A";
        } else if (score >= 80) {
            return "B";
        } else if (score >= 70) {
            return "C";
        } else if (score >= 60) {
            return "D";
        } else {
            return "F";
        }
    }

    // 自己紹介をするメソッド
    void introduce() {
        System.out.println("こんにちは！" + name + "です。");

    🔧 フィールドとメソッドの詳細理解

    フィールド（インスタンス変数）の特徴

    public class Student {
    String name;      // インスタンスごとに独立した値を持つ
    int score;        // インスタンスごとに独立した値を持つ
    String subject;   // インスタンスごとに独立した値を持つ
}

    フィールドの重要な特徴： - インスタンスごとに独立: student1.nameとstudent2.nameは別の領域 - デフォルト値: 初期化しない場合、Stringはnull、intは0になる - 直接アクセス: student1.name = "田中"のように直接値を設定可能（今回）

    メソッド（インスタンスメソッド）の特徴

    // インスタンスメソッドの例
String getGrade() {
    if (score >= 90) {  // このscoreは、このインスタンスのscore
        return "A";
    }
    // ...
}

    インスタンスメソッドの重要な特徴： - インスタンス固有: student1.getGrade()はstudent1のデータを使用 - フィールド直接アクセス: メソッド内ではname、score等にそのまま参照 - 戻り値の活用: String grade = student.getGrade()のように結果を取得可能

    staticメソッドとの違い

    // Week3までのstaticメソッド（クラスに属する）
public static void showMessage() {
    System.out.println("Hello");
}

// 今回のインスタンスメソッド（インスタンスに属する）
void introduce() {
    System.out.println("こんにちは！" + name + "です。");  // nameはこのインスタンスのname
}

    重要な違い： - staticメソッド: クラス名で呼び出し（ClassName.methodName()） - インスタンスメソッド: インスタンスで呼び出し（instance.methodName()）

    public class AdvancedStudent {
    // フィールド（学生が持つデータ）
    String name;        // 名前
    int grade;          // 学年
    int mathScore;      // 数学の点数
    int englishScore;   // 英語の点数
    int scienceScore;   // 理科の点数

    // 平均点を計算するメソッド
    double getAverageScore() {
        return (mathScore + englishScore + scienceScore) / 3.0;
    }

    // 総合成績評価を取得するメソッド
    String getOverallGrade() {
        double average = getAverageScore();
        if (average >= 90) return "A";
        else if (average >= 80) return "B";
        else if (average >= 70) return "C";
        else if (average >= 60) return "D";
        else return "F";
    }

    // 最高得点を取得するメソッド
    int getHighestScore() {
        int max = mathScore;
        if (englishScore > max) max = englishScore;
        if (scienceScore > max) max = scienceScore;
        return max;
    }

    // 最低得点を取得するメソッド
    int getLowestScore() {
        int min = mathScore;
        if (englishScore < min) min = englishScore;
        if (scienceScore < min) min = scienceScore;
        return min;
    }

    // 特定科目の成績評価を取得するメソッド
    String getSubjectGrade(String subject) {
        int score = 0;

        if (subject.equals("数学")) {
            score = mathScore;
        } else if (subject.equals("英語")) {
            score = englishScore;
        } else if (subject.equals("理科")) {
            score = scienceScore;
        } else {
            return "科目が見つかりません";
        }

        if (score >= 90) return "A";
        else if (score >= 80) return "B";
        else if (score >= 70) return "C";
        else if (score >= 60) return "D";
        else return "F";
    }

    // 全科目合格判定
    boolean isAllPassed() {
        return mathScore >= 60 && englishScore >= 60 && scienceScore >= 60;
    }

    // 簡単な自己紹介
    void introduce() {
        System.out.println("こんにちは！" + grade + "年生の" + name + "です。");
        System.out.printf("平均点は%.1f点で、総合評価は%sです。\n", 
                         getAverageScore(), getOverallGrade());
    }

    // 詳細な自己紹介
    void introduceDetailed() {
        System.out.println("=== " + name + "の詳細情報 ===");
        System.out.println("学年: " + grade + "年生");
        System.out.println("数学: " + mathScore + "点 (" + getSubjectGrade("数学") + ")");
        System.out.println("英語: " + englishScore + "点 (" + getSubjectGrade("英語") + ")");
        System.out.println("理科: " + scienceScore + "点 (" + getSubjectGrade("理科") + ")");
        System.out.printf("平均点: %.1f点\n", getAverageScore());
        System.out.println("総合評価: " + getOverallGrade());
        System.out.println("最高得点: " + getHighestScore() + "点");
        System.out.println("最低得点: " + getLowestScore() + "点");
        System.out.println("全科目合格: " + (isAllPassed() ? "はい" : "いいえ"));
        System.out.println("========================");
    }
}
