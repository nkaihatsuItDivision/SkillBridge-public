    例外処理の基本概念

    例外とは

    例外（Exception）とは、プログラム実行中に発生する予期しない状況やエラーのことです。適切な例外処理により、プログラムの安定性と信頼性を向上させることができます。

    // 従来の方法：戻り値でエラーを表現
public int divide(int a, int b) {
    if (b == 0)
        return -1; // エラーを示す特別な値
    return a / b;
}

// 使用側で問題が発生
int result = divide(10, 0);
if (result == -1) {
    // エラー処理
} else {
    // 正常処理
    // でも、result == -1 が正当な結果の場合は区別できない！
}

    例外処理の利点

    // 例外を使った方法
public int divide(int a, int b) {
    if (b == 0)
        throw new ArithmeticException(&quot;ゼロで割ることはできません&quot;);
    return a / b;
}

// 使用側
try {
    int result = divide(10, 0);
    // 正常な場合のみ実行される
    System.out.println(&quot;結果: &quot; + result);
} catch (ArithmeticException e) {
    // エラーの場合のみ実行される
    System.out.println(&quot;エラー: &quot; + e.getMessage());
}

    例外処理の4つの利点

    • エラーと正常値の明確な分離：戻り値に特別な値を混入させる必要がない
    • エラー情報の詳細化：エラーメッセージ、スタックトレース、カスタム情報
    • 強制的なエラー処理：例外を無視するとプログラムが停止するため、対処を強制
    • 呼び出し階層の飛び越え：深い呼び出し階層でもエラーを上位に伝播可能

    例外階層構造

    Java例外クラスの階層

    Throwable（すべての例外・エラーの基底クラス）
├─ Error（システムエラー・回復不可能）
│  ├─ OutOfMemoryError（メモリ不足）
│  ├─ StackOverflowError（スタックオーバーフロー）
│  └─ VirtualMachineError（JVM関連エラー）
└─ Exception（プログラム例外・回復可能）
   ├─ RuntimeException（unchecked例外）
   │  ├─ IllegalArgumentException（不正な引数）
   │  ├─ NullPointerException（null参照）
   │  ├─ IndexOutOfBoundsException（配列範囲外）
   │  ├─ ArithmeticException（算術例外）
   │  └─ ClassCastException（型変換エラー）
   └─ その他のException（checked例外）
      ├─ IOException（入出力例外）
      ├─ SQLException（データベース例外）
      ├─ ClassNotFoundException（クラス未発見）
      └─ ParseException（解析例外）

    checked例外 vs unchecked例外

    checked例外（検査例外）

    • コンパイル時にチェックされる例外
    • 必ず処理が必要（try-catchまたはthrows宣言）
    • 回復可能なエラーを表現
    // checked例外の例
public void readFile(String fileName) throws IOException {
    FileReader file = new FileReader(fileName); // IOExceptionの可能性
    // ファイル処理
    file.close();
}

// 呼び出し側は必ず例外処理が必要
public void caller() {
    try {
        readFile(&quot;test.txt&quot;);
    } catch (IOException e) {
        System.out.println(&quot;ファイル読み込みエラー: &quot; + e.getMessage());
    }
}

    unchecked例外（非検査例外）

    • 実行時にチェックされる例外
    • 処理は任意（しなくてもコンパイルエラーにならない）
    • プログラミングエラーを表現
    // unchecked例外の例
public void processArray(int[] array, int index) {
    // ArrayIndexOutOfBoundsException（unchecked）の可能性
    System.out.println(array[index]);
}

// 呼び出し側は例外処理をしなくてもコンパイルできる
public void caller() {
    int[] numbers = {1, 2, 3};
    processArray(numbers, 5); // 実行時にIndexOutOfBoundsException
}

    try-catch-finallyの実行フロー

    基本的な実行順序

    public void demonstrateExecutionOrder() {
    System.out.println(&quot;1. メソッド開始&quot;);

    try {
        System.out.println(&quot;2. try ブロック開始&quot;);

        // 例外が発生する可能性がある処理
        int result = 10 / 0; // ここで例外発生

        System.out.println(&quot;3. この行は実行されない&quot;);
    } catch (ArithmeticException e) {
        System.out.println(&quot;4. catch ブロック実行&quot;);
        System.out.println(&quot;   エラー: &quot; + e.getMessage());
    } finally {
        System.out.println(&quot;5. finally ブロック実行&quot;);
    }

    System.out.println(&quot;6. メソッド終了&quot;);
}

// 実行結果：
// 1. メソッド開始
// 2. try ブロック開始
// 4. catch ブロック実行
//    エラー: / by zero
// 5. finally ブロック実行
// 6. メソッド終了

    try-with-resources文

    Java 7以降で利用可能な、リソースの自動クローズ機能

    // 従来の方法
public void readFileOldWay(String fileName) throws IOException {
    FileReader file = null;
    try {
        file = new FileReader(fileName);
        // ファイル処理
    } catch (IOException e) {
        System.out.println(&quot;ファイルエラー: &quot; + e.getMessage());
        throw e;
    } finally {
        if (file != null) {
            try {
                file.close();
            } catch (IOException e) {
                System.out.println(&quot;ファイルクローズエラー: &quot; + e.getMessage());
            }
        }
    }
}

// try-with-resources（推奨）
public void readFileNewWay(String fileName) throws IOException {
    try (FileReader file = new FileReader(fileName)) {
        // ファイル処理
        // fileは自動的にクローズされる
    } catch (IOException e) {
        System.out.println(&quot;ファイルエラー: &quot; + e.getMessage());
        throw e;
    }
}

    複数catchブロックの処理

    public void processData(String data) {
    try {
        // 様々な例外が発生する可能性がある処理
        int number = Integer.parseInt(data);
        int result = 100 / number;
        System.out.println(&quot;結果: &quot; + result);

    } catch (NumberFormatException e) {
        System.out.println(&quot;数値形式エラー: &quot; + e.getMessage());
    } catch (ArithmeticException e) {
        System.out.println(&quot;算術エラー: &quot; + e.getMessage());
    } catch (Exception e) {
        // その他すべての例外をキャッチ（最も一般的な例外を最後に）
        System.out.println(&quot;予期しないエラー: &quot; + e.getMessage());
        e.printStackTrace();
    }
}

// Java 7以降：マルチキャッチ
public void processDataMultiCatch(String data) {
    try {
        int number = Integer.parseInt(data);
        int result = 100 / number;
        System.out.println(&quot;結果: &quot; + result);

    } catch (NumberFormatException | ArithmeticException e) {
        System.out.println(&quot;入力または計算エラー: &quot; + e.getMessage());
    } catch (Exception e) {
        System.out.println(&quot;予期しないエラー: &quot; + e.getMessage());
    }
}

    カスタム例外クラス

    独自例外クラスの作成

    // checked例外
class InsufficientFundsException extends Exception {
    private final double requestedAmount;
    private final double availableBalance;

    public InsufficientFundsException(double requestedAmount, double availableBalance) {
        super(String.format(&quot;残高不足です。要求額: %.0f円, 利用可能残高: %.0f円&quot;,
                          requestedAmount, availableBalance));
        this.requestedAmount = requestedAmount;
        this.availableBalance = availableBalance;
    }

    public InsufficientFundsException(double requestedAmount, double availableBalance, String message) {
        super(message);
        this.requestedAmount = requestedAmount;
        this.availableBalance = availableBalance;
    }

    public InsufficientFundsException(double requestedAmount, double availableBalance, 
                                    String message, Throwable cause) {
        super(message, cause);
        this.requestedAmount = requestedAmount;
        this.availableBalance = availableBalance;
    }

    public double getRequestedAmount() {
        return requestedAmount;
    }

    public double getAvailableBalance() {
        return availableBalance;
    }
}

// unchecked例外
class InvalidAccountNumberException extends RuntimeException {
    private final String attemptedAccountNumber;

    public InvalidAccountNumberException(String accountNumber) {
        super(&quot;無効な口座番号です: &quot; + accountNumber);
        this.attemptedAccountNumber = accountNumber;
    }

    public String getAttemptedAccountNumber() {
        return attemptedAccountNumber;
    }
}

    カスタム例外を使用するクラス

    public class BankAccount {
    private double balance;
    private final String accountNumber;
    private String accountHolder;

    public BankAccount(String accountNumber, String accountHolder, double initialBalance) {
        validateAccountNumber(accountNumber);
        if (accountHolder == null || accountHolder.trim().isEmpty()) {
            throw new IllegalArgumentException(&quot;口座名義は空にできません&quot;);
        }
        if (initialBalance &lt; 0) {
            throw new IllegalArgumentException(&quot;初期残高は0以上である必要があります&quot;);
        }

        this.accountNumber = accountNumber;
        this.accountHolder = accountHolder;
        this.balance = initialBalance;
    }

    private void validateAccountNumber(String accountNumber) {
        if (accountNumber == null || accountNumber.trim().isEmpty()) {
            throw new InvalidAccountNumberException(&quot;null または空文字列&quot;);
        }
        if (accountNumber.length() != 10) {
            throw new InvalidAccountNumberException(accountNumber);
        }
        if (!accountNumber.matches(&quot;\\d{10}&quot;)) {
            throw new InvalidAccountNumberException(accountNumber);
        }
    }

    public void deposit(double amount) {
        if (amount &lt;= 0) {
            throw new IllegalArgumentException(&quot;入金額は正の値である必要があります&quot;);
        }

        balance += amount;
        System.out.printf(&quot;%.0f円を入金しました。現在の残高: %.0f円%n&quot;, amount, balance);
    }

    public void withdraw(double amount) throws InsufficientFundsException {
        if (amount &lt;= 0) {
            throw new IllegalArgumentException(&quot;出金額は正の値である必要があります&quot;);
        }

        if (amount &gt; balance) {
            throw new InsufficientFundsException(amount, balance);
        }

        balance -= amount;
        System.out.printf(&quot;%.0f円を出金しました。現在の残高: %.0f円%n&quot;, amount, balance);
    }

    public void transfer(BankAccount destinationAccount, double amount) 
            throws InsufficientFundsException {
        if (destinationAccount == null) {
            throw new IllegalArgumentException(&quot;送金先口座が指定されていません&quot;);
        }

        if (destinationAccount.accountNumber.equals(this.accountNumber)) {
            throw new IllegalStateException(&quot;同一口座への送金はできません&quot;);
        }

        // 出金（例外の可能性あり）
        this.withdraw(amount);

        try {
            destinationAccount.deposit(amount); // 入金
            System.out.printf(&quot;口座%sへ%.0f円を送金しました%n&quot;, 
                            destinationAccount.accountNumber, amount);
        } catch (Exception e) {
            // 入金に失敗した場合、出金分を戻す
            this.balance += amount;
            throw new RuntimeException(&quot;送金処理中にエラーが発生しました&quot;, e);
        }
    }

    // getter
    public String getAccountNumber() { return accountNumber; }
    public double getBalance() { return balance; }
    public String getAccountHolder() { return accountHolder; }

    public String getAccountInfo() {
        return String.format(&quot;口座番号: %s%n名義人: %s%n残高: %.0f円&quot;,
                           accountNumber, accountHolder, balance);
    }
}

    実践例題

    例題1: 学生成績管理システムの例外処理

    要求仕様： - 学生の成績を管理するクラス - 成績は0-100の範囲でなければならない - 学生IDは8桁の数字のみ有効 - 成績データの読み込み・保存時のエラーハンドリング - カスタム例外を適切に活用

    解答例： 

    import java.io.*;
import java.util.*;

// カスタム例外クラス群
class InvalidScoreException extends Exception {
    private final int attemptedScore;

    public InvalidScoreException(int score) {
        super(String.format(&quot;無効な成績です: %d（0-100の範囲で入力してください）&quot;, score));
        this.attemptedScore = score;
    }

    public int getAttemptedScore() {
        return attemptedScore;
    }
}

class InvalidStudentIdException extends RuntimeException {
    private final String attemptedId;

    public InvalidStudentIdException(String studentId) {
        super(String.format(&quot;無効な学生IDです: %s（8桁の数字で入力してください）&quot;, studentId));
        this.attemptedId = studentId;
    }

    public String getAttemptedId() {
        return attemptedId;
    }
}

// 学生クラス
class Student {
    private final Map&lt;String, Integer&gt; scores;
    private final String studentId;
    private String name;

    public Student(String studentId, String name) {
        validateStudentId(studentId);

        this.studentId = studentId;
        this.name = Objects.requireNonNull(name, &quot;名前はnullにできません&quot;);
        this.scores = new HashMap&lt;&gt;();
    }

    private void validateStudentId(String studentId) {
        if (studentId == null || studentId.trim().isEmpty()) {
            throw new InvalidStudentIdException(&quot;null または空文字列&quot;);
        }
        if (studentId.length() != 8) {
            throw new InvalidStudentIdException(studentId);
        }
        if (!studentId.matches(&quot;\\d{8}&quot;)) {
            throw new InvalidStudentIdException(studentId);
        }
    }

    public void addScore(String subject, int score) throws InvalidScoreException {
        if (subject == null || subject.trim().isEmpty()) {
            throw new IllegalArgumentException(&quot;科目名が無効です&quot;);
        }

        if (score &lt; 0 || score &gt; 100) {
            throw new InvalidScoreException(score);
        }

        scores.put(subject, score);
        System.out.printf(&quot;%sの%sの成績を%d点で登録しました%n&quot;, name, subject, score);
    }

    public int getScore(String subject) {
        Integer score = scores.get(subject);
        if (score == null) {
            throw new IllegalArgumentException(
                String.format(&quot;科目&apos;%s&apos;の成績が登録されていません&quot;, subject));
        }
        return score;
    }

    public double getAverageScore() {
        if (scores.isEmpty()) {
            throw new IllegalStateException(&quot;成績が登録されていないため平均を計算できません&quot;);
        }

        return scores.values().stream()
                     .mapToInt(Integer::intValue)
                     .average()
                     .orElse(0.0);
    }

    public void displayAllScores() {
        System.out.printf(&quot;%n=== %sの成績一覧 ===%n&quot;, name);
        System.out.printf(&quot;学生ID: %s%n&quot;, studentId);

        if (scores.isEmpty()) {
            System.out.println(&quot;登録された成績はありません&quot;);
            return;
        }

        scores.forEach((subject, score) -&gt; 
            System.out.printf(&quot;%s: %d点%n&quot;, subject, score));

        try {
            System.out.printf(&quot;平均点: %.1f点%n&quot;, getAverageScore());
        } catch (IllegalStateException e) {
            System.out.println(&quot;平均計算エラー: &quot; + e.getMessage());
        }
    }

    // getter
    public String getStudentId() { return studentId; }
    public String getName() { return name; }
}

class GradeManager {
    private final List&lt;Student&gt; students;
    private final String dataFilePath;

    public GradeManager(String dataFilePath) {
        this.students = new ArrayList&lt;&gt;();
        this.dataFilePath = dataFilePath != null ? dataFilePath : &quot;grades.txt&quot;;
    }

    public void addStudent(Student student) {
        Objects.requireNonNull(student, &quot;学生オブジェクトはnullにできません&quot;);

        boolean exists = students.stream()
                               .anyMatch(s -&gt; s.getStudentId().equals(student.getStudentId()));
        if (exists) {
            throw new IllegalStateException(
                String.format(&quot;学生ID %s は既に登録されています&quot;, student.getStudentId()));
        }

        students.add(student);
        System.out.printf(&quot;学生 %s を登録しました%n&quot;, student.getName());
    }

    public Student findStudent(String studentId) {
        return students.stream()
                      .filter(s -&gt; s.getStudentId().equals(studentId))
                      .findFirst()
                      .orElseThrow(() -&gt; 
                          new IllegalArgumentException(
                              String.format(&quot;学生ID %s が見つかりません&quot;, studentId)));
    }

    public void saveToFile() throws IOException {
        try (PrintWriter writer = new PrintWriter(new FileWriter(dataFilePath))) {
            for (Student student : students) {
                writer.printf(&quot;%s,%s%n&quot;, student.getStudentId(), student.getName());
            }
            System.out.printf(&quot;データを %s に保存しました%n&quot;, dataFilePath);
        } catch (IOException e) {
            throw new IOException(
                String.format(&quot;ファイル %s への保存中にエラーが発生しました&quot;, dataFilePath), e);
        }
    }

    public void loadFromFile() throws IOException {
        File file = new File(dataFilePath);
        if (!file.exists()) {
            System.out.printf(&quot;データファイル %s が存在しません。新規作成します。%n&quot;, dataFilePath);
            return;
        }

        try (BufferedReader reader = new BufferedReader(new FileReader(dataFilePath))) {
            String line;
            int lineNumber = 0;

            while ((line = reader.readLine()) != null) {
                lineNumber++;

                try {
                    String[] parts = line.split(&quot;,&quot;);
                    if (parts.length != 2) {
                        System.err.printf(&quot;行%d：データ形式が正しくありません: %s%n&quot;, 
                                        lineNumber, line);
                        continue;
                    }

                    Student student = new Student(parts[0].trim(), parts[1].trim());
                    students.add(student);

                } catch (InvalidStudentIdException e) {
                    System.err.printf(&quot;行%d：%s%n&quot;, lineNumber, e.getMessage());
                } catch (Exception e) {
                    System.err.printf(&quot;行%d：予期しないエラー: %s%n&quot;, 
                                    lineNumber, e.getMessage());
                }
            }
            System.out.printf(&quot;データを %s から読み込みました%n&quot;, dataFilePath);

        } catch (IOException e) {
            throw new IOException(
                String.format(&quot;ファイル %s の読み込み中にエラーが発生しました&quot;, dataFilePath), e);
        }
    }

    public void displayAllStudents() {
        System.out.println(&quot;\n=== 全学生一覧 ===&quot;);
        if (students.isEmpty()) {
            System.out.println(&quot;登録された学生はいません&quot;);
            return;
        }

        students.stream()
               .sorted(Comparator.comparing(Student::getStudentId))
               .forEach(Student::displayAllScores);
    }
}

    public class ExceptionChaining {

    public void processUserData(String userData) throws DataProcessingException {
        try {
            // JSON解析処理
            parseJsonData(userData);

            // 低レベルの例外を業務レベルの例外でラップ

            throw new JsonParseException(&quot;空のJSONデータです&quot;);
        }
    }

    まとめ

    例外処理の本質的理解

    • エラーの分類：回復可能（checked）vs プログラミングエラー（unchecked）
    • 責任の明確化：誰がどの例外を処理すべきかを明確にする
    • 情報の保持：例外には十分なコンテキスト情報を含める
    • 階層の活用：適切な例外階層で柔軟な例外処理を実現

    実践的な例外処理パターン

    • fail-fast原則：問題を早期に検出し、即座に例外をスロー
    • 例外チェーン：低レベル例外を高レベル例外でラップして意味のあるエラーメッセージを提供
    • リソース管理：try-with-resourcesで確実なリソースクローズを実現
    • 適切なログ記録：デバッグに必要な情報を残す

    例外処理設計の原則

    • 具体的な例外タイプを使用：汎用的なExceptionではなく、具体的な例外クラスを使用
    • 例外メッセージの充実：問題の原因と解決方法を明確に示す
    • 適切なスコープでキャッチ：例外を処理できるレベルでキャッチし、処理できない場合は上位に委譲
    • 例外安全性の確保：例外が発生してもオブジェクトの状態が不正にならないよう設計

    堅牢なアプリケーション開発への道筋

    例外処理は、単なるエラー対応ではなく、アプリケーションの品質と信頼性を大きく左右する重要な設計要素です。適切な例外処理により、予期しないエラーからアプリケーションを保護し、ユーザーに分かりやすいエラーメッセージを提供できます。カスタム例外を効果的に活用し、例外処理のベストプラクティスに従うことで、保守性の高いコードを書けるようになります。
